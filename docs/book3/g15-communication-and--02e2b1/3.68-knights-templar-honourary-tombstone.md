# 3.68 Knights Templar honourary tombstone
Orginal: 1098-1099

3.68 圣殿骑士团荣誉墓碑
PCCTS、圣殿骑士团荣誉墓碑是为纪念欧洲抵抗运动的某些殉道者而制作的。授予、制作、分发和处理圣殿骑士团墓碑的后勤安排通常是总管的任务。
总管将主动制作司法骑士墓碑，并确保将其提供给近亲。
西欧关于侵犯坟墓的规定保护任何墓碑，并防止政府移动/更改任何石头上的文字。因此，这是一个告诉世界你真实的感受（曾经感受过）的绝佳方式，这些声明否则无法展示。

墓碑正面：
墓碑正面可用以下文字装饰：
生于马克思主义奴隶制，XX.XX.19XX。作为抗击"西欧国家名称"马克思主义犯罪政权的殉道者而死。SIR KNIGHT"全名"，PCCTS圣殿骑士团司法骑士指挥官（或行动人员），武装民族抵抗运动领导人（或成员/行动人员），于XX.XX.2XXX年为人民自由和祖国主权与独立战斗而殉道。所有自由的欧洲人永远欠你的恩情。

墓碑背面：
墓碑背面有一个骑士盔甲图案或雕塑（这也可以用盾牌替换），显示三个数字，三个重要的年份，标志着欧洲两次最重要的战役（732年和1683年），并包含第三次的预言（2083年）。
背面可用以下文字装饰：
宁殉道而不臣服！永不投降！
"永不投降"引自温斯顿·丘吉尔的话。
IHSV或IN HOC SIGNO VINCES意为"在此标志下，你将征服"，由第一位基督教罗马皇帝君士坦丁一世于公元312年创造。

描述司法骑士墓碑的主要特征：
多立克柱基：
石头简单的多立克柱基象征着司法骑士自愿选择的道路，这条路由斗争、贫困和苦难组成。他忽略了堕落之路，而是选择了牺牲的艰难之路，因为服务他的人民和国家的独立与主权对他来说比以自我为中心的生活更重要。

科林斯柱：
华丽而纤细的科林斯柱（2或4根）代表我们对完美的集体和社会追求。对完美的追求应始终是个人、社会和文明的首要目标。

刺穿墓碑的剑：
一般来说，剑象征着一位高级或重要的军事人物。剑应代表你的地区/国家，并应是你国家历史上一个重要时代的优质复制品。每把剑都应是独特的，并与相关司法骑士的国家历史传统相关联。

可添加独特的符号：
可在墓碑的基础设计上添加独特的国家、地区、文化或宗教符号。例如：一个想要强调奥丁主义文化传统的个人可以在墓碑背面添加符文或其他相关符号。司法骑士应在其遗嘱/遗书中提出此类要求。可以选择民族主义符号，但严格禁止任何与纳粹主义相关的政治符号。

常见墓碑符号：
- 天使——飞着意味着重生或复活；吹号意味着复活；哭泣象征悲伤和哀悼。天使是上帝与人之间的信使。
- 拱门——死亡中的胜利。
- 箭——必死性；殉道。
- 鸟——永生。如果飞翔，代表永生。
- 号角——复活；军事。
- 柱子——天堂入口。
- 皇冠——死后生命的荣耀；正义。
- 上帝之眼——上帝洞察一切。
- 死神——拟人化的死亡。
- 手放在颈上——牺牲。
- 月桂叶和枝——代表特殊成就和卓越。
- 月桂枝和叶——特殊成就，卓越；成功；英雄主义。
