# 3.77 对西欧警察部队（体制保护者）的请求
3.77 A request to the Police Forces (system protectors) of Western Europe
3.77 对西欧警察部队（体制保护者）的请求
关于第一阶段：
我们知道超过60%的欧洲警官同情我们的事业，甚至更多，大约90%的人感同身受。我们的事业是"打击"那些目前在为我们国家的伊斯兰殖民化提供便利的促成者。

请知晓：我们不希望将你们置于危险之中或以任何方式伤害你们。我们将你们视为不幸选择了为"促成者"——多元文化主义联盟、我们的政府和宪法服务的兄弟姐妹。

如果可能，我们将在任何行动期间在标准警用频率上联系你们（以防止无辜警官或就我们而言我们的特工受到伤害）。我们要求你们要么加入我们，要么挡开道路，允许任何司法骑士完成他们的任务。以你自己判断使用创造性的间接方法；"你的无线电不工作"，"堵在车流中"，或避免走"捷径"都是防止与任何圣殿骑士发生武装冲突的绝佳方法。通常我们将有特定目标，仅限于打击甲类和乙类叛徒的集中地，平民伤亡最小化。相信我们的判断，因为我们已经为这次行动彻底规划了数月甚至数年。

然而，如果你们因各种原因决定服从命令，协助追捕和阻止任何司法骑士，他们将假定你们怀有敌意并相应行动。他们不会允许你们将行动置于风险之中。

请知晓，在我们武装抵抗的这个关键而脆弱的阶段，我们没有以灵活方式行动的奢侈。我们被迫有效、无情且毫不犹豫地行动。任何对抗的结果将是死亡，要么是你们，要么是司法骑士。请知晓，如果你们对任何司法骑士造成伤害或死亡，你们不仅是对那个人犯下罪行，而且是对所有民族主义欧洲人犯下罪行/叛国。你们是在协助和保护多元文化主义建制，从而侵犯了自由的欧洲土著人民的意愿。一旦我们，文化保守派，在西欧夺取政权，你们毫无疑问将为你们的罪行受到惩罚。所以要警惕。间接帮助我们，或者别挡我们的道。

你们在第二阶段的角色：
与第一阶段相同。几名警官将在文化保守运动中活跃。

你们在第三阶段的角色：
你们现在终于能够以有限的风险直接支持抵抗运动了。如果可能的话，联系领导你们当地抵抗组织/民兵的军事指挥官并报名服役。要知道，你们的一些同事仍将支持文化马克思主义者。动员尽可能多的同事并联系抵抗运动/文化保守力量以获取指示，不仅是你们的权利，也是你们的责任。确保你们招募的同事可以被信任。以任何方式试图渗透抵抗运动对你们和你们的家人可能是致命的。我们没有资源来允许自己享有一般战争规则的奢侈，使我们能够展现同情和灵活，所以我们不得不有效且常常残酷地反应。

对你们的请求：
致阅读此文的任何警察监狱看守、警官或其他执法代理人；请知晓，你们的职责是间接地为任何和所有司法骑士提供便利（在时机到来时看向别处，归咎于无能或无知）。你们现在应该知道，我们，爱国的司法骑士，正在进行正义的战斗，你们的战斗，为你们而战，这样你和你的家庭可能有机会生活在一个不遭受伊斯兰教或不被其控制的未来社会中。如果你们协助迫害或逮捕一名司法骑士，请知晓，你们不仅背叛了你们的国家、你们的人民、你们的传统、你们的身份和你们的祖先。你们也在背叛你们自己

和你们的孩子！所以当时候到来时，看向别处。这就是我们所求的全部。
