# 3.75 非暴力抵抗——公民不服从/经济破坏
3.75 Non-violent resistance - civil disobedience/economical sabotage
这些方法被称为经济抵抗方法。它们代表了普通爱国者可以在机会出现时用来增加负担并从而削弱体制的方法。

尽你所能避免不必要的税收，即使这意味着违法。因为如果你在帮助维持体制，你基本上就是在资助和便利多元文化主义精英的议程，让他们能够继续将自己的人民卖给伊斯兰奴隶制。

体制本身，只要它被多元文化主义犯罪网络——政权所控制，就是敌人，每个人至少应该做的，是让他们尽可能难以收取他们"战争金库"的税款。

3.75 非暴力抵抗——公民不服从/经济破坏
每个爱国欧洲人都有责任通过任何和所有必要手段为当前文化马克思主义/多元文化主义政权的垮台做出贡献。不幸的是，绝大多数欧洲爱国者目前（第一阶段）尚不愿考虑武装抵抗。尽管如此，普通人仍然有办法贡献于削弱政权。

- 避免缴纳比你必须缴的更多的税，如果你看到一个漏洞——利用它。如果你有可能通过非法手段阻止征税——就这样做（当然前提是你知道你会逃脱）。如果你有机会增加体制负担，即使你的情况并不需要，也要这样做。
