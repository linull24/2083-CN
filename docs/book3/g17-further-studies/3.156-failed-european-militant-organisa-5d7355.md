# 3.156 Failed European militant organisations - Case studies
Orginal: 1473-1473

真正的伊斯兰教吸引了许多穆斯林。基地组织相对不为人知但最重要的成就是，他们通过扩大激进政治轴心，使温和的伊斯兰主义组织变得更可接近。这合法化了几个伊斯兰主义团体，从而改变了"极端伊斯兰"的定义。一些过去看似激进的伊斯兰政治实体现在看起来变得温和了。因此，他们与所谓的温和穆斯林组织协同运作。他们都有相同的目标：征服所有非穆斯林的一切。

保守革命运动——http://en.wikipedia.org/wiki/Conservative_Revolutionary_movement
-
意识形态：民族主义/保守主义
-
政治效果：显著
一战后的一个保守革命运动，由德国民族主义知识分子组成。他们对德国爱国主义的兴起做出了重大贡献，但大多数原始成员最终被党卫军杀害。成员包括各种类型的保守派，从温和派到传统法西斯分子，甚至纳粹。

3.156 失败的欧洲武装组织——案例研究

GRAPO（十月一日反法西斯抵抗团体）——http://en.wikipedia.org/wiki/Grapo
-
意识形态：共产主义/毛主义
-
政治效果：微乎其微
-
失败原因：有缺陷的意识形态

希腊民主军——http://en.wikipedia.org/wiki/Democratic_Army_of_Greece
-
意识形态：共产主义
-
政治效果：无
-
失败原因：有缺陷的意识形态

红军派——http://en.wikipedia.org/wiki/Red_Army_Fraction
-
意识形态：共产主义/亲伊斯兰（巴勒斯坦）
-
政治效果：显著
-
解散原因：西德实施文化马克思主义/多元文化主义改革对激进左翼产生了中和/安抚作用。

弗拉芒武装分子组织——http://en.wikipedia.org/wiki/Order_of_Flemish_militants
-
意识形态：国家社会主义、民族主义、地区主义（包括反法语区）
-
政治效果：适得其反
-
失败原因：国家社会主义在二战中已灭亡，有缺陷的意识形态。
