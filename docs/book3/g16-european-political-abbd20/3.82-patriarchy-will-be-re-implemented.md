# 3.82 父权制将被重新实施
3.82 Patriarchy will be re-implemented

以下菲利普·朗曼的文章进一步记录了西欧右翼知识分子二十多年来一直陈述的观点。父权制将会回归。西欧唯一剩下的问题是：未来的欧洲将受穆斯林还是基督徒父权制的支配？欧洲保守派能否在文化马克思主义者和其他自杀性自由主义者成功将欧洲人民出卖给穆斯林奴役之前，通过军事力量或政变夺取权力？我们只有20-70年的时间，就会被伊斯兰的洪流在人口上淹没。人口为王，除非我们设法在未来20-70年内将所有穆斯林从欧洲土地上驱逐出去，欧洲将会沦陷。

我们真诚地希望西欧的多元文化主义政权能在欧洲各国首都再次完全沦为废墟之前，以相对和平的方式向保守派力量投降。不幸的是，我们的希望被一种直觉所笼罩，即他们将不愿投降，正如我们不愿投降一样。欧洲将再次燃烧，爱国者、暴君和叛徒的鲜血之河将流淌于街头。然而，一场新的欧洲文化复兴将从灰烬中诞生。伊斯兰和马克思主义不会胜利。

**父权制的回归**
作者：菲利普·朗曼

"如果我们能够没有妻子而生存，罗马的公民们，我们所有人都会免除那个麻烦。"罗马将军、政治家和监察官昆图斯·卡西利乌斯·梅特卢斯·马其多尼库斯在公元前131年如此宣称。然而，他继续恳求道，下降的出生率要求罗马男人履行他们繁衍的责任，无论罗马女人变得多么令人恼火。"既然自然如此规定，我们不能舒适地与她们相处，也不能以任何方式没有她们而生活，我们必须为我们的持久保存而计划，而不是为了我们暂时的快乐。"

在过去200年中人类数量增加了六倍以上，现代人简单地假设，无论男女之间多么疏远，他们总会生育足够的孩子来增长人口——至少在瘟疫或饥荒降临之前。这不仅符合我们对一个日益拥挤世界的长期经验，而且还得到了托马斯·马尔萨斯及其众多现代追随者等有影响力的思想家的认可。

然而，一代多以来，世界各地吃得饱、健康、和平的人口一直在生育太少的孩子，无法避免人口下降。即使婴儿和儿童死亡率的显著改善意味着今天需要更少的孩子（在现代社会每名妇女仅需约2.1个）来避免人口损失，情况也是如此。出生率在一个又一个国家下降到远低于更替水平——从中国、日本、新加坡和韩国，到加拿大、加勒比、整个欧洲、俄罗斯，甚至中东部分地区。

由于担心老年人口超过年轻人的未来，许多政府正在尽一切努力鼓励人们生育。新加坡赞助了"快速约会"活动，希望将忙碌的专业人士聚集在一起结婚生育。法国为愿意组建家庭的人提供慷慨的税收优惠。在瑞典，国家资助日托以缓解工作和家庭生活之间的紧张关系。然而，尽管这种明确的鼓励生育政策可能会鼓励人们在较年轻时生育，但几乎没有证据表明它们会导致人们生育比原本更多的孩子。正如远至罗马帝国时期的政府所发现的那样，当文化和经济条件阻碍为人父母时，即使独裁者也无法强迫人们去生养众多。

在人类历史的广阔进程中，有许多人或阶层选择避免为人父母的代价的例子。事实上，生育率下降是人类文明反复出现的趋势。那么为什么人类很久以前没有灭绝呢？简短的答案是父权制。

父权制不仅仅意味着男性统治。事实上，它是一种特定的价值体系，不仅要求男性结婚，而且要求他们娶一位适当出身的女性。它与许多其他男性对美好生活的愿景相竞争，仅此原因就倾向于周期性出现。然而，在它退化之前，它是一种文化制度，用于在富裕阶层中保持高出生率，同时最大化父母对子女的投资。还没有任何先进文明学会如何在没有它的情况下持久生存。

通过一个文化进化的过程，采用这种特定社会制度的社会——这远非简单的男性支配——最大化了自己的人口，从而最大化了自身的权力，而没有采用这一制度的要么被吞没，要么被吸收。人类历史上的这一循环可能让开明之士感到厌恶，但它注定要卷土重来。

**保守派的婴儿潮**

父权制、人口和权力之间的历史关系对我们这个时代有着深远的影响。正如美国今天在伊拉克所发现的那样，人口仍然是权力。智能炸弹、激光制导导弹和无人驾驶飞机可能大大扩展霸权的暴力范围。但最终，往往是士兵的数量改变历史。即使生育率接近更替水平，美国也缺乏维持世界帝国角色所需的人口数量，正如英国在其出生率在20世纪初崩溃后失去这种能力一样。对于中国、德国、意大利、日本和西班牙等一孩家庭现在成为常态的国家来说，人力资本的质量可能很高，但它实际上已经变得太过稀少而不敢冒风险。

生育率下降也是导致今天头条新闻中许多金融和经济问题的原因。社会保障计划、私人养老金计划和医疗保健系统的长期融资与人们寿命延长关系不大。老年时期预期寿命的增长实际上相当温和，而美国的改善速度在过去三十年中每十年都在下降。相反，工人与退休人员比例的下降绝大多数是由从未出生的工人造成的。随着政府对日益减少的劳动年龄人口征税以覆盖支持老年人日益增长的负担，年轻夫妇可能得出结论，他们比父母更无法负担孩子，从而引发新的人口老龄化和下降循环。

出生率下降也改变了国家性情。例如，在美国，出生于1930年代末的女性中无子女的比例接近10%。相比之下，出生于1950年代末的女性中近20%正在走完生育生命而没有孩子。当代社会中大大扩展的无子女群体，其成员不成比例地来自1960年代和70年代的女权主义和反文化运动，将不会留下任何遗传遗产。他们对下一代的情感或心理影响也无法与他们的父母相比。

与此同时，一孩家庭容易灭绝。一个孩子替代父或母中的一方，但不能替代两位。一孩家庭对未来人口的贡献也不大。婴儿潮一代女性中17.4%只有一个孩子，占下一代出生儿童的仅7.8%。相比之下，婴儿潮一代子女的近四分之一来自仅有11%的婴儿潮女性，她们生育了四个或更多孩子。这些情况正在导致一个新社会的出现，其成员将不成比例地是那些拒绝曾使无子女和小家庭成为常态的社会倾向的父母的后代。这些价值观包括坚持传统的、父权制的宗教，以及强烈认同自己的民族或国家。

这种动态有助于解释，例如，美国文化从世俗个人主义向宗教原教旨主义的逐渐偏移。在2004年投票支持乔治·W·布什总统的州中，生育率比投票支持参议员约翰·克里的州高12%。这也可能有助于解释普通欧洲人中日益增长的民众对欧洲联盟等世俗自由主义瑰宝的抵制。事实证明，最可能自认为"世界公民"的欧洲人也是最不可能有孩子的。

这是否意味着今天开明但生育缓慢的社会面临灭绝？可能不会，但只是因为它们面临着一个戏剧性的、由人口驱动的文化转型。正如历史上多次发生的那样，当社会中世俗和自由主义元素无法繁衍，当坚持更传统、父权制价值观的人通过默认继承社会时，就会发生这种转型。

至少早在古希腊和罗马时代，许多世故的社会成员就得出结论，投资于孩子没有任何好处。相反，孩子开始被视为自我实现和世俗成就的昂贵障碍。但是，尽管这些态度导致了许多个体家庭的灭绝，它们并没有导致整个社会的灭绝。相反，通过文化进化的过程，一套大致可以描述为父权制的价值观和规范重新出现。

**人口为王**

在原始过去，确实大多数社会不强制生育，因为他们必须避免以比他们赖以生存的野生猎物更快的速度繁殖。事实上，在几乎所有存活到足以被人类学家研究的狩猎采集社会中，如爱斯基摩人和塔斯马尼亚布须曼人，人们发现以这样或那样的方式阻碍人口增长的习俗。这些以各种组合的方式包括晚婚、生殖器切割、堕胎和杀婴。一些早期的狩猎采集社会可能还通过给予女性高地位职位来限制人口增长。允许至少一些女性扮演如女祭司、巫师、神谕者、艺术家甚至战士等角色，将为母亲身份提供有意义的替代选择，从而将总体生育率降低到可持续的限度内。

在农业出现之前的万古时期，几乎没有军事理由来促进高生育率。战争和征服几乎不会给社会带来任何好处。没有粮仓可以劫掠，没有牲畜可以偷窃，除了强奸之外没有奴隶的用处。但随着新石器时代农业革命的出现，大约从11000年前开始，一切都改变了。植物和动物的驯化导致了食物供应的极大增加。剩余食物使城市得以出现，并释放了更多人从事诸如建造金字塔和发展书面语言以记录历史等项目。但农业革命带来的最决定性的变化是它将人口转化为权力的方式。由于食物的相对丰裕，越来越多的社会发现，对其生存的最大人口威胁不再是人口过剩，而是人口不足。

在那一点上，高生育率的社会没有因为饥饿而死亡，而是在力量和数量上增长，并开始威胁那些生育率较低的社会。在世界越来越多的地方，快速繁殖的部落蜕变为民族和帝国，并扫除了任何剩余的缓慢繁殖的狩猎者和采集者。你的战士在战斗中凶猛勇敢确实重要；更重要的是他们有很多人。

这是皮洛士国王在公元前三世纪学到的一课，当时他率领希腊军队进入意大利半岛并试图对付罗马人。皮洛士最初在阿斯库鲁姆的一场大战中获胜。但正如人们所说，这是一场"皮洛士式的胜利"，皮洛士只能得出结论："再对罗马人取得这样一场胜利，我们就完蛋了。"罗马人，到那时繁殖速度远远快于希腊人，不断涌入援军——"如同从城市的泉源中不断涌出"，希腊历史学家普鲁塔克告诉我们。在人数上毫无希望地处于劣势，皮洛士最终输掉了战争，而希腊在经历了漫长的人口下降时期后，最终成为罗马被掠夺的殖民地。

与今天的现代、粮食充足的国家一样，古希腊和罗马最终发现她们的精英对家庭生活那些常常沉闷的事务失去了兴趣。"在我们的时代，整个希腊都遭受了儿童的短缺和人口普遍衰退，"希腊历史学家波利比乌斯在公元前140年左右哀叹道，当时希腊正在屈服于罗马的统治。"这种弊病迅速无情地降临到我们身上，由于我们的男人堕落为对排场、金钱和安逸生活乐趣的热情。"但是，与全球各地文明一样，父权制只要能维持，就是维持人口从而维持权力的关键。

**父亲最知道？**

父权制社会有许多变种，并通过不同阶段演化。它们的共同点是有习俗和态度共同作用以最大化生育率和父母对下一代投资的。其中最重要的之一是对"私生"子女的污名化。衡量父权制在先进社会中削弱程度的一个指标是对非婚生育日益增长的接受，这在斯堪的纳维亚国家例如现已成为常态。

在父权制下，"私生子"和单身母亲不能被容忍，因为他们破坏了男性对下一代的投资。私生子不随父亲姓，因此他们的父亲，即使已知，也往往不为他们承担任何责任。相比之下，"合法"子女成为他们父亲和家族血脉的荣誉或耻辱来源。合法子女属于父亲家庭而不属于母亲家庭的观念（这在生物学上没有依据），给许多男性以强大的情感理由来想要孩子，并希望他们的孩子在传承遗产方面取得成功。父权制也使男性继续生育直到他们至少生出一个儿子。

父权制进化优势的另一个关键在于它如何惩罚不结婚和生育的女性。仅仅几十年前在英语世界，这样的女性，甚至被她们自己的母亲，称为老处女或老姑娘，因不育而被怜悯或因自私而被谴责。父权制使接受丈夫成为全职母亲的激励非常高，因为它给女性提供了很少的理想替代选择。

当然，按照这种原则组织的社会可能随时间退化为厌恶女性，最终导致不育，正如古希腊和罗马所发生的那样。在更近的时代，父权家庭也被证明容易受到资本主义崛起的损害，资本主义从女性劳动力从家庭转移到工作场所中获利。但只要父权制度避免屈服于这些威胁，它将比其他原则组织的社会产生更多数量的孩子，并且可以说更高质量的孩子，这就是进化所关心的全部。

这一主张是有争议的。毕竟，今天我们将父权制与对妇女和儿童的可怕虐待、贫困和失败的国家联系在一起。脑海中浮现出塔利班叛军或尼日利亚穆斯林狂热分子用石头砸死通奸妇女。然而，这些是已经退化为男性暴政的不安全社会的例子，它们并不代表在人类历史上获得进化优势的父权制形式。在真正的父权制度下，如早期罗马或17世纪的新教欧洲，父亲有强烈的理由积极关心他们妻子所生的孩子。这是因为，当男性将自己视为并被视为父权世系的捍卫者时，这些孩子的成长结果直接影响到他们自己的地位和荣誉。

在父权制下，母亲对子女的投资也增加了。正如女权主义经济学家南希·福尔布雷观察到的那样，"对女性的父权控制倾向于增加她们在生育劳动中的专业化，对下一代投资的数量和质量都有重要后果。"这些后果可能包括：更多的孩子得到母亲更多的关注，母亲几乎没有其他方式来寻找生活的意义，她们变得更加擅长保持孩子的安全和健康。在不表示任何支持该策略的情况下，必须观察到，一个基本上给女性提供三种选择的社会——做修女、做妓女，或嫁给男人生育孩子——已经偶然发现了一种减少人口下降风险的高度有效方法。

**父权制及其不满**

父权制可能享有进化优势，但没有什么能确保任何特定父权社会的生存。一个原因是男性可能厌倦父权制的要求。例如，罗马贵族最终变得如此不愿接受作为一家之主的负担，以至于凯撒·奥古斯都感到被迫颁布严厉的"单身汉税"，并以其他方式惩罚那些保持未婚和无子女的人。父权制可能有其特权，但它们可能在一个奢华的社会中与单身汉的快乐相比黯然失色——与朋友在宴会上愉快地讨论体育、战争故事或哲学，或与迷人的情妇、笛子女孩或聪明的交际花共度夜晚。

女性当然也有理由厌倦父权制，特别是当男性自己不再履行父权职责时。历史学家苏珊娜·克罗斯指出，在罗马内战的几十年中，各个阶层的罗马女性不得不学习如何在长时间内没有男性的情况下生活，从而形成了新的个性和独立意识。上层阶级中很少有女性会同意嫁给虐待性丈夫。通奸和离婚变得猖獗。

通常，维系父权家庭的唯一东西是其成员在维护一个悠久而高贵世系荣誉的理念。然而，一旦一个社会变得国际化、快节奏，并充满了新思想、新人民和新奢侈，这种荣誉感和与祖先的联系就开始消退，随之而来的是对繁殖必要性的任何感觉。"当一个高度教养的民族的普通思想开始将'生孩子'视为一个利弊问题，"德国历史学家和哲学家奥斯瓦尔德·斯宾格勒曾观察到，"伟大的转折点已经到来。"

**父权制的回归**

然而，那个转折点不一定意味着文明的死亡，而只是它的转型。最终，例如，罗马帝国不育的、世俗的、高贵的家族消亡了，随之而来的是他们祖先关于罗马的理念。但曾经是罗马帝国的地方仍然有人居住。只有人口的组成发生了变化。几乎默认地，它由新的、高度父权制的家庭单元组成，对世俗世界充满敌意，并被信仰所要求：要么去生养众多，要么加入修道院。随着这些变化而来的是一个封建的欧洲，但不是欧洲的终结，也不是西方文明的终结。

我们可能在本世纪见证类似的转型。例如，在今天的欧洲，不同的人有多少孩子以及在什么环境下，与他们对广泛的政治和文化态度的信念强烈相关。例如，你不信任军队吗？那么，根据人口统计学家罗尼·莱斯特海格和约翰·苏尔肯收集的民意数据，你比那些说他们对军队没有异议的人更不可能结婚和有孩子——或永远结婚和有孩子。或者，你是否认为软性毒品、同性恋和安乐死是可接受的？你是否很少，如果有的话，参加教会？无论什么原因，对这些问题回答肯定的人比回答否定的人更可能独自生活，或在无子女的同居关系中生活。

世俗个人主义者和宗教或文化保守派之间生育率的巨大差异预示着现代社会中巨大的、由人口驱动的变化。例如，考虑法国的人口统计。在1960年代初出生的法国女性中，不到三分之一有三个或更多孩子。但这个明显的少数法国女性（她们中大多数可能是有实践的天主教徒和穆斯林）生育了同代人所有子女的50%以上，很大程度上是因为她们的许多同时代人只有一个孩子或根本没有孩子。

许多无子女的中年人可能后悔导致他们家族血脉灭绝的人生选择，然而他们却没有儿子或女儿来分享他们新发现的智慧。只有一个孩子的大多数公民可能能够在该孩子的教育上投入巨资，但一个孩子只能替代父母中的一方，而不是两位。与此同时，有三个或更多孩子的父母的后代将在后续世代中大量过度代表，而引导他们父母拥有大家庭的价值观和理念也将如此。

可以说，历史，特别是西方历史，充满了孩子反叛父母的事件。明天的欧洲人，即使他们不成比例地在父权制的、信仰宗教的家庭中长大，难道不可能成为另一个68年一代吗？

关键的区别在于，在二战后时期，现代社会的几乎所有阶层都结婚和生育。有些人比别人多，但宗教和世俗之间的家庭规模差距并没有那么大，无子女现象也很罕见。今天，相比之下，无子女是普遍的，甚至有孩子的夫妇通常也只有一个。因此，明天的孩子，与战后婴儿潮一代的成员不同，将大部分是社会相对狭窄和文化保守的部分的后代。当然，新一代的一些成员可能拒绝父母的价值观，如常发生的那样。但当他们环顾四周寻找志同道合的世俗主义者和反文化主义者时，他们会发现大多数潜在的同行者实际上从未出生。

先进社会正在变得更加父权制，无论它们是否喜欢。除了社会保守部分的更高生育率之外，由人口老龄化和下降迫使的福利国家缩减将给这些元素以额外的生存优势，从而刺激更高的生育率。随着政府交回他们曾经从家庭中占有的功能，特别是老年支持，人们将发现他们需要更多的孩子来保障他们的黄金岁月，并将寻求通过灌输类似于圣经中"孝敬父母"的诫命的传统宗教价值观来将他们的孩子与他们绑定。

今天最世俗、最慷慨的对其资金不足的福利国家的社会将最倾向于宗教复兴和父权家庭的再生。欧洲和日本的绝对人口可能急剧下降，但剩余的人口将通过一个类似于适者生存的过程，适应一个新的环境，在这个环境中没有人可以依赖政府来代替家庭，并且一个父权制的上帝命令家庭成员压抑他们的个人主义并服从于父亲。
