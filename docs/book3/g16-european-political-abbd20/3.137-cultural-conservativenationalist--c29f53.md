# 3.137 第三阶段期间及之后的文化保守/民族主义改革
# 3.137 Cultural conservative/nationalist reforms during and after phase 3
Orginal: 1303-1304

各国必须制定一项泛欧计划，以抵御来自残余的欧苏联/美苏联文化集团的任何意识形态攻击；

新闻机构及/或相关媒体公司（私人和公共）
外国对新闻机构及/或相关媒体公司的所有权
应禁止外国拥有新闻机构及/或相关媒体公司的所有权。如果该外国实体被认为是"友好的"（文化保守主义者），则可以例外。

重组新闻机构及/或相关媒体公司
管理层、董事会和员工必须经过彻底审查。必须对所有个人进行背景调查。历史上支持多元文化主义观点的个人应被文化保守主义者替换和/或再教育。这包括所有记者/编辑/出版人。或者，至少要有50%的人必须同情文化保守主义的政治原则。

逮捕任何甲类、乙类或丙类叛徒。

处理外国或部分外国的新闻机构及/或相关媒体公司
例如在特定国家拥有广播权的外国公司（电视、卫星、广播和互联网）。
将提出要求。研究当前中国或俄罗斯如何处理这些问题。某些外国新闻机构及/或等同的媒体公司将被视为敌对政治实体（政治宣传中心）。

重组教育部门
大学、学院、高中、初中和小学等。
管理层、董事会和员工必须经过彻底审查。必须对所有个人进行背景调查。历史上支持文化马克思主义/多元文化主义观点的个人应被文化保守主义者替换和/或再教育。
此外，学校/大学课程必须彻底修订。
逮捕任何甲类、乙类或丙类叛徒。

3.137 第三阶段期间及之后的文化保守/民族主义改革
1. 防止伊斯兰影响并驱逐欧洲目前所有的伊斯兰元素。
2. 启动欧洲文化复兴（重点是教育和文化改革）。
3. 在军事上协助邻国的文化保守运动。
4. 协助邻国政变并巩固力量（与友好的新文化保守政府）。

当前的政治体系将进行改革（重点是将当前西欧政治体系改造为宪政民主（某些"管理区域"））。这将包括以下领域的改革：
防务
文化（宗教和文化改革）。伊斯兰教将被重新归类为极权主义、种族灭绝和种族主义的政治意识形态（与国家社会主义和多元文化主义相当），并随后被禁止。
-
多元文化主义，一种反欧洲的仇恨意识形态
-
国家社会主义，一种反犹太人的仇恨意识形态
-
伊斯兰教，一种反基督教、反犹太教、反印度教、反佛教、反无神论的仇恨意识形态

欧洲当前和过去所有伊斯兰影响的痕迹都将被消除。这包括清真寺和伊斯兰文化中心。所有穆斯林将被驱逐出欧洲领土。

在第一阶段和第二阶段皈依"友好宗教/意识形态"并接受了同化要求的个人将被视为欧洲公民。所有佛教、印度教、犹太教或其他"友好信仰/意识形态"的个人将被视为所有欧洲人的朋友/盟友/兄弟姐妹，现在或将来都不会受到同样的同化要求。

次要目标
禁止当前和未来的欧洲政治反俄活动。这包括禁止由几个欧盟国家（特别是英国和北欧国家）领导的当前反俄活动。
这些反俄活动已经持续了100多年，在英国首相迪斯雷利（Disraeli）统治时期达到顶峰（当时英国同意保护伊斯兰奥斯曼帝国免受俄罗斯侵害，作为回报获得了塞浦路斯）。
我们承认，整合俄罗斯和"前"欧盟将具有挑战性（由于某些欧盟国家数十年的反俄宣传），但我们将成功，前提是俄罗斯接受我们的要求。
