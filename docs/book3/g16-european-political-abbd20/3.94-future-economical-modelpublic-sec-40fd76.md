# 3.94 未来经济模式/公共部门模式/福利计划/经济区/仆人阶层
# 3.94 Future economical model/public sector model/welfare program/economic zones/servant class

Orginal: 1195-1197

如多次提及，正在进行的欧洲内战不是阶级战争，因此不是社会主义与资本主义之间的战争。它是文化保守派与文化马克思主义者之间的文化战争（民族主义与国际主义）。尽管如此，我们文化保守派必须确保我们继续拥有基于许多自由市场机制的可持续经济。虽然全球主义资本主义是一个破坏性概念，并不意味着局部资本主义机制也是如此。必须始终为创造工作场所的个人保留有力的激励。必须在所有代价下避免社会主义或甚至计划经济。

自由放任常用于指各种经济哲学和政治哲学（自由主义），旨在最小化或消除政府在大多数或所有社会方面的干预。纯粹的自由放任资本主义是一个全球主义概念（无政府干预），具有若干缺点。必须有许多经济保护措施到位，以确保我们文化保守经济区（欧洲联邦）的经济可持续性。

**未来公共部门削减以平衡大量外购新私营部门工作岗位的内购**
超级膨胀的西欧官僚机构/公共部门应从当前平均50-60%削减到约25%。将有一个缓冲，由一个相当于额外10%的公共工作场所池组成，可在经济衰退期间临时创建（从所有工作的25%到35%）。这些显著的削减将对目前拥有庞大且超级膨胀的公共部门的许多西欧公共结构产生巨大影响。然而，由于我们未来计划对非欧洲联邦经济区实施显著关税壁垒，将有数百万个新的工作岗位被内购。将有一个巨大规模的欧洲反全球主义转变，导致以前外包给非EF国家的工作场所重新创建。未来的西欧公共部门与私营部门模式重新引入了这样一个概念，即国家应该服务于其公民，而不是反过来。经过显著瘦身的未来国家，将更好地适应带头推进文明目标的焦点。

**经济团结/未来福利国家**
认为在有自由市场的国家中将有完全的经济平等是不现实的。然而，我们应该尝试在某种程度上遵循斯堪的纳维亚福利模式，以确保所有欧洲公民拥有坚实的福利计划。维持可持续的坚实福利体系取决于勤奋工作的个人和社会凝聚力/单一文化社会。福利支出不应优先于用于科学/技术、研究和发展的20%固定额度。推进长期欧洲利益和确保我们欧洲文明的长期可持续性，必须始终优先于短期的过度福利支出（贪婪）。

**未来经济区**
许多亚洲国家的主要问题是它们不尊重知识产权所有权，因此许多行业的产品被系统性地窃取和/或非法滥用。此外，我们真的无法与中国等国家竞争。将它们邀请加入WTO是过去60年犯下的最大错误之一，导致数百万欧洲就业机会的丧失。日本、韩国和其他几个国家也通过操纵本国货币来在进口/出口问题上获利，在规则之外行事。

为了说明日本的犬儒主义，只需研究柯达案例。柯达本应在日本拥有至少30-40%的市场份额。然而，问题在于许多欧洲/美国产品由于日本政府和人民民族主义/保护主义倾向/心态而被公开抵制。与欧洲和美国不同，日本人更多被灌输爱国主义，而不间断的社会凝聚力（日本和韩国拒绝了多元文化主义）是他们的"大众民主模式"仍然运作良好的原因。

我们应该创建一个新的经济区——欧洲联邦——在那里：欧洲、美国、俄罗斯（+加拿大、澳大利亚、新西兰、以色列）享有某些特权。只有拥有该三方个体/公司/国家多数所有权的全球主义公司才被允许在欧洲联邦市场享有完全特权。该区域之外需要更多的保护主义机制。外包工作到该区域之外不应发生。或者，我们可以在这些区域内创建某些孤立的区域，像是一个枢纽（允许第二/第三世界者最多工作12个月（12个月合同））。

若干国际（全球主义）公司目前拥有太多影响力，我们正目睹它们推动更多移民，甚至土耳其的欧盟成员资格。至关重要的是，我们确保国家政府在这些公司中拥有控制性股份，或者禁止它们在我们的市场享有当前权利。应禁止国际公司（我们未拥有多数股份的公司）游说，并且不应能关于移民、外包政策和国家主权拥有任何影响力。

**未来仆人阶层**
使用廉价外籍劳工，特别是建筑部门内，将是合乎逻辑的。这些工人应被给予12个月的"专注合同"，并且必须在任期结束后返回其原籍国。专注合同涉及每天工作长达12小时。这些人将获得有竞争力的工资（与其原籍国相比）。

现代大规模移民的主要论点之一是证明对劳动的需求是合理的，这些工作是原住民欧洲人不愿意做的。那么，当我们完全停止移民并驱逐所有穆斯林时，谁会接这些工作呢？当前大规模移民模式的问题在于，每个人，根据国家，在7年内被授予公民身份。这些第一代移民的孩子不关心父母一代接的工作，所以我们每二十年必须接受更多移民来填补这些空缺。它发展成一个永无止境的大规模移民循环，最终导致原住民成为少数群体的情况。

以下解决方案将提供一个仆人阶层，涵盖以下职业：
- 清洁
- 建筑工作
- 园艺
- 出租车
- 农耕/收获

6-12个月的"专注合同"将被提供给来自孟加拉国、巴基斯坦和印度的个人（考虑到我们对全球伊斯兰乌玛的敌对立场，这些服务可能保留给基督徒、印度教徒、锡克教徒和佛教徒）。这些人将每月成批飞来，并在合同结束时离开。在他们的逗留期间，他们将在合同期间（6或12个月）每天工作12小时，然后飞回他们的家园。每12个月合同之间应有至少6个月的隔离期，以防止个人变得过于文化附着。

这些人将居住在每个主要城市预定义区域的隔离社区中，并且必须提供免费医疗服务、限制兵营城镇的免费住房和补贴食品/必需品以及补贴交通费用。这将允许他们带回更大比例的工资。他们将获得相当于其原籍国收入的300%的补偿率。例如，如果他们在孟加拉国每月赚200欧元，他们将在这里赚600欧元。

这对特别是女性来说可能听起来犬儒（这不是奴隶制，因为奴隶制是剥夺人民的自由），但事实上这比目前阿拉伯国家（沙特阿拉伯、迪拜、阿联酋等）提供给他们来自孟加拉国、印度、巴基斯坦和菲律宾的外来工人的要慷慨得多。超过90%的阿联酋总劳动力由外国人组成（主要来自印度、巴基斯坦和孟加拉国）。外国人中没有一个被授予公民身份，但通常以非常低的工资在12个月合同下工作。此外，这些安排与美国南部各州目前提供给拉丁美洲外来工人的安排没有太大不同。

最重要的是，大规模实施这些"外来工人"安排将被证明是一种非常有效的廉价劳动力来源，这些安排可以很容易地调节以适应牛市和熊市。因为会有经济衰退或缓慢时期，当我们需要为我们的自己人提供许多这些工作时，而可能有经济繁荣，将涉及增加使用这些安排。

广泛使用这些"外来工人"安排的经济影响将对我们文明的竞争力、我们的经济区产生重大影响，并将确保我们有足够的资金（GDP的20%）来投入到科学、研发（当然还要防止非EF国家窃取这一科学资本）。它将使我们成为世界的经济强国和全人类的灯塔。它将使我们能够为人类的进一步进步做出重大贡献。
