# 3.91 未来民主模式与单一文化主义
# 3.91 Future democracy model and monoculturalism

Orginal: 1187-1187

已毫无疑问地证明，西欧过去50年使用的大众民主模式已经为我们的当前问题铺平了道路。"大众民主"已经允许文化马克思主义劫持社会的政治结构，最初是通过学校、大学和媒体。当西欧的文化马克思主义/多元文化主义政权在第三阶段（2070-2090）被击败后，必须创建一种新型的政治结构，对未来的马克思主义渗透尝试免疫。这一结构应受一个保守的监护委员会保护。

某些政治原则应为未来制定并不可更改。将"受管理"的政治领域：

至少应有一个议会席位保留给"文化和科学监察官"，该官员将有权否决议会提出的任何违反国家主要教义的新法案（最好仅在极其罕见的情况下使用，且仅当主要价值观受到威胁时）。

我们必须避免一党制国家体系，因为这在长远来看是不可取的。只要我们在文化领域（文化、生育、国防安全、移民）实施了永久机制，我们就可以负担得起在大多数其他领域继续保证平衡的政治代表性。

文化方面——单一文化、基督教世界的位置、科学目标（预算的20%将预留用于科学和发展）。保护国政策——未来的欧洲将成为中东基督教国家和以色列的保护者，并有义务帮助世界各地的所有基督教群体在未来圣战中自卫。如果我们的佛教和印度教兄弟请求帮助对抗圣战攻击或运动，应提供援助并发送给他们。

文化马克思主义/欧洲多元文化主义/极端平等主义将被标记为种族主义、种族灭绝性的、反欧洲的仇恨意识形态，因此将被永久禁止。

重新引入单一文化主义的正当理由和驱动力将基于与社会凝聚力相关的实际问题和情况，而非基于仇恨。这一单一文化主义的重新引入将类似于今天日本、韩国和台湾的做法。
