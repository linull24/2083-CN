**3.57 利用欧洲核电站作为大规模毁灭性武器**

"除了我们自己施加的限制之外，没有任何限制……！"

切尔诺贝利，一次事故还是有史以来最成功的反马克思主义袭击？许多政治分析家声称，切尔诺贝利事故是压垮苏联的两个主要因素之一（切尔诺贝利+阿富汗）。这也许是真的，至少它对这一马克思主义恐怖政权的垮台做出了重大贡献。但如果这根本不是一场事故，而是一个成功渗透了该核电站的乌克兰民族主义军事细胞的行为呢？这些勇敢的反马克思主义行动人员可能渗透了该设施并伪装成核电站工人。他们破坏了该设施并导致了事故，清楚地知道这会给苏联造成多大的损害。

尽管这种情况似乎牵强，但很难完全排除这种可能性。显然，承认这样一场"军事"失败不符合苏联的利益，而承担这次袭击的责任也不符合乌克兰武装民族主义运动的利益。因为承担如此毁灭性袭击的责任会促使苏联暴君彻底消灭所有乌克兰民族主义者和爱国同情者。苏联屠杀了多少百万人？4000万到6000万？如果不是因为这个勇敢的乌克兰民族主义细胞，它还会屠杀多少更多的人？

他们造成了巨大的平民苦难，数万人死于癌症（尽管大多数是由于疏散的严重延迟造成的，不能归咎于该细胞）。然而，与此同时，他们可能实际上将数百万人从必然的死亡中拯救出来。

**道德上是否正当？**

这也提出了一个重要问题。发起如此规模的袭击以结束一个致力于消灭一切欧洲事物的种族灭绝邪恶政权，在道德上是正当的吗？重要的是要理解我们，欧洲民族抵抗运动，对人类生命拥有多大的权力。正义骑士在基于务实的评估和考虑做出公正选择方面承担着巨大的责任。正义骑士绝不能盲目行动，也不能仅仅基于仇恨行动。仇恨会驱使你做出最不成比例的行为，导致过度和不必要的苦难。

必须做彻底的务实估计，以便我们作为法官做出决定：

在我们当前政权下，在未来50年：

- 我们有多少姐妹已经被并将被穆斯林强奸？
- 有多少欧洲人将被穆斯林蹂躏、抢劫、殴打、恐吓或杀害？
- 我们有多少兄弟姐妹将因这些暴行而自杀？
- 在欧洲土地上未来的基督教-穆斯林战争中，有多少欧洲人将死亡？数十万，也许数百万？
- 我们能承受对未来基督教-穆斯林战争结果的猜测吗？如果我们输了怎么办？如果我们输了，有多少欧洲人将在欧洲土地上被伊斯兰势力屠杀或奴役？数十万，也许数百万，就像在安纳托利亚或印度一样？
- 有多少爱国主义导向的个人将被我们的多元文化主义压迫者嘲弄和迫害？

请参阅所有指控1-8，了解当前文化马克思主义政权直接或间接犯下的暴行的完整概述。

未来10年预期的穆斯林暴行报告（参见：1960-2010（2020年）针对西欧人的穆斯林暴行）：

- 100万+我们的姐妹被强奸。
- 300-400万+我们的兄弟姐妹被蹂躏、抢劫、殴打、恐吓
- 3万-4万+被直接/间接谋杀（因暴行自杀）

当你将这些数字与未来切尔诺贝利袭击的伤亡报告进行比较时，很明显上述数字将相当于10-50次切尔诺贝利袭击。

**袭击的考虑和评估**

欧洲抵抗运动将必须根据我们使用常规大规模毁灭性武器（WMDs）取得的思想成功，密切重新评估我们未来的攻击方法。

如果我们未来的大多数细胞被欧罗巴苏维埃社会主义共和国特工消灭，或者他们以某种方式设法通过严厉的监控法律安抚我们怎么办？如果他们成功地镇压和迫害我们怎么办？根据我们使用常规武器的成功或失败，我们可能最终陷入这样一种局面，即我们必须考虑使用严厉措施以在经济上摧毁我们的敌人。务实的作战推理涉及使用能够以最小的资源投入（每欧元、每小时、需要最少牺牲的殉道者数量）产生最大效果的方法。

如果我们未能成功地从他们那里获得让步，并且他们继续对欧洲人进行间接的持续种族灭绝，我们将不得不考虑使用过去被认为过于极端的方法。

切尔诺贝利灾难压垮了苏联。一场新的切尔诺贝利灾难在西欧的心脏地带将压垮欧罗巴苏维埃社会主义共和国，不要怀疑。在一个多元文化主义政权上造成一场新的切尔诺贝利灾难将彻底削弱他们，并可能最终导致欧盟或任何特定目标政权的崩溃。然而，它可能会导致大规模的平民苦难和/或数十万甚至数百万的境内流离失所者（难民）。在欧盟心脏地带的一场新的切尔诺贝利灾难不会像乌克兰/白俄罗斯的那场那样具有毁灭性（平民苦难）。西欧政权不会等一周再通知和疏散公众，因此受影响的人数将显著减少（估计少于1万-5万）。

可能有必要做出这样的牺牲，以结束这种实验性的多元文化主义噩梦和种族灭绝。有时，形势要求我们残酷无情，杀死有限数量的平民（冲击/破坏袭击中的间接伤亡），以防止对我们的民族造成更严重的长期暴行。事实上，从务实角度看，当替代方案更糟时，不以残酷的方式行动才是不人道的。只需关注可以在道德上正当化的目标，并接受远远不超过替代性未来暴行的伤亡数字。

数百万人的重新安置、大规模建造新的住房单元、对多达数百万人的财政补偿将使任何多元文化主义欧洲国家破产，并将迫使他们停止所有穆斯林移民。此外，这将导致该政权削减所有少数群体和外国援助的资金，转而专注于重建。成本将是天文数字，以今天的货币计算高达1000亿至5000亿欧元，足以让任何政权破产。该政权将别无选择，只能大幅削减其过度膨胀的公共部门，导致数十万人失业。所有西欧国家将被立即停止所有穆斯林移民，因为他们被迫帮助欧洲内部的"难民"。

已经在受苦的人们最终将从阻止他们站起来为信仰而战的思想枷锁（政治正确）中解脱出来。为理想而战将涉及苦难。然而，在这一点上，苦难已经普遍存在，所以他们没有什么可失去的了。这些条件将有利于进一步巩固和创建爱国武装民兵。最终，军事政变的条件将具备，我们将充分利用这一机会窗口。

如果我们在第一阶段成功夺取至少一个西欧国家的政治和军事控制权（由于核电站袭击导致的政权崩溃），我们可能实际上在穆斯林成为多数之前将所有穆斯林驱逐出境，从而可能将数百万欧洲人的生命从当任何特定国家的穆斯林成为主导力量并变得过于自信时几乎肯定会等待着我们的战争中拯救出来。让1000-10000名欧洲人遭受残酷的癌症死亡（在第一阶段）实际上可能是我们所做的最人道的行动，如果他们的牺牲有助于防止导致数百万人死亡的血腥基督教-穆斯林内战（在第三阶段）。而如果我们在第三阶段输掉了基督教-穆斯林内战呢？最坏的情况将是兴都库什大屠杀（8000万印度教徒被屠杀，1亿人被奴役）的重演，导致一旦穆斯林有一天统治我们，同等数量的欧洲人被屠杀和奴役。
