# 3.59 Radiological Dispersal Devices, RDDs; creating, deploying and detonating radiological bombs in Western European capitals

Orginal: 1058-1063

**3.59 放射性散布装置（RDD）：在西欧首都制造、部署和引爆放射性炸弹**

"放射性炸弹"（脏弹）一词指的是一种将放射性材料与常规炸药结合的放射性武器。虽然放射性散布装置（RDD）将被设计为将放射性材料散布到大面积区域，但使用常规炸药的炸弹可能比放射性材料产生更直接的致命效果。在最可能来源产生的水平上，不会有足够的辐射存在来导致严重疾病或死亡。美国能源部进行的试验性爆炸和后续计算发现，假设不采取任何措施清理受影响区域，并且每个人在受影响区域停留一年，辐射暴露将是"相当高的"，但不会致命。最近对切尔诺贝利灾难沉降物的分析证实了这一点，表明对周边地区许多人（尽管不是那些在近距离的人）的影响几乎可以忽略不计。

由于脏弹不太可能造成许多死亡，许多人并不认为这是一种大规模毁灭性武器。其目的是通过大规模恐慌和恐怖造成心理而非物理伤害。由于这个原因，脏弹有时被称为"大规模破坏性武器"。此外，遏制和去污数千名受害者，以及受影响区域的去污将需要相当多的时间和费用，使区域部分无法使用，并造成毁灭性的经济损失。

历史上该术语也曾用来指代某些类型的核武器。由于早期核武器的效率低下，在爆炸过程中只有少量核材料被消耗。"小男孩"的效率仅为1.4%。"胖子"使用了不同的设计和不同的裂变材料，效率为14%。因此，它们倾向于散布大量未使用的裂变材料，以及以核沉降物形式的裂变产物，这些产物平均而言更加危险。在1950年代，关于能否生产"清洁"炸弹存在相当大的争论，这些通常与"脏"弹形成对比。"清洁"炸弹通常是一个既定目标，科学家和管理人员表示，高效率核武器设计可以产生几乎全部以核聚变形式释放能量的爆炸，这种形式不会产生有害的裂变产物。

但1954年的Castle Bravo事故，其中一枚热核武器产生了大量沉降物并散布在人类群体中，表明这并不是现代热核武器实际使用的东西，现代热核武器大约一半的当量来自最终裂变阶段。虽然一些人提议生产"清洁"武器，但其他理论家指出，人们可以通过用某种材料对核武器进行"加盐"处理，故意使其变得"肮脏"，这种材料在被武器核心辐照时会产生大量持久的沉降物。这些被称为加盐炸弹；经常被提到的一个特定子类型是"钴弹"。

**历史上的实际使用**

只有过两例含铯炸弹的案例，且都没有被引爆。两起都涉及针对俄罗斯的全球圣战。第一次放射性恐怖主义尝试于1995年11月由一群车臣圣战分子实施，他们在莫斯科的伊兹梅洛夫斯基公园埋下了一个包裹在炸药中的铯-137源。一名车臣叛军领导人向媒体报警，炸弹从未被激活，该事件仅相当于一次宣传噱头。1998年12月，车臣安全部门宣布了第二次尝试，他们发现了一个装有放射性材料的容器，附着在一个爆炸性地雷上。

还有其他几起案例，包括基地组织在内的一些组织计划在美国引爆放射性炸弹。2002年，一名基地组织细胞行动人员何塞·帕迪利亚（又名阿卜杜拉·穆哈吉尔）被捕，原因是他所在细胞的指挥官阿布·祖巴伊达在审讯中透露了敏感信息。他透露他的细胞接近于制造一枚放射性炸弹。尽管帕迪利亚在被捕时尚未获得放射性材料或炸药，但执法当局发现了证据表明他正在侦察可用的放射性材料和可能的引爆地点。大约有5个类似的故事涉及基地组织，但也涉及两个欧洲民族主义细胞（一个乌克兰和一个欧洲-美国人）。

**PCCTS，圣殿骑士团与放射性武器**

PCCTS，圣殿骑士团打算将放射性武器纳入我们的武器库，因为它们相当容易制造，对于那些具有基本知识的个人来说也相对容易获取。然而，我们不打算在给予西欧犯罪多元文化主义政权的投降最后期限（即2020年1月1日）之前引爆放射性武器。然而，获取足够铯和其他放射性组件的准备工作应立即开始，以便我们在最后期限后处于有利位置实施攻击。我们的放射性攻击（RDD）将造成最小甚至没有平民伤亡，但将对目标文化马克思主义/多元文化主义政权造成毁灭性的思想、心理和经济损害。

**处理放射性材料时的防护**

不要怀疑；使用保护系统保护操作者免受辐射的成本和复杂性是不现实的。因此，我们的目标是使用防护系统（危险品防护服、临时制造且相对便宜的铅容器），使炸弹的制造者/运输者能够存活足够长时间以成功部署和引爆它。确保在行动开始前准备好抗辐射病的药物。这些药物将治疗受损器官，减轻症状并控制疼痛。去污是尽可能多地去除外部放射性颗粒。脱掉衣服和鞋子可以消除约90%的外部污染。用水和肥皂轻轻清洗可以去除皮肤上的额外辐射颗粒。去污防止放射性材料的进一步分布，并降低通过吸入、摄入或开放伤口造成内部污染的风险。

内部污染的治疗包括减少由放射性颗粒对内部器官造成的损害的治疗。这些治疗包括以下内容：

- 碘化钾是一种非放射性形式的碘。因为碘对正常的甲状腺功能至关重要，甲状腺成为身体中碘的"目的地"。如果你内部受到放射性碘（放射性碘）污染，你的甲状腺会像吸收其他形式的碘一样吸收放射性碘。使用碘化钾治疗可以填充甲状腺中的"空缺"并防止放射性碘的吸收。放射性碘最终通过尿液从体内清除。
- 普鲁士蓝，一种染料，与称为铯和铊的放射性元素颗粒结合。放射性颗粒随后通过粪便排出。这种治疗加速了放射性颗粒的消除，减少了细胞可能吸收的辐射量。
- 二乙烯三胺五乙酸（DTPA）是一种与金属结合的物质。DTPA与放射性元素钚、镅和锔的颗粒结合。放射性颗粒通过尿液排出体外，从而减少吸收的辐射量。

**支持性治疗**

如果你患有辐射病，你应该服用额外的药物或采取干预措施来治疗：

- 细菌感染
- 头痛
- 发烧
- 腹泻
- 恶心和呕吐
- 脱水

长期暴露于辐射可能会损害你的骨髓。然而，任何正义骑士不太可能长时间暴露于辐射，除非他在几个月的时间里寻找放射源，所以骨髓受损的情况是不现实的。在这种情况下如果这是个问题，一种叫做粒细胞集落刺激因子的蛋白质，它能促进白细胞生长，可能对抗辐射病对骨髓的影响。使用这种基于蛋白质的药物治疗，包括非格司亭（Neupogen）和培非格司亭（Neulasta），可能增加白细胞产生并帮助预防后续感染。

**临终准备**

吸收了高剂量辐射（8戈瑞或更高）的正义骑士几乎没有康复的机会，将会死亡。根据疾病的严重程度，死亡可能在两天内或两到三周内发生。接受致命辐射剂量的正义骑士应确保他有药物来控制疼痛、恶心、呕吐和腹泻。如果可能的话，他还应该接受同伴骑士的心理鼓励。

**危险品防护服和铅容器**

处理放射性材料时要极其小心。我们可以学习的一个例子是1987年9月至1988年3月间发生在巴西戈亚尼亚的放射性事故：两名金属拾荒者闯入一家废弃的放射治疗诊所，取出了一个含有粉末状铯-137、活度为50 T Bq的远程治疗源胶囊。他们将其带回家中，准备拆开作废金属出售。当天晚些时候，两人都表现出急性辐射病的迹象，伴有呕吐，其中一人手肿胀并腹泻。几天后，其中一人刺穿了胶囊1毫米厚的窗户，使氯化铯粉末泄漏出来，当他意识到粉末在黑暗中发出蓝色光芒时，将其带回家里向家人和朋友炫耀。经过2周的接触污染传播，导致越来越多的不良健康影响，正确的急性辐射病诊断在医院做出，适当的预防措施可以投入程序。到此时，249人受到污染，151人表现出外部和内部污染，其中20人病情严重，5人死亡。

戈亚尼亚事件在某种程度上预测了当爆炸散布放射性材料时的污染模式，但也显示了即使是非常少量的摄入放射性粉末也可能是致命的。目标必须是获取并使用粉状阿尔法发射材料（或等效强度的材料），如果摄入可能会造成严重的健康风险。这也是已故前克格勃间谍亚历山大·利特维年科的案例，他摄入了（无论是吃、喝还是吸入）钋-210。基于阿尔法发射体的"烟雾炸弹"可能很容易与贝塔或伽马发射的脏弹一样危险。

正义骑士指挥官或行动人员应采取所需的安全措施，至少提供和使用安全的铅容器。我还没有研究便携式抗辐射容器，但我们很可能必须自己设计和制造它们，因为我们需要为了运输目的适当地为物品添加伪装效果。这些容器，也许设计成公文包，必须用来防止引起周围环境的警觉。当接近放射源时，你应该装备你的抗辐射危险品防护服并将材料安全地放入你的容器中。当你在安全距离时，你必须脱下你的防护服（并将其放在单独的袋子中运输），以避免引起周围环境的危险信号。

**获取放射性炸弹的材料**

在评估获取放射性材料的来源时，一个好的标准应该是关注活度潜力至少超过5 T Bq的材料。2010年发布的最新政府安全概览估计，超过2000吨放射性材料在世界各地最奇怪的地方处于无安全保障状态。材料在数千个地点处于无保护状态，例如俄罗斯灯塔、废弃的放射治疗诊所、国家工业或军事倾倒场、废弃的钚矿、废弃的军用船只、沉没的军用船只（潜水员可获得的材料）等。

为了制造和引爆放射性炸弹，我们必须通过盗窃或通过合法或非法渠道购买来获取放射性材料。可能的RDD材料可能来自于世界范围内在工业、医疗目的和学术应用（主要用于研究）中使用的数百万个放射源。在这些源中，只有九种反应堆产生的同位素突出地适合用于放射性恐怖主义：

- 镅-241（广泛用于烟雾探测器，每个1毫克）
- 锎-252
- 铯-137
- 钴-60
- 铱-192
- 钚-238, 239
- 钋-210
- 镭-226
- 锶-90

在这些材料中，镭-226和钋-210被认为是最不具强度的。在这些源中，美国核管理委员会估计在美国，每年大约每天有一个源丢失、被遗弃或被盗。在欧盟，年度估计为70个。存在数千个这样的"孤儿"源散布在世界各地，但在那些报告丢失的源中，估计不超过20%被分类为如果用于RDD则具有潜在的高安全关切。特别是俄罗斯被认为藏有数千个孤儿源，这些源是在苏联解体后丢失的。其中大量但未知数量的源可能属于我们的俄罗斯民族主义兄弟或同情者或俄罗斯黑手党网络。值得注意的是俄罗斯非常强的贝塔发射锶-90源，被用作偏远地区灯塔信标的放射性同位素热电发电机。这些最近（2009年）已被俄罗斯当局保护起来，但许多在这次行动之前被偷盗或"遗失"。这样的发射器在前苏联加盟共和国可能仍然广泛可用。2001年12月，三名格鲁吉亚伐木工偶然发现了这样一个发电机，并将其拖回营地用作热源。几小时内他们就患上了急性辐射病并寻求医院治疗。国际原子能机构（IAEA）后来表示，它含有的锶量如果用于RDD将构成巨大风险。

应该认为，如果你至少拥有关于适当来源的最低信息，至少获取其中一种材料是相对容易的。因此，任何试图获取放射性材料的正义骑士细胞很可能会成功。我们，PCCTS，圣殿骑士团或任何同盟的解放组织，很可能通过"黑市"或我们自己直接去获取来源来获得放射性材料。由于我们等级结构的性质，我们不太可能负担得起从犯罪组织购买它，因为价格实在太高。无论如何，黑市仍然是获取的一个可行选择。从1996年到2004年，非法贩运放射源的事件稳步增加。然而，这些记录的贩运事件主要涉及重新发现的孤儿源，没有任何犯罪活动与爱国解放运动相关的迹象。

**正义骑士细胞面临的挑战和考虑**

除了获取可用的放射性材料的任务之外，还有几项要求或考虑我们必须评估：

1. 源应该"足够"具有放射性，能在爆炸处造成直接的放射性损害，或至少造成社会损害或破坏。
2. 源应该可运输，具有足够的屏蔽来保护携带者，但又不能过重而无法操作。将足够的材料装载到卡车上，连同一个小到中型化肥炸弹似乎是最务实的方法。
3. 放射性材料在引爆期间必须充分散布，以有效污染爆炸周围的区域。需要制定一个稳固的散布策略，应该包括例如设计带有小内腔（含有放射性材料）的金属/钢制成的空心/浸泡弹丸作为"载体"。金属弹片，或者也许是特定设计的带有小内腔（含有放射性材料）的钢球，要么设计为在撞击位置散布，要么是专门设计的金属球/弹片用于从引爆位置到撞击位置的空中散布。如果设计正确，球内的材料将根据弹丸的速度与设计尺寸和气压的关系进行散布。如果你用数千个这样的金属载体弹片/球覆盖炸弹，取决于引爆位置，它可能污染多达3平方公里的区域。如果你添加足够的这些球，将导致一种情景，即它实际上会在城市的大部分地区"下放射性球雨"，加剧思想、心理和经济破坏。
4. 合适的地点：西欧首都，最好靠近或位于最受欢迎的市中心旅游景点/国会大厦/市政厅或任何具有象征/经济价值的位置，以造成最大的思想、心理和经济损害。想象一下，如果犯罪法国文化马克思主义/多元文化主义政权不得不放弃埃菲尔铁塔并在其周围建立一个1公里的封闭区，持续几十年，这将对它造成多大的思想、心理和经济损害。对政权造成的总损害转化为经济损失将高达数千亿欧元，并可能导致连锁反应导致政权垮台。

最佳情景的一个例子是当一个正义骑士细胞（1-3人）拥有高放射性材料源，例如锶-90热发电机，能够造成与切尔诺贝利事故相当的事件。使用这样的源引爆放射性炸弹将是最优的，并不会证明太困难或不现实。这些灯塔热发电机直到最近在俄罗斯都是容易获得的，在前苏联加盟共和国仍然可以获得。我们将在安全地点使用防护装备组装炸弹，并通过卡车将炸弹运输到引爆地点。参与这次行动的正义骑士在组装和运输过程中将遭受严重辐射暴露。然而，暴露不会立即致命，参与的正义骑士将能够在提供的药物帮助下正常运作直到引爆。在运输期间，装置必须密封或至少用大型铅胶囊/罩覆盖，以防止对运输它的骑士造成过多的辐射。然而值得注意的是，有效地屏蔽源将使其几乎不可能运输，并且如果引爆则效果大打折扣。因此，参与组装和/或运输的正义骑士（1-2人）在牺牲自己，将作为殉道者为事业而死。

**考虑替代散布方法**

代替爆炸或在爆炸之外，为了有效散布的目的，应该考虑简单地使用汽油/橡胶混合物作为主要散布方法。在这种情况下，一名正义骑士必须走到卡车后面并移除铅密封壳（防护措施），确保所有放射性材料都没有屏蔽。然后他将不得不通过远程引爆或通过死人开关点燃5-20桶汽油/橡胶混合物。在那一刻，由于严重辐射中毒，他将没有长期生存的机会。因此，最有效的方法是如果他使用远程引爆并建立一个靠近火灾的狙击哨位，从而有效防止消防队在长达60分钟的时间内扑灭火灾（直到他被系统保护者狙击手消灭）。火灾将产生显著的放射性烟雾（放射性颗粒随烟雾携带），由此产生的烟雾云将作为"散布装置"，将高放射性颗粒输送到数百公里外，并可能污染邻国，从而引发国际危机。
