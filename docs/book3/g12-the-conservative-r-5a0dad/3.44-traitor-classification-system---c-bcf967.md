## 3.44 叛徒分类体系——A类、B类和C类叛徒

"这个国家被分裂了，一半是爱国者，一半是叛徒，没有人能区分谁是谁。"
——马克·吐温

此分类系统用于识别各种个人的文化马克思主义/多元文化主义叛徒。该系统的目的是更容易识别优先目标，并且还将作为未来"纽伦堡审判"的基础，一旦欧洲文化保守派重新掌控任何给定国家的政治和军事控制权。

任何A类、B类或C类叛徒都是故意利用其影响力的个人，其行为使其间接或直接犯有本文件中指定的指控：第1-8项。这些个人中的许多人将试图声称对他们被指控的罪行"不知情"。

**A类叛徒**
- 政治领袖（包括NGO领袖）
- 媒体领袖（总编辑）
- 文化领袖
- 行业领袖

A类叛徒通常是任何现任国家元首、部长/参议员、某些组织/委员会等的负责人和领导人，他们犯有指控1-8项。A类叛徒由最有影响力和最高调的叛徒组成。

每100万公民中10人。
惩罚：死刑和没收财产/资金。

**B类叛徒**
B类叛徒是文化马克思主义/多元文化主义政治家，主要来自被称为"MA 100"的欧洲政党联盟（支持多元文化主义的政党）和欧盟议员。他们可以是当选和未当选的议员，他们的顾问，以及任何曾经和仍然直接或间接参与实施以下行为的公共和/或企业公务员。

B类叛徒也可以是来自各种职业群体（但不限于此）的个人：记者、编辑、教师、讲师、大学教授、各种学校/大学董事会成员、公关人员、广播评论员、小说作家、漫画家和艺术家/名人等。他们也可以是来自其他职业群体的个人，如：技术人员、科学家、医生甚至教会领袖。此外，直接或间接资助相关活动的个人（投资者等）。重要的是要注意，刻板的"社会主义者"、集体主义者、女权主义者、同性恋和残障活动家、动物权利活动家、环保主义者等仅按个案考虑。并非每个与这些群体或运动之一相关联的人都被视为文化马克思主义者/多元文化主义者。

前A类叛徒；国家元首、部长/参议员等，某些组织/委员会等的负责人和领导人可以因实际目标原因被重新分类为B类叛徒（他们已失去影响力，不会产生与当前A类叛徒相同的目标价值/效果）。

某些ANTIFA领袖或与ANTIFA运动相关的组织者（以及其他专注成员）被视为B类叛徒。非核心成员被视为C类叛徒。许多专业人士，例如记者、有影响力的社会学家或大学教授等，被视为并归类为B类叛徒，因为我们认为他们是政治活动家而不仅仅是专业人士。他们当然会声称不知情，并声明他们是非政治的。这种策略可能对他们有效，直到他们被正义骑士拜访的那一天——他们的法官、陪审团和刽子手。

每100万公民中1000人。
惩罚：死刑和没收财产/资金。在某些情况下惩罚可减轻。

**C类叛徒**
C类叛徒是影响力较小且优先级较低的目标（通常是为A类和B类叛徒提供便利的个人），但他们仍然犯有指控1-8项。

每100万公民中10000人。
惩罚：罚款、监禁、没收（在涉及大规模杀伤性武器的较大行动中被视为可接受的间接伤亡）。

**D类个人**
D类个人几乎没有或没有政治影响力，但通过各种方式为B类和C类叛徒和/或MA100政党/媒体公司提供便利。他们没有犯有指控1-8项，但与犯有这些指控的个人一起工作或为他们工作。此分类在计算/估计涉及大规模杀伤性武器的较大行动中的间接伤亡时具有相关性，因为任何D类个人都不被视为无辜的"平民"，而是被视为次要仆人/提供便利者。

每100万公民中20000-30000人。
惩罚：无（不被视为平民）。
