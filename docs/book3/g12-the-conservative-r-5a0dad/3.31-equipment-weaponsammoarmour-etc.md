# 3.31 装备——武器/弹药/防弹衣等
3.31 Equipment – weapons/ammo/armour etc.
3.31 装备——武器/弹药/防弹衣等
军事装备应根据行动的性质来选择。你最终得到的武器往往不是你想要的，因此必须在给定时间内充分利用可用的装备。然而，你将尽一切努力获取绝对最好的装备。你将依靠你的装备来让你活着足够长的时间来完成你的目标，所以确实要花必要的时间来获取可靠的装备。
城市行动的最佳装备：
进攻/防御武器
- HK416突击步枪，配备"红点"瞄准镜（4个额外长弹匣）
- 替代方案：任何突击步枪（AK 47等）。尽量获取具有防后坐力功能的现代武器。
- 格洛克手枪，配备消音器和激光（2至4个额外长弹匣）
- 3枚破片手榴弹
- 1至2枚震撼手榴弹
- 2个手臂防御装置（刀）
- 防毒面具
弹药（弹匣管理）
突击步枪弹药：1发穿甲弹，1发达姆弹（比率1/1）
手枪弹药：1发穿甲弹，1发达姆弹（比率1/1）
你很可能面对穿防弹衣的特工（防弹衣与你自己的相似），所以你的主要武器应有穿甲弹。铅或其他软合金弹头不是最优选择。获取这些装备中的一些将非常困难，但请参阅后勤章节以获取具体建议。
防弹衣
- 弹道护颈
- 弹道头盔/面罩（带1.5至2厘米厚的弹道面罩）
- 弹道背心（升级版）
- 考虑使用腿部护甲（你可能需要自己制作）
- 考虑使用额外护甲，只要不过多影响你的机动性。
带有刚性面板的护甲提供卓越的保护。顶级SWAT背心带有钢或陶瓷面板，能承受穿甲手枪和步枪子弹。它们重达20公斤以上，比许多其他变体更不舒服，因此你将依靠摩托车/汽车进行运输。
防弹衣有三个认证级别。最常穿戴的IIA级，提供针对所有.22、.25、.32、.380和.38口径手枪弹药的现实保护，针对大多数9mm、.357马格南、.40 S&W、.45 ACP和.44马格南手枪弹药，以及针对000号鹿弹猎枪弹的保护。II级和IIIA级护甲可抵御更大的威胁，包括12号口径猎枪独头弹和"最热"的.44马格南子弹。
防御性防弹盾牌
尝试获取小型警察版盾牌。如果无法在市场上获取，如果你能提供材料，你可以自己制作一个。为了我们的目的（并且假设你有腿部护甲）的最佳尺寸是从50 x 50到70 x 70厘米的任何尺寸，尽管市场上大多数盾牌都更大。你可能需要重新装备把手、携行带（这样你可以背在背上）、警察徽章、盾牌正面的一或两枚10厘米尖刺和/或剃刀片（主要作为威慑，防止人们从背后袭击你）。2个把手应最佳设计成用一只手握持时能稳定盾牌的方式（一个把手在中心），第二个把手在中心右侧，以便在遭受猛烈火力时用另一只手加强。这样的盾牌在采用TotM机动时非常出色（如果被包围，突击侧翼/向敌方位置突进，通常是装甲警车后面的体制保护者位置）。在这种情况下使用盾牌非常有效。重量不应超过5公斤。最佳情况下，盾牌应有一个小窗口，由防弹玻璃/塑料制成，以便透过观察。盾牌的其余部分通常由特殊类型的金属或其他防弹材料制成。在两个位置之间移动时，盾牌挂在背上，仅在多于一名武装体制保护者（警察）接近时才使用。盾牌相当有效，因为大多数西欧国家使用的体制保护者子弹（通常来自MP5类武器的9mm铅弹）无法穿透它。

穿甲弹
16发Kozap子弹是世界上穿甲能力最强的9毫米手枪弹头。特氟龙子弹可穿透大多数警察背心。
硬黄铜子弹对手枪枪管造成异常磨损，这个问题通过用特氟龙涂层子弹来解决。特氟龙涂层对提高穿透力没有作用；它只是减少了对枪管的损害。
主要由铅制成的常规子弹在对付硬目标时通常效果不佳，尤其是在以手枪速度发射时。
至于突击步枪弹药，标准弹药将轻易穿透最常穿戴的防护背心。然而，特氟龙涂层黄铜子弹更好。
近战武器
你应该使用一把常规战斗刀并在你的主要武器（突击武器）上安装刺刀。你也可以考虑获取/建造一个更高级的防御装置。
目前市场上没有为单人城市战设计的有效近战武器。如果被两个人袭击——一个人抓住你的手臂，一个人将你击倒——刀几乎没有价值。
一个建议可以是建造并在你的手臂上（从手腕到肘部）装备剃刀般锋利的尖刺。这将把你的手臂转变为潜在的致命近战武器，并将使个人更难以袭击你（或可能起到威慑作用）。请记住，你必须使装置足够小，以免使你的手臂显得"笨重"或在其他方面显著限制你的机动性或切换/装备或使用主要武器时的精准度。
