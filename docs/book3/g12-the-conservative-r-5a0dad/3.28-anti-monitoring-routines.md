# 3.28 反监控例程
3.28 Anti-monitoring routines
3.28 反监控例程
快速清单
1. 手机。关掉它，否则他们可以在任何地方通过三角定位你的位置。某些手机即使关机甚至取出电池也能被监控（它们有用于某些功能的内部电池）。
2. 上网/电子邮件。把它当作你知道会被监控来对待。即使你使用电子邮件账户的草稿箱与他人通信，若要匿名这也几乎不是一种安全的方法。另一种选择是创建一个更安全的电子邮件账户（hushmail或任何提供更高匿名度的服务），同时使用Tor网络（掩盖你的IP）。Tor的替代品有anonymize.net或Ipredator。这是一个烦人但必要的过程，因为Tor会让你的连接非常慢，你必须确保每次登录电子邮件或每次研究敏感话题时它总是开启的。我通常用Mozilla Firefox配合Tor，用Internet Explorer进行普通浏览。考虑购买一台未注册的笔记本电脑，在家外的LAN站点上网（许多麦当劳或其他服务商店提供免费无线LAN，你可以坐在车里在他们的场所外接入。只需记住，如果你为此目的使用未注册和匿名的笔记本电脑，那么必须记住，你绝不能将其用于可能在以后某个时间点暴露你身份的目的。
3. Facebook，将此渠道当作你知道会被监控来对待。
4. 欧洲收费站。许多欧洲国家/城市有收费站，你需支付少量费用通过。其中一些会拍摄/拍照你的车甚至司机。这因情况而异，但在某些情况下，这些数据将被存档长达10年，允许情报机构回溯追踪活动和移动模式。
5. 旅行。确保有可信的旅行理由（记住关于大会、博物馆、某些机构名称等的细节）或携带从网上打印的宣传册。欧洲主要城市每周总是有很多大会或展览。例如，考虑在旅行前加入特定的Facebook群组——兴趣领域/大会/展览，以增加你"不在场证明"的可信度。
6. 在国外旅行时绝不要使用信用卡，始终使用现金。确保在出发前准备好正确货币，并携带比你估计使用量多20-30%的现金。
7. 使用你的网络。作为一般规则。每涉及一个人，你被捕的几率就会增加100%。除非绝对必要（这应该永远不是必要的），否则不要信任任何人。绝对一切自己做。
8. 如果你处于与炸弹（化肥等）所需组件相关的WMD"获取阶段"中，那么按以下顺序将"获取阶段"分为五个子阶段：
- 初级爆炸物阶段
- 化肥阶段
- 硝基甲烷/喷气燃料/柴油阶段
- 远程引爆器/定时引爆装置阶段
- 运输（租用卡车/货车——你在行动前1周才开始这个子阶段）
在完全完成每个子阶段之前，不要开始单独的阶段。不要同时启动两个或更多阶段！始终从最困难的部分开始——获取TNT/炸药/塞姆汀塑胶炸药或类似物质，然后进入下一阶段。第一个子阶段是获取TNT/炸药。一旦你获取了例如100至500公斤，就将其固定并在一个独特且偏远的地点埋入地下隐藏点。
然后你可以开始化肥阶段。当你获取了足够的化肥（2至5吨），将其固定/密封防潮，并埋在一个安全且不同的地点（最好在地下隐藏点）。该子阶段完成后，你可以开始获取硝基或喷气燃料。如果你无法获取这些物质，那么就用柴油。如果情报机构在任何子阶段期间收到关于你活动的情报，他们将在知道有大量证据可用的时机进行打击。通过将获取阶段分为子阶段，这将使他们即使收到任何关于你的情报，行动也变得困难得多。此外，你应为每个子阶段建立掩护。例如，当你收集化肥时，你应该有一个商业计划和一个可信的掩护，以确认你正在将化肥用于收获。你需要创建一个商业计划，详细描述你将要如何使用化肥（显然，为你的掩护选择一种你知道需要最大化化肥量的作物类型，并在季节初获取，以避免触发"红旗"）。租用/租赁一个小型农场区域（这个农舍也可以作为你的获取总部）将增强你那一个子阶段掩护的可信度。每个子阶段都应建立等效的掩护。例如，在你开始/尝试获取TNT之前，你应该有一个小型矿产勘探公司的商业计划（矿产勘探——爆破/钻探）。关键是你的掩护要足够可信，能够经受住潜在的审查。创建并成立一家公司，创建一个商业计划，制作商业名片，一个网站等。记住，你不需要在这里重新发明轮子。只需利用任何可用的关于该主题的商业计划，因为"矿产勘探"领域有很多可用。在获取相关的各个子阶段中重复使用这些原则，因为这可能是如果有人收到任何关于你的情报时你的"免于入狱的通行证"。创建可信的掩护是"获取管理"的重要组成部分。
