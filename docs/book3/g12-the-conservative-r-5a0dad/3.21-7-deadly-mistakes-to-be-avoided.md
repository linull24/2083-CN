# 3.21 应避免的7个致命错误
3.21 7 Deadly mistakes to be avoided
3.21 应避免的7个致命错误
1. 使用单个或双人细胞系统。欧洲和美国都有若干较大的等级制网络/团体因本可轻易避免的关键错误而被发现和摧毁。典型错误包括与政府监控名单上的个人有关联，在Facebook等平台上添加你团体中的人。即使你仅仅向你认识的人暗示你正在策划什么，你就是将他们牵连入罪（因此间接迫使他们把你供出来）。几年前，美国一个大型军事民族主义网络（约100人组成）被摧毁。有明确证据表明这些个人之间有关联（来自线人、电话记录等的证据），他们都受到了严厉的判决。最理性的做法本应是创建50到100个细胞，并彼此完全切断联系（在攻击前12至24个月）。然而这个团体作为一个大细胞运作，这使他们非常脆弱。一个关键线人就能搞垮整个网络。显然，如果你独自工作，你就对线人/背叛免疫。
2. 第二个错误是缺乏经验。假设敌人愚蠢，低估敌人的情报能力，认为一切都很容易，结果留下可能导致被捕的证据。由于缺乏经验，他也可能高估敌人的足智多谋，认为他们比实际更聪明。被这种假设所愚弄，他变得胆怯并保持不安全和优柔寡断，瘫痪而缺乏胆量。
3. 第三个错误是选择一个受到极其严密保护的个人作为暗杀目标。对一个保护极其严密的个人的12次失败尝试，本可以变成对较小目标的12次成功攻击，处决超过50名主要目标。目标应该是有影响力的媒体人物——多元文化主义政治家、记者/编辑、文化马克思主义教授、马克思主义作家/艺术家、非政府组织领导层、全球主义投资者。显然，要聚焦于那些没有武装保镖的个人。
4. 第四个错误是吹嘘你已经采取或即将采取的行动，并将其广而告之。拥有强烈的使命感是好的，但如果你被过度的自我满足需求所困扰，你很可能将敏感信息信任给那些会出卖你的人。如果你渴望关注和"爱/欣赏/赞美"，你很可能冒不必要的风险，最终导致被监禁。不要通过吹嘘你的成功、行动或与其他爱国者"竞争"来牵涉你的自我。作为审判骑士，你被严格禁止向外界（无论他们是否可能同情你的核心原则）透露敏感信息。我估计大约50%的武装抵抗战士在他们甚至有机会执行任务之前就因完全缺乏谨慎而被捕入狱。
5. 第五个错误是高估你伙伴的力量/信念，并着手进行你目前还缺乏足够装备或能力的行动。
6. 第六个错误是鲁莽行动。你失去耐心，神经发作，不等任何准备，便冲动地投入行动，导致行动失败。
7. 第七个错误是未能彻底规划一切，并自发行动。
