3.22 以恐怖作为唤醒大众的方法——我们中的许多人会为此而憎恨我们

"在变革的时代，爱国者是稀缺的人。他被憎恨和蔑视。然而当他的事业成功时，胆怯者会加入他。因为那时做爱国者不需要付出代价。"

马克·吐温

你可能记得第一部《黑客帝国》电影中的那个角色；这个角色不知何故从他的"培养舱"中被唤醒了，在那里他与一个幻想世界、一个软件程序相连。他不喜欢他现在面临的现实，因为这涉及痛苦和一场对抗机器的、胜算渺茫的战斗。他不想战斗，也不想受苦。他只想重新连接回他的培养舱，在那里他可以重新过上他旧的幻想生活——与现实脱节。这个角色甚至与机器合作，背叛了自己的人民，以换取被重新连接回培养舱的承诺。他根本不愿意面对现实，甚至愿意杀死自己的人民来避免面对它。

在某种程度上，这就是主流欧洲人的缩影。我们想要从正在进行的欧洲种族灭绝中拯救的许多人不想要我们的帮助（或者至少，相信他们不想要我们的帮助）。事实上，他们中的许多人会尽其所能避免面对现实。这往往解释了为什么那些试图警告周围人即将来临的危险的好心人会被谴责、嘲笑甚至迫害。唤醒人民脱离"培养舱"并招募他们加入欧洲抵抗运动，已经成为我们这些勇敢少数人的吃力不讨好的任务。不幸的是，对我们来说，人性往往对我们不利。许多人已经适应了这个灭绝性体制，有些人甚至间接接受了他们的命运。他们接受文化马克思主义/多元文化主义政权正在抹杀欧洲身份、我们的文化、传统甚至我们的民族国家。许多人被洗脑到会用自己的生命来捍卫这种灭绝计划的程度。

为了唤醒大众，唯一理性的方法是确保现行体制内爆。这将造成大量短期痛苦：福利金削减、失业率上升，甚至在极端情况下出现饥荒。我们的大部分人民将拒绝加入抵抗运动，因为他们觉得失去的太多。他们在漫长的教育中投入了数年时间和大量资源，大多数人还有必须偿还的抵押贷款/贷款。许多人接受了政府提供的特殊激励以保持对体制的"忠诚"。所有这些责任和激励导致了受害者（人民）与灭绝者（政权）之间的一种共生关系，达到了一种集体大规模"斯德哥尔摩综合征"（对俘获者的同情和忠诚）的程度。我们的目标是打破这些纽带，这只能通过促使创建一个这样的情景来实现：即我们希望摧毁的反欧洲仇恨意识形态消亡/内爆或因千刀万剐而死。每一次成功的行动都是一道新的伤口，将促成这种
