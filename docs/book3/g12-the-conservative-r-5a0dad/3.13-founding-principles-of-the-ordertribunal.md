# 骑士团/法庭的创立原则
Founding principles of the Order/Tribunal
欧洲军事骑士团和刑事法庭（PCCTS——圣殿骑士团）由欧洲自由原住民创建并为之服务。法庭和骑士团的主要目的之一是试图/促成夺取西欧多元文化主义政权的政治和军事控制，并审判、裁判和惩罚西欧文化马克思主义/多元文化主义罪犯（A类、B类和C类叛徒），因为他们自1955年至今对欧洲原住民犯下了罪行。

Pauperes commilitones Christi Templique Solomonici——PCCTS（基督和所罗门圣殿的贫穷战友），圣殿骑士团于2002年在伦敦由来自八个欧洲国家的代表重建，目的是服务于欧洲自由原住民的利益，并抵抗正在进行的欧洲圣战（称为"第三次圣战"）。圣殿骑士团被重建为一个泛欧洲民族主义军事骑士团和一个军事/刑事法庭，具有两个主要目标。骑士团将作为一个武装的原住民权利组织和一支十字军运动（反圣战运动）服务。

创立会议（两场会议，由4名创团成员和东道主组成，作为安全预防措施）于2002年4月在英国伦敦举行。

3.13 骑士团/法庭的创立原则

骑士团/法庭PCCTS的目的是巩固欧洲人，并通过"开源战争"，通过营销具有自组织功能的去中心化平台，创建和发展一支重要的武装防御力量。这将通过我们持续努力呼吁和要求所有欧洲人接受每个自由、爱国个人所期望的职责来实现。所有自由欧洲人都有权利和义务成为骑士团/法庭的"正义骑士"，目的是：

- 反对所有仇恨意识形态：共产主义（反个人主义）、文化马克思主义/多元文化主义（反欧洲）、伊斯兰教（反卡菲尔/非穆斯林）和国家社会主义（反犹太人）。
- 百年计划：促成夺取目前由反民族主义者（文化马克思主义/多元文化主义政权）控制的西欧国家的政治权力。为所有欧洲人带来自由以及文化和人口上的可持续性
- 服务并保护欧洲原住民和所有其他忠诚和爱国的欧洲个人
- 保护欧洲文化、传统和遗产
- 阻止由文化马克思主义者/多元文化主义者、自杀式人道主义者和资本主义全球主义精英促成的持续的欧洲文化和人口灭绝
- 防止欧洲基督教世界的解构
- 防止进一步的伊斯兰人口战争——不允许穆斯林入侵/殖民欧洲
- 通过发起未来的驱逐运动将伊斯兰教从欧洲击退
- 对西欧文化马克思主义/多元文化主义/全球主义罪犯执行惩罚，因其对欧洲原住民犯下的罪行
- 使用任何和所有必要手段反对并击败欧洲的多元文化主义联盟（MA 100）
- 支持以色列对抗圣战的斗争
- 将中东基督徒从伊斯兰统治中解放出来
- 为事业殉道的意愿、运用谨慎判断、自愿贫穷和对原则的虔诚服从等美德是所有正义骑士的强制性要求
- 尊重我们祖先的愿望和记忆，他们过去保障了欧洲的安全和繁荣
- 遵循PCCTS的座右铭——"殉道而非顺服"（Martyrdom before Dhimmitude）。
