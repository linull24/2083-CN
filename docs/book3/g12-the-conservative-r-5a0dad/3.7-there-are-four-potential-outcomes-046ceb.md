# 第二阶段和第三阶段有四种可能的结果

There are four potential outcomes of Phase 2 and 3
3.7 第二阶段和第三阶段有四种可能的结果

1. 多元文化主义联盟可能试图在欧洲内部创建几个"迷你巴基斯坦"（授予穆斯林领土/自治权）。必须不惜一切代价阻止这种情况。
   > 对于少民的问题如此
2. 伊斯兰教的改革，产生一个世俗化和无力的版本，拒绝大部分伊斯兰教法（Sharia）、古兰经废除（Naskh）的概念、塔基亚（Al-Taqiyya）、圣战和每一个"不宽容"的苏拉和圣训。这种情况非常不可能，因为它只会得到世俗穆斯林的支持。
3. 政变——欧洲抵抗运动在国家军队的协助下，将成功推翻西欧的文化马克思主义/多元文化主义政权，导致穆斯林被驱逐（相对较少的伤亡）。这种情况是首选结果。
4. 内战（文化保守主义者对文化马克思主义者和穆斯林力量）——导致双方犯下暴行（黎巴嫩/塞尔维亚变体）。种族灭绝或无节制的屠杀等暴行可能在情景1、2、3或4中出现。许多居民将逃往美国、俄罗斯、澳大利亚或新西兰。

上述情景之一很可能在第二阶段结束前（2030-2070）或在第三阶段发生。
