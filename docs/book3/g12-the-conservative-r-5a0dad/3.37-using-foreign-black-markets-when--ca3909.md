# 3.37 Using foreign "black markets" when acquiring equipment
Orginal: 908-909

在尝试获取武器/护甲/爆炸物时，有两种不同的方法可以使用。利用其他欧洲民族主义组织或利用犯罪网络。最务实的方法可能是利用后者。原因是许多民族主义组织受到国家情报机构的严密监视。你想采取哪种方法完全取决于你。在追求这一任务时，要做好面对一些失败的准备（如果你没有任何联系人）。

结束注释：通过Ebay和类似服务购买某些护甲是可能的。但要记住，通常不允许出售"完整护甲"，所以你必须从一家供应商购买携行具（无插板的空背心），从另一家供应商购买插板（金属、陶瓷或Honeywell Goldflex插板）。这些插板通常以25 x 30厘米出售。如果可能的话，你应该尝试获取更大的，并且你应该至少选择IIIA级插板（防护大多数威胁，但步枪射击及以上除外）或III级插板（防护步枪射击）。有很多选择。创伤板（放置在弹道插板后），甚至弹道盾牌和带面罩的弹道头盔也偶尔有售。只需在一段时间内彻底检查，你应该能够找到新物品和二手物品。你也可以使用某些雇佣兵商店，它们应提供许多（如果不是全部）物品。你可能必须创建一个伪装才能使用某些供应商。你可能必须通过重新设计现有的二手物品并结合弹道插板来设计你自己的护甲（你应该有超过2块插板的更多保护，应包括腿部、肩部、手臂、颈部保护）。由于我还没有开始武器/护甲获取阶段，此时我无法提供更多见解。通过邮政服务订购时要非常小心，因为在最坏的情况下，他们可能在交付时检查包裹并向当局发送警告。

## 3.37 获取装备时使用外国"黑市"

前往任何西欧或东欧主要城市。例如，如果你是法国公民，请记住你想避免与你自己城市的任何团体互动，因为当地警察部队可能能够识别你并与监视名单交叉比对。你的国家警察部队在犯罪网络和/或民族主义组织中拥有许多线人，所以要避免将你自己的国家作为获取装备的基地。你可能还要考虑避免邻国（当然取决于你的原籍国），因为个人可能识别你的国籍或身份。如果你的国籍被揭露并且信息传递给当地警察部门，他们可能会被识别出来（通过使用海关信息/航空公司等）。如果可能的话，在你的国家购买假身份证（护照、银行ID或驾照），并在需要欺骗国外任何人时使用它。永远不要在任何武器交易中同时携带两种身份证件。绝不要乘飞机旅行。在你自己国家租一辆车，开车前往目的地。

**利用犯罪网络**
谨慎接触犯罪网络，尝试找到当地的黑市军火商。将自己呈现为枪支收藏家。你可能想为实际交易使用中间人/稻草人（当地毒贩或任何合适的个人，以固定费用或百分比费用补偿他。尝试完成一次"测试交易"，以验证你的中间人的信任程度。为最坏情况做好准备，不要携带大量现金（许多人将没有什么可提供的，会试图抢劫、诈骗你）。不要透露你的政治信念，因为许多欧洲罪犯是阿尔巴尼亚穆斯林。犯罪网络通常比民族主义组织有更好的装备出售。如果可能的话，尽量避免穆斯林犯罪组织。

例如，进行4次单独交易：
1. 格洛克17 + 消音器
2. 突击步枪
3. 特殊弹药（穿甲/达姆弹）
4. 分裂手榴弹/震荡手榴弹

**利用民族主义组织/个人**
很多时候，这些人中有一个愿意代表你充当中间人（如果他部分或完全支持你的政治意识形态）。显然，不要告诉任何潜在的NS你亲以色列等，因为他可能视你为敌人。

**出国旅行时的住宿和掩护故事**
首先，在访问任何特定的欧洲目标城市之前，要有一个有效的借口（掩护故事）。最好参加某种可能使你的逗留合法化的会议（出于事后/现实原因）。

使用你的假护照/身份证租用酒店或公寓。当被公共代理人（包括海关）接近时使用你的真实身份证。

**在线购买非法配件**
绝不要使用你的真实地址或姓名在线购买或订购。如果海关截停货物，可能会产生疑问，你有可能被列入"安全监视名单"，这可能在以后危害你。

不要在任何交易中使用电子邮件、手机或任何技术设备（因为它们太容易被监控）。使用人对人的方式或中间人。

**获取爆炸物材料**
将需要收集用于摧毁建筑物或其他固定目标的爆破材料。

大型爆炸装置（2000-5000公斤）使用高等级化肥（硝酸铵）与柴油（或更强效的燃料如喷气燃料，或最佳的硝基甲烷）混合制成；当今的首选炸药是semtex、炸药、TNT、PETN RDX、PETN（季戊四醇四硝酸酯）、过氧化氢、NM/HNO3、APAN。
