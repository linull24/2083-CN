# 3.51 防御与攻击方法——战场策略
[Original English title: 3.51 Defence and Attack Methods - Field Strategies]
如果大规模杀伤性武器获取阶段失败，则进行基本的冲击行动。

始终先获取武器和装甲，并将其保存在储藏处。只有在你完成之后，才开始获取大规模杀伤性武器的组件。原因如下：如果大规模杀伤性武器获取阶段失败，导致你被捕，你可能基于现有证据被短期监禁。一旦你出来，你将受到严密监控，并将很难获取任何大规模杀伤性武器所需的组件。到那个时候，你仍然可以使用你的武器/装甲储藏，从而能够执行一项不涉及大规模杀伤性武器的较小冲击袭击。根据若干因素，估计处决人数仍可能高达50人。

暗杀——准备和装备

创建一个隐藏的移动哨点，可以俯瞰目标住所。这将使你能够在不引起邻居怀疑的情况下监控目标的行动。哨点将使你能够绘制个人的行动模式，从而使你能够预期合适的袭击窗口。购买一个防水/防潮摄像头，带有实时传输到附近接收器/显示器的功能。你的基地通常将是距离一定距离的一辆卡车，以避免被发现或引起任何不必要的注意。在住所周围徘徊是容易被抓住的好方法，因为你会惊动邻居。

3.51 防御与攻击方法——战场策略

在常规战争中，战斗通常是以远程武器在一定距离进行的。在非常规战争中，其中包括城市游击战，战斗是近距离的，而且往往非常近。为了防止自己死亡，城市游击队员必须先开枪，而且在有射击机会时不能犹豫。他不能浪费弹药，因为他并不拥有大量弹药，所以必须节约使用。他也不能迅速补充弹药，因为他不是团队的一部分，在这个团队中每个审判骑士都必须能够照顾自己。城市游击队员不能浪费时间，因此必须能够立即射击。

当与敌人面对面时，他必须总是从一个位置移动到另一个位置，因为停留在一个地方会使他成为固定目标，因此非常脆弱。

游击战意味着战士必须明智地选择目标，按照自己的条件战斗，并知道何时撤退。圣殿骑士必须学会欺骗敌人，并知道何时使用重要的震慑战术。

审判骑士必须准备好快速移动和快速思考。他必须警觉，身体健康，不依赖酒精或其他损害性物质。然而，麻黄碱+咖啡因组合（Stack）和类似物质会增强你的感官，毫无疑问在任何行动中都将证明是有价值的。

我们将模拟和讨论几种"反警察部队"和"反突击部队"策略。

在有城市街区或类似地形的城市区域，基本的警察部队策略将是试图从两侧包抄你并将你固定在一个位置。这种基本策略是他们试图获得足够的时间，以便专业突击部队能够到达该位置。一旦专业突击部队到达那里，他们将试图部署"突击部队狙击手"。如果是这种情况，将极其危险，战略性撤退将非常困难。必须通过遵循预定的时间框架，不惜一切代价避免被"压制"。你必须迅速行动，以侵略性和高效的方式对付现场的第一批警察，然后继续前进。

你被夹在城市街区的两个出口之间

如果他们尽管一切困难仍然成功地执行了经典的包抄机动，以下是适当的应对方式：

通过从两侧包抄你，他们被迫分裂了他们的突击部队（或常规警察巡逻队）。仔细注意实际突击部队特工（SFA）和常规警察（RPO）之间的区别。你应该在行动前就能够分辨出区别。你想从抵抗力弱的地方突破。你有几秒钟通过观察两侧来评估这一点。

用车辆突破侧翼

你显然会尝试"征用"一辆车，并穿过RPO侧翼（如果可能，不使用武力）。如果车辆不可用，你将不得不攻击RPO侧翼以突破。他们会试图射击你车辆的轮胎，采取必要措施避免碰撞或不必要的损坏。

徒步突破侧翼

利用地形（汽车等）作为掩护，冲向侧翼。在这一点上，他们没有狙击手，有限的防弹衣，只有标准配发警用手枪，因此他们明显处于劣势。做好无情且毫不犹豫地逐个打击（双发爆头）一个接一个目标的准备。在这一点上，他们不应该能够击中你，因为你的掩护和/或装甲应该能够抵挡任何来袭弹丸（特别是如果你有5公斤的小型防弹盾牌）。他们会因为你的"向前猛冲"而完全惊讶和困惑，并寻找掩护或惊恐地试图逃跑。

烈士冲锋（TotM）

突破侧翼是一个危险的举动。然而，请记住，他们不会预料到你会如此侵略性地冲击他们（瞄准头部射击），所以你拥有突袭优势。让我们把这种策略称为："烈士冲锋"，因为攻击的严重性和危急情况。这是一个致命的举动，要么你或你的对手将死亡或受伤。根据你的优势和他们劣势，如果面对2名特工（场景1），成功率约为75%。然而，面对4名特工，即使考虑到你的信念、优越的装甲和火力，成功率将下降到30-45%（场景2）。如果你允许他们有足够的时间在任何场景中部署1-2名狙击手，你将不太可能活着离开，生存率低于20%。然而，在正常情况下，你通常只会面对2名警察（每辆车2人），这应该使你能相当容易地消除两者。无论如何，投降不是一个选项。你要么突破自由，要么在尝试中死亡。

最佳情况

由于我们在冲击袭击中严重依赖突袭优势，我们应该有足够的时间来避免这种情况。我们最好的反制措施是迅速执行行动，并在当地警察或特警队能够以协调的方式集结反应或制定反击之前，匆忙进行战术撤退。

征用车辆

根据自己的判断征用平民的车辆。基本上站在路中间，拦下最近的汽车。考虑到你扮演的是政府特工（在你的情况下，是警察），这应该会很顺利。与司机交谈；命令他尽快交出车辆。

如果他拒绝，坚持并用妨碍刑事调查的刑事指控威胁他，并向他提供你的服务编号（每个警察通常都有唯一编号）。如果他仍然拒绝，威胁如果在10秒内不离开他的车就开枪射杀他。如果他仍然拒绝，朝他的头部开枪。这个过程（从你实际拦下汽车到你夺取车辆）不应超过30秒（最好少于15秒）。如果你正在被警察追赶，尽快拦下最近的车辆。使用任何必要手段完成这一目标。

这个操练应在行动前反复模拟。

近战防御策略

在移动时最大的威胁之一是避免和防止来自愤怒的平民或卧底特工的攻击。预期以下场景：

一个平民/卧底特工去抓你的侧臂或抓住你的手臂；另一个平民攻击并将你打倒在地。你该怎么办？

使用侧臂或刀具消除一个目标。瞄准脖子或任何暴露的脆弱区域。一旦你消除了目标1，立即切换到目标2。如果其他所有方法都失败，尝试逃跑。

目前市场上没有为单人城市战设计的有效近战武器。如果你被两个人扑倒，一个人抓住你的手臂，一个人将你打倒，刀具将没什么价值。

一个建议可能是建造并装备锋利的尖刺在你的盾牌和/或手臂上（从手腕到肘部）。这将把你的手臂变成致命的近战武器，并将使别人更难扑倒你（它也可能起到威慑作用）。

3.52 化肥类大规模杀伤性武器与武器禁令

在1990年代早期，制造大型卡车炸弹（500-2000公斤）是一项具有挑战性且相当困难的事业，但实际上对于我们中最聪明、最有动力且采取了必要预防措施以避免被系统保护者情报机构列入黑名单的人来说，是一个可以实现的任务。

时代在变化，提摩西·麦克维先生时期对我们可用的可能性已不复存在。美国/欧苏联的文化马克思主义霸权已经适应，所以我们必须不断适应。自1990年代以来，欧盟使炸药的制造变得相当困难。他们通过监管和禁止制造炸药所需的物质和化合物来实现这一点。举例说明：自1990年代以来，欧盟已经禁止了数百种化肥、杀虫剂和其他广泛可用的化学品，他们通过指示生产商添加中和化学剂使数百种化合物失去活性。此外，他们创建了一个与前苏联非常相似的系统，利用零售柜台服务员作为情报特工。

通过互联网可获得的许多造弹指南现在相当无用，因为这些指南中所需的大多数化学品对大多数人来说非常难以获得甚至不可能获得。20年前，你可以轻松地走进杂货店、园艺中心或药房，在柜台上购买大部分这些材料。当时可以获取材料制作70-100种不同类型的炸药，而现在这个数字减少到10-20种炸药。现在在许多国家，平民很难买到硝酸铵（AN）化肥。在仍销售AN的国家，他们已对分配进行了监管，限制为600公斤袋装。所以当你决定购买一袋时，要准备好一个稳固的掩护故事。此外，硝酸铵化肥正在被尿素化肥取代，尿素化肥在使用前需要经过纯化过程（转化为硝酸尿素），而最终产品具有更不稳定的化学成分。硝酸尿素会迅速分解，通常在30天内，这要求操作者相应地计划。

当然，还有生产AN的替代来源。你可以从简单的家用产品中制造它，或购买冰袋。这在制造小于50公斤的小型炸弹时是合适的。我相信他们很快也会用另一种不同的惰性化合物取代AN基冰袋。但仍有许多机会，而且总会有几种方法可以用来制造有效的炸药。至于常规武器，枪支法律将继续受到限制，以至于以合法方式获取火器将相当困难。
