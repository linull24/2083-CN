# 第一阶段的组织策略：文化保守团体之间更好的沟通与合作
# Organisational strategy for phase 1; Better communication and cooperation among cultural conservative groups
Orginal: 652-655

第一阶段的组织策略：文化保守团体之间更好的沟通与合作

向文化马克思主义者/多元文化主义者学习
很明显，他们所做的不仅仅是搞乱我们的国家；他们还非常刻意地搞乱了我们的思想。

事实证明，搞乱我们的思想不仅仅是计划的一部分；而是整个征服计划（实施欧盟的欧拉比亚项目/欧洲多元文化主义/欧洲伊斯兰化）的基本目标。他们利用社会学、社会心理学、语言学和关于人类动机的微妙理解进入我们的头脑，改变我们处理现实本身的方式，以使其无法质疑他们正在做的所有其他事情。

结束文化马克思主义/多元文化主义的主导地位需要我们消除他们对整整两代欧洲人造成的巨大模因论和本体论损害。我们别无选择，只能用我们自己的火来对抗这场火。我们需要做的第一件事是理解，非常具体地，他们是如何做到的。幸运的是，这并不难：基础知识都陈列在他们最初的书面计划中。

我们必须在文化上赢得人们——通过界定人应当如何行动，他应当如何感知周围的世界，以及过好生活意味着什么。

生命的意义是什么？我们应该如何彼此相处？我们的家庭和社区？其他国家？上帝？地球？什么是善，我们如何识别它？什么是恶，我们应该如何回应？

这些是基本的本体论问题，我们解析现实其余部分的能力取决于这些问题——每个人认知世界模型的基础。改变这些基本假设，我们优先考虑和评估世界其他一切的方式也必然发生变化。我们必须认识到这一点，并专注于通过每一个可能的渠道销售文化保守主义的世界观。我们必须使这成为我们运动的中心焦点。一旦我们让他们接受我们关于现实的基本假设（理性主义），我们就知道，我们议程的其余部分将自然随之而来。

我们必须将自己定位为一种大胆而有争议的反文化，为盛行的文化马克思主义/多元文化主义假设体系提供一种原创和反叛的替代方案。

我们作为一个群众运动的地位始于也终于我们激励大众分享我们世界观的能力。推广这种世界观是唯一重要的目标；我们采取的每一个行动都应该旨在推动我们朝着这个结果前进。当那种认识论被广泛接受时，实施我们的政策将轻松而自然地推进，遇到的阻力最小。

不幸的是，我们是处于弱势地位开始的。文化马克思主义/多元文化主义的理想更加引人注目（情感主义、多元文化主义、人本主义，这些已经引导我们走向正在进行的欧洲伊斯兰化），然而它们对任何文明都是自杀性的，不能被接受或容忍。历史教导我们，所有过去衰落的伟大文明都是自我击败的。文明诞生于理性主义，并败亡于情感主义。我们的目标是通过基本上"重置"当前价值观，用我们"旧的核心、理性和文化保守的价值观"替换它们，来防止我们的文明死亡。在整个欧洲实施我们的原则——这将被认为是第二文化欧洲文艺复兴——只能在获得政治权力并得到军队支持后才能完成。在这成为可能之前，欧洲将再次燃烧，人们会死去，但欧洲将重生，像它曾经那样强大。

欧洲人所做的一切，我们建造的制度和物质基础设施，我们进行的投资和决策，我们设定的目标和我们珍视的理想，都是从我们关于世界如何运作的基本假设中产生并根据这些假设进行评估的。让人们理解并接受保守主义（理性主义、文化保守主义世界观）的基本前提，是在欧洲创建一个持久的进步时代的第一步也是最关键的一步。当这一点实现后，我们可以着手改革社会的每一个机构，使其反映这些价值观，从学校课程开始，并禁止大众媒体中的文化马克思主义宣传。

人们可能渴望改变和一些新想法——但即便如此，我们最好还是不要低估消除他们对人们思维施加的所有限制需要多长时间。我们更聪明的方法是非常积极地推广自己，作为一个新的、新鲜的替代反文化，不怕面对一个陈腐和崩溃的现状。

让欧洲人相信我们值得信赖来领导
我们最大的问题可能是公众（经过几十年的文化马克思主义/多元文化主义洗脑）根本不信任我们来领导。所以我们必须专注于修复这种看法，这个过程可能需要几十年。我们必须说服欧洲各国，文化马克思主义/多元文化主义政权曾经是且现在是自杀性的、腐败的、无能的，完全不值得信赖。如果我们想要一个永久的文化保守主义政权（尽管我们采取了非民主的方法），我们必须努力激励并赢得国家对我们思想和领导的深切信任。我们需要他们长期的忠诚。

将战略与战术对齐……
我们有越来越多出色的、充满活力的、技能娴熟的活动家在那里进行组织和传递信息。我们还有一个规模较小但非常被忽视的、具有大局观的战略思想家核心群体，他们放眼很远的前方，思考我们想去哪里以及如何最好地到达那里。这两个派别不仅很少交谈——当他们交谈时，他们经常发现他们甚至不说同一种语言。活动家轻视战略家想得太大局，不理解地面上的现实。战略家看到活动家们朝着各种方向前进，而不是将他们的精力对齐，聚焦于经过精心选择的小战役，这些小战役将在未来的道路上带来更大的胜利。

重要的是要以大范围的概括来谈论原则、价值观和大规模的世界愿景。这对具有战略思维的人来说是振奋人心的，他们看到同样的大局，他们明白在填补细节之前，你必须创建那种你想要创造的变革的总体愿景。然而，同样的风格让那些更偏向细节的人发疯：他们对那种缺乏细节感到非常不自在。他们不想要大局的东西；他们想要确切知道期望什么。

令人欣慰的是，意识到文化马克思主义者/多元文化主义者一直面临这个完全相同的问题。但这也指出了一个清醒的事实：除非我们也学会如何弥合这一差距，以便最大化这两个群体的技能，否则我们在掌权时不会经历平稳的过渡。我们需要让那些有能力规划长期战略的人与那些拥有执行它的战术技能的人紧密联系起来——双方都需要有智慧去认识和尊重对方为桌子带来的不同但重要的东西。

投资创建精英战术家
没有一个成功的运动能够没有一个紧密团结、值得信赖、训练有素的核心精英活动领袖，他们都在为同一个目标而努力。拥有少数几个充满激情的成员比拥有大量大致冷漠的成员更重要。如果核心是有活力的、聪明的和强大的，其余的自然会围绕着它各就各位。

文化马克思主义者缺乏尊重他们最有经验领导人的传统。认为我们中的任何一个人都能做他们所做的事情是常见的自由主义自大。

另一方面，保守派有尊重并服从我们最有经验领导人的传统。优秀的追随者，没有不断的质疑。有大型达成共识的全员对话的时间；但也有停止说话、服从命令、去做需要做的事情而不背后中伤或质疑决定的时间。这是我们最大的优势。我们相信某种永恒的东西，我们的文化和国家认同，我们许多人愿意为此而死。

文化马克思主义/多元文化主义心态的最大弱点是他们缺乏宏伟蓝图；他们依据短期直觉、情感和短视行动。他们只考虑未来5-10年，而我们考虑未来30-50年。

文化马克思主义者失去了许多优秀的领导人，仅仅因为他们厌倦了试图把所有青蛙都留在手推车里，这让他们把注意力从更重要的任务——把手推车开到它要去的地方——上转移开。这是他们消耗掉自己最有才能的人的最典型方式之一。

我们最大的缺陷是我们不愿意包容。当人们受够了文化马克思主义/多元文化主义文化时，我们需要欢迎他们，让他们在我们中间感到宾至如归。

然而，他们需要自愿来。建立一个运动就是一份销售工作——但当他们接受我们的条件时，交易就完成了，而不是当我们屈从满足他们的条件时。

我们的世界观和原则对整个长期目标绝对核心，永远不应为任何人妥协。如果有人不同意，好吧。接受或离开。我们不会为了吸引更多来自不那么重要子群体的支持者而模糊我们自己的信念。我们知道，随着时间的推移，这些小妥协形成了破坏整个运动的裂缝。

每个人都需要知道我们是一个纯粹防御性的运动。我们没有开始这场导致我们走向文化自杀的战争。是他们，文化马克思主义者/多元文化主义者，开始的。我们只是在做保护欧洲文化所必需的事情，并确保我们不会最终落入沙里亚法之下。

自1960年代以来，文化马克思主义者一直利用国家的权力压制我们的运动，使用他们能够编造的任何借口。不幸的是，这些迫害并没有创造出预期的同情。

我们所面临的战斗与我们的祖先战斗过的相同；我们欠他们的记忆，要打得好、打得狠。

不要低估抵抗
没有任何借口对文化马克思主义/多元文化主义攻击的凶猛或巧妙感到惊讶。

但如果我们要在这场战斗中获胜，我们需要认真地为做出牺牲做好准备，并毫不留情地攻击他们最疼的地方。毕竟，我们害怕他们的远不如他们害怕我们的。

他们被情感主义的枷锁束缚着，这是他们的阿喀琉斯之踵，最终将导致他们的垮台。我们作为理性思考者，不会受到阻止他们采取必要措施的强烈情感主义伦理线的困扰。
