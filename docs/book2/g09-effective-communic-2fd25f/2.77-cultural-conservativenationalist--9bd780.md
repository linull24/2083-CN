# 2.77 文化保守派/民族主义者的修辞策略
# 2.77 Cultural conservative/nationalist rhetorical strategies
Orginal: 661-665

不幸的是，英国国家党（BNP）就是这种情况，他们在几个方面缺乏"声誉盾牌"，缺少次级平台和"未被玷污的"领导层。
例如，一个有着农民党传统的政党将能够利用其声誉盾牌来抵御其他精英行为者指责该党搞种族主义和极端主义的批评。相比之下，旧法西斯政党或全新政党在面对这种批评时就没有这样的声誉盾牌可用。第二，如果一个政党也以推动"其他政策"而闻名，选民可以更容易地向同龄人和社区证明投票给一个提出激进限制移民政策的政党不是支持种族主义或极端主义的行为。第三，当相关政党拥有"声誉盾牌"时，选民可以更容易地向自己证明他们的选择并非出于种族主义动机。
BNP和其他一些此类政党确实需要一个坚实的"次级平台"。例如，作为一个"减税"政党或"法律与秩序"政党。如果他们避免未来犯错（如果他们实施这个议程的话），这将需要几年时间。潜力是存在的。
西欧所有反伊斯兰化政党都应该研究"挪威模式"。进步党，目前是欧洲最成功的反移民政党（www.frp.no），目前在民意调查中享有27%的支持率（09年第一季度），上次选举获得了22%的选票。它拥有"声誉盾牌"，因为它最初是一个"减税"政党，但在90年代初开始反对移民。他们有一个很好的策略，其中"现代修辞"和若干"女性"发言人是其中的一部分。
英国、法国和瑞典尤其陷入了这个泥潭。被玷污的"右翼"领导人（具有种族保守运动背景）间接地阻止了圣战抵抗运动充分发挥其潜力。然而，正如另一章所指出的，在可预见的未来保持过于温和的态度不太可能让任何政党获得任何影响力。也许保持小规模且更为"核心"才是出路？另一方面，德国是一个特例。这个国家仍然遭受严重的心理创伤，根本无法组织任何形式的政治防御。
2.77 文化保守派/民族主义者的修辞策略
首先要专注于一个具体议题。使用进步和策略性的修辞。使用恰当的修辞对于成功传达信息至关重要。
"种族"、"白人"、"族裔"或"民族主义"这样的词，永远不应该在与受过严重思想灌输的对手或个人的现代辩论中使用。这些词被历史及战后媒体报道如此深刻地玷污，以至于主动使用它们你基本上就是在破坏自己和你想传达的信息。明智的做法是限制使用所有带有污名的词语，否则文化马克思主义/多元文化主义主流媒体会试图给你贴上偏执狂的标签。如果你使用"种族"这个词，你基本上是在帮助对自己进行人格暗杀，或者将导致你所代表的组织目标的自我挫败。你需要理解以下事实：现代欧洲男女已经被洗脑或条件化，以至于如果你试图用包含这些词的修辞来接触他们，他们很可能会逃之夭夭或启动潜意识心理防御障碍。我会尝试更彻底地解释这一点，因为这也适用于我自己。在一个绝对最大的罪过是成为纳粹的世界里，与纳粹相关的词语无论是否合理都必须不惜一切代价避免。我对这个问题的研究相当深入，多年来我与成百上千的人讨论过移民、欧洲身份和文化问题。我讨论的目的通常是让对方同意我的观点来确证我的信念（这是你与人讨论政治时的正常做法）。换句话说，我首先评估对方的知识水平，然后决定使用哪种修辞策略。如果他是一个（天真的）人道主义者，我会使用特定的方法；如果他是一个真正的文化马克思主义者，我会使用另一种方法；如果他是一个资本主义全球主义者，我又会使用另一种方法。我大概编写了上百种不同的修辞方法，无论对方是民族布尔什维克还是自由放任资本主义者。
我把我在这个简编中使用的大部分事实和论点都包括了进来，这样每个人都有机会成为一个辩论大师，如果他或她选择这样做的话。赢得新同情者或支持者的有效方法是告诉他们世界是如何运作的，但方式是通过提出正确的问题，或者更确切地说，是伪装成问题的出色论点。
我经常使用人口统计数据，例如：黎巴嫩的例子（曾经是一个基督教国家，现在是穆斯林国家），并补充道：
从人口统计的角度来看，你有什么理由认为西欧不会最终成为黎巴嫩？
而且，假如说西欧有可能最终成为黎巴嫩，难道欧洲人不应该被允许阻止这种情况发生吗？
或者其他几个例子：
你说保护西藏文化是正义的事业，说中国使用汉族移民进行人口统计战的策略是不道德的。那你为什么反对欧洲爱国者试图保护欧洲文化？
你声称自己是反法西斯主义者。但你为什么要嘲笑在瓦哈比清真寺外示威的欧洲人，而瓦哈比主义显然是一种法西斯意识形态？这难道不让你成为一个伪君子吗？
使用伪装成问题的基于事实的论点是无敌的，而且效果极好。我已经指导了世界各地数百人的修辞策略，因此帮助赢得了成百甚至上千的新支持者和同情者。
我目前大约有2500个Facebook联系人。每当我发布一个新故事（通常涉及欧洲当前的自我毁灭），我都会收到很多评论。我的通用规则是删除所有包含不恰当词语（如上所列）的帖子。原因是这些词语起到威慑作用并使辩论瘫痪。那些通常会更注重"政治正确"的爱国者个人在看到任何包含例如一个或多个词语如"白人"、"种族"、"种族灭绝"、"内战"的"不当帖子"后，会害怕发表评论。
在他们的脑海中，他们害怕以某种方式与那个人扯上关系，因为他们害怕被贴上种族主义者的标签。这种想法使他们瘫痪，激活了他们的自我保护机制，而自我保护机制又告诉他们不要发帖。这很可悲，但我们除了适应现实之外别无选择，即这么多爱国者被灌输到了一个完全被洗脑的地步。
每当我在辩论中忽略对这些词语的审查，我最终只有5个或更少的评论者。另一方面，如果我审查任何包含这些词语的评论，我通常会得到超过15个不同的评论者（三倍的数量）。
我们可以从这个例子和其他例子中学到的教训是：最终目标是唯一相关的事情。将你的修辞控制在某些政治正确的参数内是唯一合乎逻辑的做法。不这样做往往会适得其反或自我挫败。
自我约束的例子也应该用于其他修辞场合。我们拥有真相。那么我们如何最有效地传达真相呢？基于你要沟通的对象是哪个个体或目标群体，存在适得其反的方式和最佳方式。
如果你的目标群体明显是系统性制度化灌输的受害者，就像大多数人道主义者一样，那么用"简单的话"直接说出真相，只会激活他们的心理防御机制，从而把他们吓跑。相反，你必须通过非常模糊的例证来表达自己，这些例证几乎间接地让个体自己问出关键问题（也被称为心理条件化或反灌输修辞策略）。例如，请参见在更温和的文化保守派路线网站上发布的文章。
评估一个例子：
多元文化主义是一种反欧洲的仇恨意识形态，用于解构欧洲文化、传统、身份和民族国家。
对许多被灌输的人来说，这个陈述会太过残酷和直接，可能会吓跑他们。总是有空间考虑什么是最佳的说法，什么是适得其反的。我读过几篇十页的长文，最终归结为上述结论，写得很精彩。在很多情况下，那篇十页长文可以赢得一个支持者，而简短的陈述则可能吓跑他们。这完全取决于个体，当然也取决于你的目标是什么。对于一个有巨大抱负的大型泛欧组织来说，从长远考虑、轻步慢行是明智的；而一对一辩论或面向较小受众的个体通常会使用更"有力和直接的修辞"，通常旨在嘲笑或削弱修辞对手。
维护你的声誉盾牌以在所有目标群体中实现最大渗透/影响
假设你是一个发言人，想要面向各种保守派群体（具有不同意识形态背景——百分比表示当前的同情者潜力）：
1. 核心白人至上主义者或某些核心国家社会主义者（>5%）
2. 白人民族主义者——14字原则等（>10%）
3. 文化保守派（15-30%）
4. 温和文化保守派（对多元文化持批评态度）（30-35%）
5. 支持多元文化主义的保守派（40-60%）
必须对所有上述群体使用不同类型的修辞策略。通常，一个严肃的政党会使用相当温和的文化保守派修辞，知道默认情况下会赢得所有更坚定的同情者基础。
如果你使用为第5组设计的修辞策略，你也会吸引所有其他群体，等等。然而，如果你使用为第3组设计的修辞，你将被目标群体4和5认为过于极端。
维护你的声誉盾牌涉及使用正确的修辞。你未来想要接触哪些群体？思考10年、20年、30年的未来。因为如果你曾经选择变得"更有雄心"，过去的言论选择将被文化马克思主义建制派用来攻击你。
可悲的是，左翼并非如此，巴罗佐就是一个明显的例子。众所周知，他曾是葡萄牙共产党的成员，然而没有一家媒体公司曾试图对他进行人格暗杀，就像他们对任何和所有文化保守派所做的那样。
机会主义的政党或较大型组织可能决定"出卖"他们最基本的原则以增加潜在支持。欧洲有一些政党的例子，曾经属于第2类，但改革成为第5类，希望变得"在政治上具有相关性"。这可能以政治灾难告终，因为那个组织或政党最终陷入一个自我审查到实际变得无关紧要的境地。这也可能为位于他们右侧的更可靠原则的新参与者打开可能性。一些保守党最终支持了多元文化主义，这恰恰强调了他们已经自我挫败，不应再被允许自称为保守派。
至于要接触的目标群体：忘记文化马克思主义者或更核心的马克思主义者。无论你的论点和文件多么好，他们已经决定不想听到真相。相反，专注于人道主义者和天真的或无知的人——占人口的大部分。那才是支持潜力的所在。
在必要时使用欺骗手段，最糟糕和最无能的辩论者总是最真诚的辩论者。他们几秒钟内就被吞没和嚼碎，通常容易受到人格暗杀。这就是为什么95%的政治家都是说谎者或真相的操纵者，不幸的是，如果他们不是那样，他们就不会成为成功的政治家。
现代政治家在描述"欧洲人"时倾向于使用以下词语：非穆斯林、保守派、文化保守派、基督教欧洲人、"欧洲原住民"或欧洲本地人，或者如果我绝对必要的话——族裔欧洲人。这些词语中有些是相对"包容的"，可以吸引/接触所有光谱的右翼，包括基督教运动和自由市场自由主义者。
此外，限制使用像"族裔群体"这样的词是明智的，而是聚焦于伊斯兰教、非伊斯兰教、穆斯林、非穆斯林。同时，使用"反伊斯兰化"而非反移民一词。至于试图描述威胁，聚焦于全球伊斯兰乌玛和齐米制度（是的，需要包括对一些"新词"含义的某种教育）。人们通常不熟悉许多"新词"，因此需要进行某些解释性评论。
使用明确定义的"修辞策略"是所有保守派政治家、辩论大师和领导人的成功本质。不幸的是，40岁以上的人往往时不时冒出"肮脏过时的修辞"，而40岁以下的人通常在修辞约束方面更为自律。
此外，我之前说过好几次了。在政党名称中使用"民族主义者"或"民族"是一个战略错误——英国国家党和国民阵线就是例子。这个词附带了太多污名，许多人会潜意识地看到与纳粹德国的相似之处。
此外，在涉及移民和伊斯兰化的辩论中使用女性发言人可能是一个战略上明智的做法（这些话题对许多人来说传统上围绕"愤怒、单身、白人男性"展开）。
我们不是这些规则的制定者，但如果我们想要更成功地传播我们的信息，遵循这些规则符合我们的利益。
将伊斯兰教视为一种你可以利用的疾病。欧洲的伊斯兰化以及全球伊斯兰乌玛对欧洲发动的人口统计战基本上就是那个将把文化保守派团结起来并最终（在40-70年内）带向权力的"某件事"。因此，在此期间，老一代文化保守派开始适应这种"新现实"并开始更新他们的修辞策略，或者闭上嘴，这一点至关重要。
另一方面，种族保守主义已经死亡，不应以任何方式在修辞上与文化保守主义联系起来（种族保守主义在二战中已死）。
另一方面，我们的战斗涉及文化保守主义，我们抵抗文化种族灭绝和伊斯兰人口统计战的责任和权利。文化保守主义与种族保守主义毫无关系。从过去的错误中学习，并实行修辞约束。
但"族裔"和"种族"这些术语的相关性又如何呢？
族裔在每个冲突中始终扮演着重要角色，我相信无论我们喜欢与否，它在未来也会如此。只需看看伊拉克。这是一个教科书式的族裔战争例子——库尔德人对逊尼派阿拉伯人对波斯什叶派。土耳其：土耳其人对库尔德人等等。族裔与全球大多数国家的文化紧密相关，这已不是秘密。事实上，伊斯兰教是世界上最种族主义的政治意识形态之一。通常当一个社会被90%以上伊斯兰化时，往往会得到沙特和其他阿拉伯国家的财政支持的阿拉伯化进程就开始了。毛里塔尼亚和几个非洲国家就是这种情况。系统性的"阿拉伯化"努力被西方媒体和学术界所忽视，而他们却盲目地热衷于推行多元文化主义。
在今天的欧洲，大多数人几十年来一直是制度化多元文化主义灌输的受害者，我们必须实行修辞约束。目的不是为了安抚文化马克思主义者/多元文化主义者，而是为了保护我们自己免受文化保守主义这片雷区的伤害。文化马克思主义者/多元文化主义者希望我们使用"标记性"修辞，因为这将使他们很容易给我们贴上偏执狂的标签。
我不认为使用"标记性修辞"的人理解"声誉盾牌"的概念。事实上，我非常怀疑他们是否研究过现代右翼修辞策略，这对个人和政党都极其重要。造成关键差异的不是信息本身，而是传递信息的行为者的可信度。使用"标记性"修辞如"种族"、"种族战争"、白人、黑人、犹太人（例如在同一句话中与种族一起出现）、族裔（在同一句话中与犹太人一起出现）——会触发为他设置的所有可想象的地雷，从而彻底摧毁他的"声誉盾牌"（如果他曾经有过的话）。因此，他不仅破坏了自己的努力，还冒着推开其他"温和保守派"的风险。他们可能会因为与标记性个体有关联而疏远自己，因为这可能会影响他们个人的"声誉盾牌"。
