# 反种族主义猎巫行动
The Anti-Racist Witch-Hunts
反种族主义猎巫行动
作者：Fjordman

挪威主要博客Document.no[1]讲述了两个体面的白人男子在2007年因其所谓的"种族主义"而被媒体私刑暴徒攻击的可怕故事。后来事实证明，主流媒体当时并未呈现几乎所有与该案相关的信息，但这并没有阻止他们几乎毁掉这两个人。这发生在针对本地居民的一系列犯罪和暴力浪潮日益增长的同时。当局对针对本地居民日益上升的种族主义暴力浪潮的回应是加大对"种族主义"的打击——由白人本地居民实施的。2005年，挪威议会——在85%议员的支持下——通过了一项新的《反歧视法》[2]，由当时的保守党融合部长埃尔娜·索尔贝格起草，她之前曾呼吁在挪威建立沙里亚委员会。

右翼进步党的发言人佩尔·桑德伯格担心，这项法律会危及守法公民的权利。举证责任倒置与支付赔偿的责任结合在一起，这意味着无辜者有可能为他们没有做过的事情支付巨额赔偿。如果一个移民声称本地人以某种方式歧视了他或发表了歧视性言论，本地人必须提供自己无罪的证据。这项严厉的法律是在大多数移民自己声称很少遇到歧视的情况下通过的。

我后来发现，类似的法律已在西欧大部分地区通过，受到欧盟和欧洲委员会（CoE）在与国际伊斯兰组织合作下的推动。挪威法律是在欧洲委员会的一项倡议之后制定的。关于这项法律几乎没有公众辩论，它是在当年全国选举前相对沉默中通过的。没有一位记者真正批评过它，大多数人在通过之前几乎没提到它。同样的记者在其他方面往往非常关心伊斯兰恐怖分子的法律或"人权"，但显然不关心他们自己人民的权利。

平等与反歧视监察员[3]比阿特·甘戈斯，一位白人女同性恋女权主义者，在2007年市政选举前警告所有政党不要发表关于移民政策的"歧视性"言论，但呼吁积极减少[4]政治中白人异性恋男性的数量。那一年，严重左倾的媒体中几乎没有关于移民问题的真正辩论，但却有一场更为激烈的猎巫行动寻找种族主义者，而我所指的仅是白人。左翼联合政府在会见移民组织后宣布，种族主义者——显然仅指白人本地人——应该从所有公共部门工作中"被熏出"[5]。

同一政府于2008年10月资助了奥斯陆的一次会议[6]，涉及与中东一些强硬的沙里亚赞助团体进行"对话"，包括穆斯林兄弟会的埃及分支。外交部和平与和解处资助了由奥斯陆大学的比约恩·奥拉夫·乌特维克领导的研究项目"伊斯兰主义的断层线" [pdf][7]。代表外交部的工党成员格里·拉尔森计划会见这些团体的代表。因此，左翼政府可以与激进伊斯兰团体会面，但却想压制那些不希望持这种观点的穆斯林定居在本国的本地人的声音。不与他们进行"对话"。

2007年8月，奥斯陆的两名救护车司机（均为白人挪威本地人）卷入了一起后来被大肆炒作的所谓涉及"白人种族主义"的案件。救护车抵达时接一名受伤的非洲男子。正如救护车司机埃里克·申肯[8]几个月后解释的那样，那名男子"脱下裤子，在我同事的腿上小便。我的同事感到惊讶，躲开了，并骂他是猪。就在那时，我们将该男子视为问题，并决定最好由警察将他送到诊所。"涉案的索马里男子阿里·法拉赫头部受伤比司机当时估计的更严重。"我们犯了一个错误，因为我们将他的小便解释为故意和挑衅，但并不是因为我们有种族主义或歧视性动机，"申肯写道。

基于"种族主义"的薄弱嫌疑，大众媒体、主要知识分子和政治家发起了一场只能被描述为针对这两名救护车司机的猎巫行动。"这永远不会发生在一个白人身上，"代表法拉赫及其家人的著名挪威-巴基斯坦律师阿比德·Q·拉贾[9]说。曾担任工党司法部长的作家安妮·霍尔特在《晚邮报》上写了一篇文章，在我看来相当于对司机的口头处决。两人都被停职，并成为广泛负面媒体报道的目标。他们后来在挪威卫生监督委员会的调查后被澄清。然而，平等与反歧视监察员——或我喜欢称之为多元文化宗教裁判所——裁定法拉赫是歧视的受害者，救护人员违反了反歧视法律。如前所述，反歧视法规定本地人几乎在被怀疑"歧视"之时就是有罪的。

这里应该提到的是，这位非洲男子最初受伤是因为他被另一名来自加纳[10]的非洲男子殴打。在挪威这个横跨北极圈且没有殖民历史的国家，一个非洲男子殴打另一个非洲男子，结果是白人救护车司机——他们将毕生奉献于帮助他人——成为由该国媒体领导的私刑暴徒的目标。司机埃里克·申肯需要专业帮助，因为他被带到了自杀的边缘。

根据主要博客Document.no[11]的汉斯·鲁斯塔德所说，反种族主义意识形态在某些情况下类似于我们从共产主义政权看到的东西。"种族主义者"一词类似于在共产主义下被称为"阶级叛徒"。仅仅指控就足以摧毁生命。鲁斯塔德担心反种族主义在某些情况下导致无法无天。救护车司机申肯是一名备受好评的员工，执行过数千次任务并挽救了众多生命，然而由于一个在我看来考虑到当时情况可以理解的判断错误，他的生活被毁了。这一切都因为他的肤色在政治上不正确，而另一个人不是白人。

使这一切更加荒谬的是，在挪威，正如整个西方世界一样，白人对非白人的暴力极其罕见。绝大多数种族主义和种族主义暴力来自非白人对白人，或在不同非白人群体之间。在奥斯陆，年轻女孩被强奸；学童受到死亡威胁、被抢劫和攻击。警方警告全国城市地区"街头暴力惊人上升"。这是由大规模移民直接造成的，而大规模移民仍然被攻击这两名救护车司机的同一媒体所拥护。

例如[12]，一名17岁的索马里人因在奥斯陆强奸一名年轻女孩被判罪。法院指出，强奸异常残忍并持续了几个小时。该男子掐住女孩脖子如此之久，以至于为她检查的医生说，她本可能死掉。在袭击之后，女孩患有严重的心理问题。这位非洲青年被判处四年半监禁。这一判决包括另一起强奸案，其中他的挪威-摩洛哥朋友强奸了一名13岁女孩，而索马里人则帮忙威胁她和把风。她自然因此而受到创伤。

无数本地人因类似的袭击而生活被毁，然而反白人种族主义很少被主流媒体视为问题。白人显然是可以被攻击的对象。针对白人在自己国家的强奸、抢劫和刺伤越恶毒，对"白人种族主义"的猎巫行动就越激进和歇斯底里。法国哲学家阿兰·芬基尔克罗[13]警告说，"'反种族主义战争'的崇高理念正在逐渐转变为一个可怕虚伪的意识形态。而这种反种族主义将成为21世纪共产主义在20世纪的角色：暴力的根源。"

奥斯陆大学的西格德·斯基尔贝克教授[14]指出，"1994年，德国期刊《焦点》指出在德国、法国和英国进行的民意调查中，分别有55%、52%和50%的人觉得他们的国家接受了太多移民。在挪威，我们有一项1987年的代表性研究，显示51%的人觉得国家应该接受更少的移民；25%的人觉得政治家应该坚持现行做法，而只有8%的人希望接受更多移民。几年后在瑞典进行的一项类似研究显示，54%的瑞典人觉得有太多人移民到瑞典。在后来的研究中，数字有所变化；但总是有更多人支持限制性政策而不是支持自由化。"因此，根据斯基尔贝克的说法，"近期移民的程度不能基于民意来解释[我的强调]。"

斯基尔贝克质疑[15]反种族主义运动是否存在一种准宗教的暗流，以及它是否实际上是过去时代猎巫的等价物：

"一些研究人员逐渐认识到，移民辩论中的某些问题具有宗教内涵。例如，挪威社会人类学家英格·莉丝·利恩写道，在公共移民辩论中，'种族主义'已成为一个用来标记我们中间恶魔的词，是所有正派人都应远离的不洁之物。我们有充分理由相信，在我们这个时代使用'种族主义者'一词与前三百年前使用'异端'一词有许多功能上的相似之处……很可能完全可以加入反种族主义运动，唯一的动机是认同某种看起来政治正确的东西，或者为了成为一个集体的一部分，这个集体允许人去示威并骚扰那些没人愿意去维护的边缘群体。"但是，"在'粉碎种族主义者'的口号背后，很可能存在比对行使暴力的原始欲望更多的东西。这场战斗还包含一种为纯洁对抗不洁而斗争的元素。由于种族主义是某种模糊不清的东西，反种族主义及其所代表的所谓多彩社区，成为了纯洁的表达。"

挪威左翼作家托格里姆·艾根[16]警告大规模移民带来的"种族战争"，却继续支持它。当被问及我们能做什么来避免这种情况时，他说："这是一个对作者来说非常愚蠢的问题。这预设了我希望大家快乐、过得好、没有问题。如果是这样的话，他们想让我写什么呢？"

我会赞扬他的坦诚：这是我见过的最坦白的承认，即有些人并不希望社会和谐；他们认为这很无聊。对于一个自诩的知识分子来说，最糟糕的命运就是生活在一个总体上繁荣、和平和运转良好的国家，因为没有人会关心他的建议或遵循他的指导，就像他这样聪明的人应该得到的那样。

在1990年代多元文化狂热期间，艾根在一篇题为《精神病式的种族主义》[17]的文章中警告了大规模移民造成的动荡。对此的解决方案不是限制移民，而是限制对移民的批评。据他说，仇外心理和对大规模移民的反对应被视为精神疾病，因此"解决这种仇外心理的办法是，你应该向那些受到严重影响的人分发药物。我已经与社区医学教授佩尔·富格利博士讨论过这一点，他喜欢这个想法。"富格利先生建议在城市饮用水中投放抗精神病药物。

这可能听上去过于极端而不被认真对待，但富格利继续斥责那些批评国家移民政策的人。艾根警告说，关于普通人对大规模移民表示担忧的论点不应该被接受，因为这可能导致法西斯主义："人们应该警惕那些以他人名义唤起仇外心理的人，尤其是政治家。如果某些人用诸如'普通人感觉'这样的短语开始他们的推理，一个人不应该争论，应该[打他们]。"

奥斯陆大学社会人类学教授托马斯·希兰德·埃里克森领导着一个由国家赞助、耗资数百万的项目[18]，试图设想新的多元文化社会将如何运作。他是一个职业多元文化主义者和该国知识界的名人，经常参与公共辩论，据他自己说，住在城市里一个无聊的、单一文化的区域，与文化多样性的影响隔绝。希兰德·埃里克森宣布了（西方）国家的死亡，仿佛他从中获得快乐，并声称尼达罗斯大教堂（Nidarosdomen）——该国最著名的教堂——不应再作为我们多元文化社会的国家象征。

埃里克森先生曾与奥勒-约根·安芬森发生冲突，后者运营着双语质量网站HonestThinking.org，并警告失控的大规模移民的影响。据[19]希兰德·埃里克森说，"世界主义坚持要求一个包含比黑白更多颜色的世界。在这样的世界里，奥勒-约根·安芬森提出的问题不仅微不足道，而且无关紧要。"

安芬森先生提出的问题是什么？嗯，他公布了数字，表明如果当前的移民持续下去，挪威本地人将在几代人之内在自己国家成为少数族裔。鉴于在自己土地上成为少数族裔的族群通常处境艰难，并且当新来者是穆斯林时总是受到迫害，人们会认为这是有趣的信息。但对于自称的"多元文化世界主义者"来说，甚至考虑这可能代表一个问题也是"微不足道和无关紧要的"。埃里克森称安芬森"愚蠢和无知"，并暗示"也许安芬森的议程是受了某种变态基督教的影响（他有基督教背景）。"

"他有基督教背景。"这应该是侮辱并取消一个人担忧他的孙辈是否会被迫害的资格吗？埃里克森先生，像其他西方多元文化主义者一样，担心伊斯兰恐惧症，但却非常愿意嘲笑基督教。一篇由埃里克森合著的报纸文章说："他[安芬森]是在要求我们再次重复那些显而易见的事情吗？即西奥·梵高的谋杀、各种恐怖主义行为以及针对报纸编辑的死亡威胁与伊斯兰无关？"

与伊斯兰无关？真的吗？

穆罕默德·布耶里[20]，出生于阿姆斯特丹，摩洛哥父母，于2004年11月2日在阿姆斯特丹杀害了正在骑自行车的西奥·梵高，先是开枪射击和刺伤，然后割开他的喉咙，用一把刀将一封信念钉在他身上。"我做我所做的纯粹出于我的信仰，"他在紧握一本古兰经[21]时告诉法官。"我想让你知道，我是出于信念行事，并不是因为他是荷兰人或因为我是摩洛哥人而夺走他的生命，"而是因为他相信梵高在他批评穆斯林妇女待遇的电影中侮辱了伊斯兰。

因此，一个和平的基督徒被指控有黑暗、秘密的议程，而一个自夸其伊斯兰动机的穆斯林凶手却与伊斯兰无关？一位来自前南斯拉夫的塞尔维亚医生——那里一个多元文化社会最近在内战中崩溃——警告了不受控制的大规模移民的影响。托马斯·希兰德·埃里克森通过斥责她"缺乏远见[22]"来回应。

显然，你作为知识分子的价值是以你的想法有多宏大来衡量的。你的愿景越宏伟，你的才智就越耀眼，因此应授予你的声望就越大。那些愿景是否实际符合现实和人性是次要的。事实上，许多自诩的知识分子会因他更普通的同胞们的琐碎考虑——关心他的想法在现实生活中会产生什么影响——而感到被冒犯。他的想法可能会伤害某些人这一事实并不让他气馁。人类真正伟大的进步只能通过牺牲来实现，最好是由除他自己以外的人做出。

那些拥护大规模移民的人对他们的政策带来的暴力没有任何自我批评。相反，他们想要更多的移民。在2007年联合国人口报告[23]发布之后——该报告表明未来几十年的全球人口将增加数十亿——挪威左翼报纸《每日新闻》的政治编辑玛丽·西蒙森写道，任何地方的人都应该享有迁移到任何他们想去的地方的普遍人权，这份报纸数十年来一直在谴责右翼进步党因其限制大规模移民的"种族主义"政策。这将意味着这个微小、富裕和天真的斯堪的纳维亚国家几乎必然的湮灭。西蒙森女士因此为她自己人民的逐步消灭背书，毫无疑问她为她的容忍而自鸣得意。对这份声明，没有任何其他记者提出一个字的抗议。

"人权"最初是为了确保自由的一个概念。现在却被用来消灭整个文明，以容忍和多样性的名义，而本地人则被明确禁止对此提出抗议。
