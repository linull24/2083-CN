# 吉哈德摧毁瑞典模式
2.55 Jihad Destroys the Swedish Model
吉哈德摧毁瑞典模式
作者：Fjordman

我决定在马尔默本周末发生骚乱之后写这篇文章。马尔默是瑞典第三大城市，也是斯堪的纳维亚穆斯林侵略最严重的城市。我最近读到，一名在马尔默接受采访的阿拉伯女孩说，她太喜欢那里了，感觉几乎像一个阿拉伯城市。本土瑞典人多年来一直在迁离这座城市，在自己国家被吉哈德变成了难民，这与菲律宾、泰国南部或印度克什米尔等地区的非穆斯林，或者科索沃的基督教塞尔维亚人的情况并无太大区别。

瑞典在冷战期间被呈现为资本主义和共产主义之间的中间道路。当这种社会模式崩溃时——它将会崩溃，在伊斯兰吉哈德、欧盟、多元文化主义和意识形态过度扩张的联合力量之下——因此崩溃的将不仅仅是瑞典国家，还有瑞典的象征，即一整套意识形态世界观的展示品。我两年前写道，如果这种趋势不被阻止，瑞典民族将在本世纪上半叶不再以任何有意义的方式存在。那个给我们带来伯格曼、ABBA和沃尔沃的国家，可能会成为北欧的波斯尼亚，"瑞典模式"将成为对意识形态疯狂的警告，而非钦佩的对象。我仍然担心我的评估是正确的。

乔纳森·弗里德曼，一个住在马尔默郊外的美国人，提到所谓的《1997年融合法案》宣布"瑞典是一个多元文化社会"。该法案的注释还指出："由于许多人起源于另一个国家，瑞典人口缺乏共同的历史。因此，与瑞典的关系以及对社会基本价值观的支持，对融合来说比共同的历史起源具有更大的意义。"本土瑞典人因此被降格为瑞典的又一个族群，对该国的主张权并不比上周才抵达那里的库尔德人或索马里人更多。该国的政治当局已经抹去了自己人民的历史和文化。

延斯·奥尔巴克，社会民主党的民主、大都市事务、融合和性别平等部长，在2004年瑞典电台的一次辩论中说："我们必须对伊斯兰教和穆斯林开放和宽容，因为当我们成为少数时，他们也会对我们如此。"这是一个完全清楚自己的人民将在自己国家成为少数派的政府，却没有采取任何措施来阻止这种状况。相反，皮埃尔·舒里，移民部长，在1997年的一次议会辩论中说："种族主义和仇外心理应该被禁止和追逐，"不应该接受"移民和难民政策存在缺陷之类的借口"。换句话说：本土人口不协助自我消灭应被视为犯罪。

奥尔巴克的态度是一旦你宣布文化无关紧要之后的必然结果。我们的文化，即使我们试图忘记它，是沉浸在基于互惠黄金律的犹太-基督教道德之中的："你们愿意人怎样待你们，你们也要怎样待人。"（路加福音6:31）另一方面，穆斯林沉浸在基于穆斯林至上主义的伊斯兰传统中。穆斯林将缺乏武力视为软弱的标志，他们蔑视软弱，这正是阿道夫·希特勒表达他对伊斯兰教钦佩的原因，认为它比基督教更适合纳粹主义，因为基督教有那种幼稚的同情心观念。

一名瑞典男子因为穿着带有自己国旗的衣服而险些丧命，当时瑞典正在参加2006年足球世界杯。一些"多元文化青年"认为这是无法容忍的挑衅，这名24岁的男子在马尔默被一辆汽车撞倒，在那里，穆罕默德正在成为新生男孩最常用的名字。

费里兹和帕吉廷，马尔默"阿尔巴尼亚帮派暴徒部队"的成员，解释了他们在市中心抢劫行人的方式。他们锁定一个孤独的受害者。"我们围住他，殴打和踢他，直到他不再反抗，"费里兹说。"你总是比受害者多很多人。懦弱？""我从许多人那里听到过这种说法，但我不同意。关键就在于他们不应该有机会。"他们对受害者没有任何同情。"如果他们受伤了，只能怪他们自己太弱了，"帕吉廷耸耸肩说。

马尔默市目睹的抢劫浪潮是"一场针对瑞典人的战争"的一部分。这是具有移民背景的年轻抢劫者在接受佩特拉·奥克森采访时给出的解释。"当我们在城市里抢劫时，我们是在进行一场战争，一场针对瑞典人的战争。"这个论点被反复提及。"对我来说，权力意味着瑞典人应该看着我，趴在地上亲吻我的脚。"男孩子们笑着解释，"抢劫时身体里有一种刺激的感觉，你感到满足和快乐，感觉好像成功了，就是感觉很好。""我们每天都在抢劫，想抢多少次就抢多少次，想什么时候抢就什么时候抢。瑞典人什么都不做，他们只是把东西给我们。他们太软弱了。"

《告别人民之家瑞典——一种社会模式的解体》是2005年出版的一本关于移民和被称为"人民之家"的瑞典福利国家模式的书。据作者所述，瑞典多元文化精英首先将自己视为世界公民。为了强调和突出多样性，一切瑞典的东西都被故意贬低。反对这种政策被视为一种种族主义形式："在瑞典占主导地位的意识形态，是通过强大的压制和沉默方法被确立为主导的，是一种极权主义意识形态，精英们反对民族国家的民族方面。问题在于，被描述为瑞典人的族群被含蓄地视为民族主义者，因此被视为种族主义者。"作者担心，移民政策的处理严重侵蚀了民主，因为公民对一个他们不再认为是自己的国家失去了忠诚。"政府没有增加公民的积极参与，反而对思想自由、言论自由和集会自由施加了明确的限制。"

莫娜·萨林曾在社民党内阁中担任多个职位，包括民主、融合和性别平等部长。萨林说过，许多瑞典人嫉妒移民，因为移民不同于瑞典人，他们有文化、有历史，有将他们联系在一起的东西。注意瑞典当局如何首先正式声明瑞典人没有历史或文化，然后接着哀叹瑞典人没有历史或文化。一个巧妙的把戏。萨林还表示："如果在移民很少的工作场所有两名同样合格的申请人，那个叫穆罕默德的人应该得到这份工作。拥有不同于瑞典人的族群背景应被视为一种优势。"2004年，她被引述说："必须推行一项旨在教育瑞典人移民是对其国家的祝福的协调努力，"强调她的同胞必须接受新社会是多元文化的。"不管你喜欢与否，这就是新的瑞典。"莫娜·萨林于2007年当选社民党领袖，因此成为未来瑞典首相职位的竞争者。

为什么政府要取消社会契约并如此攻击自己的人民？首先，因为它能做到。瑞典目前可以说是西方世界政治上最具压制性和极权主义色彩的国家。它还拥有最高的税率。这可能是一个巧合，但我不确定。国家已经变得如此庞大和强大，以至于它已经成为一个有自己意志的自主有机体。人民是为了服务国家而存在的，而不是相反。而且因为国家权力渗透到社会的每一个角落，包括媒体，如果国家决定攻击你，就没有任何地方可以组织防御了。

有人开玩笑说，其他国家是拥有军队的国家，而巴基斯坦是拥有国家的军队。同样，可以说瑞典最初是一个拥有官僚机构的国家，最终成为一个拥有国家的官僚机构。事实上，官僚机构正式废除了它本应服务的民族。它的代表不再是人民的领袖，而只是只顾通过润滑和维护（如果可能的话，扩大）官僚机器来推进自己职业生涯的管理者。瑞典人缴纳着任何（所谓的）自由国家中最高的税率，而他们得到的却是存在缺陷的社会保障、不存在的身体安全保障，以及一个致力于摧毁他们自身的国家机器。

安娜·埃克伦德在《晚报》上写道："我们是一个每天被政府侮辱的人民。我们不被认为能够独立思考，决定自己读什么，或者管理自己的钱。瑞典人就像酗酒者的妻子一样共同依赖。然而我们并不急于去投票箱推翻现有的制度。不是因为我们不想，而是因为我们中有太多人把自己逼入了死角。"

此外，瑞典人敏锐地意识到，他们的国家被许多外人视为"模范社会"。瑞典是一个深度意识形态化的国家，致力于将某种世界观强加给其公民，而且因为国家是意识形态化的，异议者确实被当作国家的敌人对待。

在《新极权主义者》一书中，英国历史学家罗兰·亨特福德在20世纪70年代初指出，由于文化原因，在瑞典建立社团国家的法西斯模式比在墨索里尼的意大利更容易，因为瑞典有集中的官僚机构，而意大利人对国家权威持怀疑态度。简而言之：瑞典人倾向于信任他们的官僚，而任何头脑正常的意大利人绝不会这样做。据他说，"瑞典人害怕争议，认为争议是不愉快的、低效率的且隐约不道德的。他们追求的不是对抗而是共识以求得内心平静。共识指导一切：私人谈话、知识生活和国家运作。" 当时的教育部长英格瓦·卡尔松将学校教育的目的定义为："培养一个适应良好的社会好成员。它教人尊重共识，不要破坏共识。"他还曾说过"学校是社会主义的先锋"。卡尔松先生最迟在1996年还是瑞典首相。卡尔松在社民党中的导师和前任瑞典首相奥洛夫·帕尔梅公然表现出他对西方文明的漠视，如果不是蔑视的话："所谓的文艺复兴？西方文化？这对我们来说意味着什么？"在工党运动的监督下，瑞典教育数十年来故意攻击西方文化，使其看起来可疑。

据亨特福德先生所述，"当瑞典人改变观念时，他们做得彻底，不给批评或保留留下任何空间。这个国家缺乏知识防线；任何新事物都会在没有阻力的情况下征服一切。"共识"假定技术进步是通往幸福的唯一途径，国民生产总值是衡量国家成功的唯一标准。它还假定集体的善在任何时候都必须优先于个人的善。它规定瑞典社会的基本原则绝不能被质疑或讨论。"马伊·博索姆-诺德博夫人，当时学校总局的部门主管，这样表达道："培养个性是无用的，因为除非人们学会适应社会，否则他们会不快乐。我们不强调自由。相反，我们谈论放弃自由的自由。重点是儿童的社会功能，我不否认我们强调集体。"

罗兰·亨特福德以这样的警告结束了他的书：这种软极权主义制度可能会被输出到其他国家。他已被证明是正确的："瑞典人已经展示了当前技术如何在理想条件下被应用。瑞典是对一个孤立和消毒过的对象的对照实验。作为新极权主义的先驱，瑞典人是一个警告，提醒我们其他人应该怎样警惕，除非我们小心抵制控制和中集权，除非我们记住政治不是要被委派的，而是个人的关切。新极权主义者，依靠说服和操纵，必须比依赖武力的旧极权主义者更有效率。"

2006年9月选举后，弗雷德里克·赖因费尔特成为瑞典首相，主持一个中右翼联合政府。在我看来这是积极的。瑞典被一些人描述为"一党制国家"，因为社民党在过去74年中的65年里一直掌权。然而，瑞典左翼和右翼之间的差异并不总是那么大。这些党派上次掌权时，在首相卡尔·比尔特领导下（1991年至1994年），他们主持了大规模移民，此后也未曾强烈反对多元文化政策。新任外交部长比尔特在担任联合国巴尔干专员时曾呼吁承认伊斯兰教是欧洲文化的一部分。

首相赖因费尔特表示，原始的瑞典文化只不过是野蛮："谦卑地提醒人们这样一个事实有时是有益的，即构成瑞典的很大一部分是在一个进化过程中创造出来的，恰恰是因为我们已经开放接受其他人和经验。"赖因费尔特在访问斯德哥尔摩附近南泰利耶一个叫罗纳的地区后说了这番话。一年前，南泰利耶的一个警察局在移民青年与警察发生重大冲突后被自动武器射击击中。罗纳的骚乱是在一名瑞典女孩被称为"婊子"并对这种说法做出反应后开始的。民族学家玛丽亚·贝克曼在她的研究"白人性与性别"中，跟踪了斯德哥尔摩外移民郊区林克比的一群瑞典女孩。贝克曼叙述说，几个金发瑞典女孩说她们染了头发以避免性骚扰。

我称瑞典为一个软极权主义国家，但我有时对"软"这一部分不太确定。民意调查显示，三分之二的瑞典人怀疑伊斯兰教是否能与瑞典社会相容，而且很大一部分人口多年来一直希望更多限制移民。然而，议会中没有真正批评多元文化社会的党派。这仅仅是巧合吗？欧洲大陆上避免战争时间最长的国家瑞典，也是西方一个政治正确达到最严重程度的国家？也许漫长的和平时期创造了一种环境，在这种环境中，层层的意识形态废话被允许一代又一代地堆积起来，永不停歇。我不知道一代人后的瑞典会是什么样子，但我相当肯定，它不会再被视为模范社会。如果战争的缺席是其当前弱点的原因之一，我担心这是一个很快就会得到纠正的问题。
