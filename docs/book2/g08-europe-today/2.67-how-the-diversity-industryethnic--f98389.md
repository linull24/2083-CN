# 多元化产业/族裔产业/嘻哈等亚文化如何与不受限制的媒体权利共同毁灭社会
How the diversity industry/ethnic industries/sub-cultures such as Hip-Hop in combination with unrestricted media rights contributes to destroy society
多元化产业/族裔产业/嘻哈等亚文化如何与不受限制的媒体权利共同毁灭社会

约翰·H·麦克沃特

不久前，我在哈莱姆的一家肯德基吃午饭，坐在八名约14岁的非洲裔美国男孩附近。由于1）那是上学日下午1:30，2）他们背着书包，并且3）他们似乎不赶时间，我推测他们逃学了。他们非常吵闹和不守规矩，互相扔食物并将其留在地板上。

黑人在经营这家餐厅，构成了顾客的大部分，但在这里很难看到多少健康的"黑人社区"。在多次警告男孩们停止扔食物并保持安静后，经理最终让他们离开。孩子们无视了她。直到她叫来一名男保安，他们才开始慢慢走出去，嘲弄地绕着餐厅转了一圈，然后晃晃悠悠地离开了。这些少年显然不是怪物，但他们似乎认为自己不受公共行为规范的约束——仿佛他们已经开始退出主流社会。

然而，最让我震惊的是，男孩们的音乐——尖刻的说唱，鼓吹对权威根深蒂固的反感——为他们的反社会行为提供了持续的配乐。说唱如此完全地融入了他们的意识，以至于每隔一会儿，他们中的一个就会突然唱起傲慢的、充满咒骂的说唱歌词，伴随着典型的说唱表演中那种生硬、好斗的手势。然后他的几个伙伴也会加入。说唱是他们对话中不断出现的装饰。

许多作家和思想家在说唱和嘻哈中看到了一种有见识的政治参与，甚至是革命的潜力。他们大错特错了。通过强化那些长期阻碍黑人的刻板印象，并教导年轻黑人，暴徒式的对抗姿态是对一个假定为种族主义的社会的恰当"真实"回应，说唱阻碍了黑人的成功。

在1960年代之前，弥漫在说唱中的恶意在黑人流行文化——实际上在黑人态度中——几乎没有立足之地。嘻哈精神可以追溯到那个十年中一种黑人意识形态的出现，这种意识形态将黑人力量和真实的黑人认同等同于对美国社会的激进对抗立场。在马尔科姆·X高举的拳头所传达的愤怒新情绪中，许多黑人（以及更多的白人自由主义者）开始将黑人犯罪和暴力视为对所谓的种族主义社会所施加的非人化和贫困的自然、甚至恰当的回应。

短暂地，这种激进精神——以上面最突出的是黑豹党——注入了黑人流行文化，从勒罗伊·琼斯的戏剧到"黑人剥削"电影，比如梅尔文·范皮布尔斯的《甜蜜甜背的坏蛋之歌》，这些电影将黑人犯罪反叛者作为英雄来庆祝。

但黑人剥削电影和类似题材很快就消亡了。白人公然刻板化黑人的记忆太近，以至于像《甜蜜甜背》这样的电影中对角色定型不能不冒犯许多黑人。黑人历史学家勒罗恩·贝内特观察到："在这样的事实中，有一种残酷的白人幽默：1960年代的黑人游行和示威在艺术上得到了实现"，以"昔日所有萨菲尔和斯塔德形象的挑衅性和最终阴险的再版"。

早期的说唱大多避开了萨菲尔和斯塔德，它开始不是作为底层的咆哮，而是作为快乐的派对音乐。第一个大热的说唱歌曲，糖山帮1978年的《说唱者的喜悦》，以一个朗朗上口的低音节拍推动音乐前进，快乐的说唱歌手将自己庆祝为一位讨女人喜欢的男人和伟大的舞者。很快，全美各地的孩子们都在跟着荒谬的副歌说唱：

我说一个嘻，一个跳，嬉皮士，嬉皮士，
到嘻-嘻跳，啊你不要停
摇滚到嘣嘣布吉，说
向上跳布吉，
到布吉的节奏，节拍。

在随后的几个月里，一连串欢快的说唱作品接踵而至。当时，我以为这是一种无害的潮流，很快就会失去动力。

但说唱在1980年代初发生了黑暗的转变，这种"泡泡糖"音乐让位于"匪帮"风格，后者从黑人剥削电影停下的地方继续。现在顶级说唱歌手开始写尖锐的歌词，庆祝街头战争或毒品和滥交。大师弗莱什1982年的不祥热门歌曲《信息》，其副歌是"有时像丛林一样，让我想知道我如何不沦陷"，标志着感性上的变化。它将贫民窟生活描绘成极度荒凉：

你在贫民窟长大，过着二等生活
你的眼睛会唱出深深仇恨的歌。
你玩耍的地方和你住的地方
看起来像一条巨大的小巷。
你会钦佩所有账本上的接受者，
暴徒、皮条客和毒贩，以及大钱制造者。

音乐评论家争相赞美《信息》，将其视为街头诗歌——正如精英媒体此后一直对嘻哈的定性。这首歌的冷酷宿命论引起了共鸣；两次，我听到在关于种族问题的演讲的观众中，黑人引用了副歌来强调关于黑人受害者身份的观点。它也传递了它所承载的警告："别逼我，因为我接近边缘，"梅勒·梅尔威胁性地说唱道。《信息》的最终信息——贫民窟生活是如此无望，以至于暴力的爆发既是合理的也是迫在眉睫的——在随后的岁月中将成为嘻哈的口头禅。

《信息》重新引入黑人流行文化的愤怒、对抗姿态将说唱从一种时尚转变为价值数十亿美元的产业，2002年在美国销售了超过8000万张唱片——占所有录制唱片销售的近13%。对于像拉塞尔·西蒙斯这样的说唱制作人来说，早期的黑人流行音乐只是娘娘腔的音乐。他鄙视像迈克尔·杰克逊或路德·范德罗斯这样的艺术家的"柔软、非侵略性音乐（和非威胁性形象）"。"所以我一有机会，"他说，"我做了完全相反的事。"

在《信息》之后的二十年里，嘻哈表演者生产了无数说唱歌曲，庆祝充满持续暴力和犯罪的贫民窟生活。Schooly D的《PSK What Does It Mean?》就是一个例子：

拿起我的手枪，跳进车里。
到了酒吧，拿了点防弹衣，
拿了些大麻，一点不差。
到了地方，我看到谁了？
一个混蛋黑鬼试图学我说话。
把我的手枪顶在他头上——
我说，"混蛋黑鬼，我应该把你打死。"

KRS-One（一位后来反对说唱暴力的嘻哈明星）的一段韵律中的主角实际上扣动了扳机：

认识一个叫彼得的毒贩——
不得不用我的9毫米把他干掉。

在匪帮说唱的想象中，警察力量成为了掠夺性入侵者。已故西海岸说唱歌手图帕克·沙库尔表达了这种态度：

你得知道怎么抓蛇，黑鬼，
因为警察喜欢抓捕黑鬼，
把他送到州上去，因为他们就是恨黑鬼。

然而，与Ice-T臭名昭著的《警察杀手》相比，沙库尔的反警察长篇大论似乎温和些：

我穿着我的黑色衬衫。
我戴着我的黑色手套。
我戴着我的滑雪面罩。
这狗屎太久了。
我拿着我的12号锯断猎枪。
我关掉了我的前灯。
我正要开几枪。
我正要干掉几个警察……
我正要去杀点什么
一个猪拦住了我，什么理由都没有！
警察杀手，你死比我死好。
警察杀手，去他妈的警察暴力！……
死，死，死猪，死！
去他妈的警察！……
去他妈的警察，耶！

说唱也开始提供一些人类历史所知的最冰冷厌恶女性的音乐。再来看Schooly D：

告诉你吧，兄弟，这不是开玩笑，
她把我带回家，把我放在床上，
我从脚趾到头干了她。
我终于意识到那女孩是个妓女，
给了她十美元，她还向我要更多。

Jay-Z的《那是你的婊子吗？》挖掘了类似的主题：

我不爱她们，我干她们。
我不追她们，我躲她们。
我用另一个换掉她们……
她全在我的鸡鸡上。

或者，正如N.W.A.（"有态度的黑鬼"的缩写）简洁地总结了嘻哈世界观："生活不过就是婊子和钱。"

说唱的音乐伴奏以其刺耳和重复反映了说唱歌词的残暴。西蒙斯以蔑视悦耳和谐的方式来制作他的录音。"我们用于旋律的是隐含的旋律，我们用于音乐的是声音——节拍、刮擦、倒放的东西，没有漂亮或甜蜜的东西。"嘻哈的成功导致了一种讽刺的反转。在七十年代，尖叫的硬摇滚在年轻白人中流行，而甜蜜、曲折的放克和灵魂乐统治着黑人电台——我对这种差异感到自豪。但在八十年代，摇滚安静下来，黑人音乐成为了对耳朵和灵魂的攻击。在八十年代美国城市长大的任何人都不会很快忘记那些年轻人在街上漫步，从他们的手提音响中轰鸣着这种声音武器，带着挑衅的怒视挑战任何人让他们把音量调低。

嘻哈与音乐视频同时期爆发进入大众意识，很快说唱歌手就充满了MTV，以图像强化说唱歌词中描绘的丑陋世界。一个又一个视频中，说唱明星炫耀珠宝，开着改装车，展示武器，愤怒地对镜头指手画脚，并与可互换的、无脑扭动的、衣着暴露的女人嬉戏。

当然，并非所有嘻哈都是好斗的或亵渎的——一张完全由帮派火拼、挑衅警察、辱骂女性的谩骂组成的CD对大多数听众来说很快就会厌倦。但最恶毒的说唱卖得最好，最恶毒的曲目成就事业。在我写这篇文章时，十大最畅销嘻哈唱片是50 Cent（目前在全国所有音乐类型中拥有第二畅销的唱片）、Bone Crusher、Lil' Kim、Fabolous、Lil' Jon和the East Side Boyz、Cam'ron Presents the Diplomats、Busta Rhymes、Scarface、Mobb Deep和Eminem。这些团体或表演者中的每一个人都体现了对社会蓄意的、舞台式的反抗——Lil' Jon和他的团队甚至用一首名为《不在乎》的歌曲来娱乐我们——每一个都庆祝贫民窟是"最棒的地方"。因此，偶尔那些出于义务而写的歌曲——在歌曲中说唱歌手鼓励男人为他们的孩子负责或哀叹无意义的暴力——仅仅是装饰。将暴徒形象保持在前沿和中心位置已经成为了成为明星的最快和最有可能的方式。

没有哪位嘻哈名流比肖恩·"P. Diddy"·康姆斯更努力地培养匪帮形象了——他是极为成功的说唱歌手、制作人、时尚大亨和Bad Boy唱片公司首席执行官——以至于他模糊了扮演坏男孩和真正成为坏男孩之间的界限。康姆斯可能在纽约州芒特弗农的中产阶级家庭长大，甚至还在霍华德大学学习过一段时间，但他已经证明他可以和最坏的人一起帮派火拼。警察在1995年指控康姆斯持有致命武器。1999年，他因袭击一位竞争对手的唱片高管而面临指控。最臭名昭著的是，警方那年指控他在一家夜总会因受辱而开枪，伤及三名旁观者，并与随行人员（包括当时的朋友詹妮弗·"J. Lo"·洛佩兹）逃离现场。康姆斯脱身了，但他年轻的弟子说唱歌手贾马尔·"Shyne"·巴罗因开枪被判入狱。

正如评论家凯莱法·桑内精辟地指出的那样，康姆斯和他的团队远不是唯一在"说唱和案底"之间保持联系的说唱歌手。几位著名说唱歌手，包括超级巨星图帕克·沙库尔，已在枪林弹雨中倒下——其他说唱歌手经常被怀疑是凶手。死囚唱片公司制作人马里恩·"Suge"·奈特刚刚完成了因袭击和联邦武器违规被判的五年刑期。当前的火热人物50 Cent在照片中炫耀他的子弹伤疤；警方最近逮捕了他，因为他在车里藏了攻击性武器。上述十大嘻哈畅销者中，有五位曾与法律有过摩擦。2000年，至少五起不同的斗殴在Source嘻哈奖颁奖典礼上爆发——这本来应该是说唱行业的格莱美奖。最后一场涉及多达100名观众的斗殴，甚至蔓延到了舞台上，在刚刚结束的被杀说唱歌手的视频致敬之后，关闭了颁奖典礼。难怪一个流行的说唱网站名为rapsheet.com。

许多歌迷、说唱歌手、制作人和知识分子为嘻哈的暴力（无论是真实的还是想象的）及其厌女症辩护，称之为来自被剥夺权力的年轻人的革命性挫败呐喊。对西蒙斯来说，匪帮说唱"教导听众关于创作者生活的一些东西，并提醒他们这些人存在。"50 Cent最近告诉《Vibe》杂志，"主流美国可以看着我说，'那是来自贫民区的年轻人的心态。'"宾夕法尼亚大学黑人研究教授迈克尔·埃里克·戴森写了一本关于沙库尔的整本书的赞歌，赞扬他"挑战对黑人身份的狭隘艺术视野"和"艺术性地探索黑人道德和社会亚文化的吸引力和局限性"——这只是近年来出版的无数对说唱的谄媚论文中的一篇。全国英语教师委员会推荐在城市公立学校课堂中使用嘻哈歌词（正如在奥克兰、洛杉矶和其他城市的学校已经发生的那样），热情洋溢地说"嘻哈可以用作连接看似广阔的街道和学术世界之间差距的桥梁。"

但如果说我们在2003年——民权革命证明成功之后很久，在非洲裔美国人机会突飞猛进之时，当黑人发现自己处于社会和政界的高层——认为黑人孩子像俄罗斯人背诵普希金一样背诵暴力、性别歧视、虚无主义的歌词意味着进步，那我们就严重缺乏想象力。有些人也为黑人剥削电影辩护称其为革命性的，但时间的流逝已经暴露了这种说法的愚蠢。"《甜蜜甜背》的信息是，如果你能团结起来，站起来对抗白人，你就能赢，"范皮布尔斯曾告诉一位采访者。但赢什么？从我们在电影中看到的，甜蜜甜背所做的只是避免入狱——如果能得到比"踢白人的屁股"更有用的关于克服困境的建议就好了。关于说唱政治潜力的主张在未来看起来也会同样做作。将生活描述为除了"婊子和钱"什么都不是，这怎么会是进步？或者告诉易受影响的年轻黑人孩子——他们只要努力工作和学习就会发现每一扇门都向他们敞开——打爆对手的头是"真实的"？在充斥着猖獗非婚生率和极低结婚率的社区中，说唱的性别歧视有多大的帮助？

当你想起来，说唱是对压迫的真实呐喊这一想法更加愚蠢：美国黑人在过去有更多要感到沮丧的事情，但从未创作或享受过像50 Cent或N.W.A.那样虚无主义的音乐。相反，黑人流行音乐几乎总是肯定性和充满希望的。在像埃塞俄比亚或刚果这样充满巨大痛苦的地方，我们也没有发现这种暴力的音乐——除非是进口的美国嘻哈。

鉴于嘻哈世界的反射性异化，它明确的政治努力，如其所是，几乎不是进步的，这并不令人惊讶。西蒙斯创立了"嘻哈峰会行动网络"（HSAN），将说唱明星和歌迷聚集在一起，以打造"嘻哈与政治之间的桥梁"。但HSAN的政策立场大多是陈旧的陈词滥调。例如，固守着城市学校因吝啬的种族主义白人建制派提供不足资金而失败的早已被否定的理念，HSAN与教师工会联手抗议纽约市长布隆伯格提出的教育预算，称其缺乏慷慨。HSAN还因布什总统入侵伊拉克而抨击他。它甚至大声抗议在说唱CD上贴上限警告家长其中淫秽语言的警示标签。为说唱歌手淫秽言论的权利而战：这也算某种革命！

好吧，有些人会承认，也许说唱在任何有意义的方面都不是进步的；但难道它不只是一群孩子发泄情绪，没什么好担心的吗？我认为这种回应太轻率了。随着音乐视频、DVD播放器、随身听、互联网、服装和杂志都使嘻哈成为一个人整个存在的伴奏，我们需要更认真地对待它。事实上，我会争辩说它对黑人社区严重有害。

虚无主义说唱的兴起反映了过去几十年里内城青年社区规范的崩溃。就在匪帮说唱走向其高峰时，邻里长者开始真正注意到他们失去了对年轻黑人男性的控制，这些人经常陷入帮派暴力和毒品交易的生活。一直到七十年代，贫民窟还是城镇中破旧的部分，尽管存在失业和不断上升的私生子率，还是有相当数量的人在尽力"把头保持在水面上"，正如老黑人情景喜剧《好时光》的主题曲所说的那样。

到了八十年代，贫民窟已经成为了一个无法无天的战区，在那里黑人是自己最大的敌人。当然，把这种可悲的螺旋下降归咎于嘻哈是愚蠢的，但通过美化"战区"中的生活，它使许多困在那里的孩子更难摆脱困境。看到一个像肖恩·康姆斯这样的特权明星像一个街头暴徒那样行事，告诉那些孩子，即使有了无法想象的财富，也没有什么比贫民窟的病理更真实。

嘻哈"身份"中表达的态度和风格使黑人无法进步。几乎所有的嘻哈，无论是否是匪帮说唱，都是以一种傲慢、对抗性的韵律传达的，这种韵律正迅速成为——正如肯德基那些吵闹的孩子所证明的——年轻黑人男性中的常见说话风格。同样，说唱表演者甩胳膊、挥舞手势的方式已经进入了许多年轻黑人随意的姿态中，成为他们自我表达的组成部分。这样的说话方式和姿态的问题在于，它们让潜在雇主对年轻黑人男性保持警惕，并可能阻碍年轻黑人与同事和客户舒适互动的能力。黑人社区经历了太多，不能为了对抗性嘻哈"身份"的短暂快感而牺牲向上流动性。

在更深的层面上，关于黑人已成为传播自己有损形象——我敢说是种族主义的——形象的主要代理人这一点，确实有令人不安和悲剧性的东西。说唱大师拉塞尔·西蒙斯声称"美国文化中最酷的东西——无论是语言、穿着还是态度——都来自下层阶级。一直如此，也将永远如此。"然而在过去糟糕的日子里，黑人经常抱怨——有些道理——媒体经常简单地将黑人描绘成未开化的。今天，即使电视和电影描绘了黑人在各个层面的成功，嘻哈却传达了黑人就是……未开化的信息。我发现显著的是，那些大声疾呼种族主义的人群并未谴责它。

对于那些坚持认为即使社会无形的结构也强化种族主义的人，举证责任应该落在他们身上，去解释为何嘻哈的血腥和性别歧视歌词和视频以及许多说唱歌手的犯罪行为不会对白人对黑人的看法产生强大的负面影响。

可悲的是，一些黑人领袖似乎根本不在乎说唱传达了什么样的教训。考虑一下萨凡纳的黑人高中，它们在本土说唱歌手卡莫弗洛奇今年早些时候被谋杀前曾多次邀请他作为客座演讲人。这是他的代表性歌词：

给我车钥匙，我准备好开战。
当我们骑上这些黑鬼，像雪茄一样烧掉他们的屁股。
用格洛克冲你的街区，用Tech清理现场……
你以为我在开玩笑，看你还笑不笑当手枪在冒烟——
让你的头敞开大口
你的骨头被打断……

超过少数几个邀请这位"艺术家"去对萨凡纳易受影响的青年讲话的"关心黑人"的人们，大概会是第一个大声疾呼"白人在媒体上如何描绘黑人"的人。

许多嘻哈辩护者不仅不谴责说唱中当今黑人剥削的猖獗刻板印象，还玩弄"白人也是"的把戏。他们指出《教父》电影或《黑道家族》作为暴力和低俗在美国流行文化中普遍存在的证据，因此单独挑出嘻哈来谴责只是偏见。然而，这样的辩护是可怜地无力的。没有人真在《黑道家族》中寻找一种生活方式来效仿或一个政治项目来采纳。但对于许多倡导者来说，嘻哈以其革命、社区和政治的幻想，不仅仅是娱乐。它构成了年轻黑人身份的基石。

争辩说嘻哈不是"黑人"音乐，因为其大多数购买者是白人，或者因为"嘻哈革命"在名义上对所有人开放，无论肤色，也是不行的。白人购买的嘻哈唱片比黑人多，这并不令人惊讶，因为白人在全国的数量远远超过黑人。更重要的是，任何声称说唱不是黑人音乐的人，都需要将这种说法与黑人对像埃米纳姆这样的白人说唱歌手的普遍警惕——他被指责"偷我们的音乐并把它还给我们"——调和起来。

不久前凌晨2点，在纽约地铁上，我看到了另一个场景——比我在肯德基遇到那些吵闹说唱少年的经历更令人沮丧——捕捉到了说唱破坏性的本质。一个年轻黑人男子进入车厢，开始大声说唱——粗俗地、傲慢地——伴随着通常的狂野手势。这持续了五分钟令人讨厌的时间。当没人理会他时，他移到另一节车厢，一直滔滔不绝地念着他那蹩脚的打油诗。这就是这个年轻黑人男子向世界展示的信息——他的演说，如果你愿意这么称呼的话。

任何将这种行为视为通向更美好未来的道路的人——任何人，比如戴森教授，坚持认为嘻哈是对"产生对暴徒角色需求的社会"的紧迫"批判"——应该退后一步，问问自己，民权时代的黑人在缺乏嘻哈革命的情况下究竟可能错在哪里。他们创造了我生活在其中并蓬勃发展的平等、奋斗和成功的世界。

嘻哈所衍生的大多数表现什么也没有创造，而是像一股破坏力一样在行动。
