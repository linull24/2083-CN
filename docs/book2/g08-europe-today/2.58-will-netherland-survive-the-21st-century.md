# 荷兰能否在21世纪生存？
# 2.58 Will Netherland Survive the 21st Century?
Orginal: 596-600

你明白了吗？我们不能为我们的过去感到自豪，因为它全是坏的；我们不能用英国式幽默，因为它是冒犯性的；我们不能用轻描淡写来应对危机，因为国家级顾问大军说我们必须为每一件小事无法控制地抽泣。

荷兰能否在21世纪生存？
作者：Fjordman

让我们看看荷兰的情况。伊斯兰习俗在那里获得了越来越多的公众接受。有说法要将伊斯兰节日设为公共假日，因为荷兰是一个"犹太-基督教-伊斯兰"社会，不管这是什么意思。还有计划建立穆斯林专属医院，前司法部长皮特·海因·唐纳表示，荷兰人应该给予穆斯林更多自由来按照他们的传统行事："对我来说很清楚：如果三分之二的荷兰人口明天想要引入伊斯兰教法，那么这种可能性就应该存在，"据他说。"说'这是不允许的！'将是一种耻辱。"

这个想法绝非牵强附会。非官方的伊斯兰教法法庭在欧洲主要城市已经运作多年，并逐渐获得官方认可。2008年9月，据透露英国当局将在某些事务上正式接受伊斯兰教法法庭的裁决。这意味着英国不仅在文化上，而且在法律上已经向它的新穆斯林主人投降了。政治精英们将对持续伊斯兰化毫不抵抗。

英国是西方世界最糟糕的国家之一，考虑到许多其他国家也有多糟，这已经很说明问题了，但它属于一个不断增长的、当局或多或少已经向伊斯兰统治投降的国家群体，公开推广伊斯兰"文化"，并骚扰那些抵抗的人。可以将瑞典、荷兰、比利时，可能还有法国加入这个名单，这个名单还在不断扩大。挪威、西班牙、奥地利、德国和其他国家也不远落后。

荷兰，几个世纪以来一直是寻求更多思想自由的人的避风港，正在成为越来越极权主义的社会，这是大规模移民的直接结果，特别是穆斯林移民。这就是为什么富有洞察力的汉斯·詹森，乌得勒支大学现代伊斯兰意识形态教授，说一个希望继续存在的和平社会"将不得不找到一种通过非和平手段来保卫自己免受那些不和平的人的侵害的方式。"据詹森说，穆斯林原教旨主义者经常发出威胁，但荷兰媒体对此保持沉默。

公共广播公司NPS正在制作一个电视节目，目标是给荷兰人口一个对穆斯林和其他移民群体的积极看法。与此同时，2008年，鹿特丹警察队的一名摩洛哥裔警察被揭露为摩洛哥情报部门的间谍。他领导了一个项目，培训了57名摩洛哥问题少年成为鹿特丹机场的地勤人员。公诉机关决定不起诉他，据称是因为这可能对"多元文化"社会产生负面宣传。即使在这件事被揭露后，这名被揭露为外国间谍的警察仍继续在鹿特丹机场工作。机场管理层认为没有理由拒绝给他工作。

相比之下，荷兰工党试图压制一名当时为该党成员的人，他正在为穆斯林叛教者的权利而斗争，因为他们担心他会给党带来选举损失。与西方世界大多数其他左翼政党一样，他们获得了不成比例的移民选票。所谈之人，勇敢的前穆斯林埃赫桑·贾米，在2008年春季决定关闭他的前穆斯林组织，这些前穆斯林挑战传统上对脱离伊斯兰教者处以死刑的制度。他声称人们因为穆斯林的威胁而不敢加入该组织。

阿姆斯特丹的摩洛哥裔聚居区斯洛特瓦特一直被骚乱和纵火所困扰。就像在瑞典、法国、英国和其他地方的类似区域一样，消防队和救护车服务的应急人员在进入时有遭受人身攻击的危险。西欧主要城市以其迅速扩大的"禁区"而臭名昭著，从伯明翰、伦敦和布拉德福德，到马赛、巴黎、安特卫普和布鲁塞尔，再到哥本哈根、柏林和汉堡。现在有许多区域即使是警察在没有大量后备支援的情况下也不敢进入。在某些城市区域，消防队和救护车服务不愿在穆斯林社区回应呼叫，那里的"青年"纵火焚烧公共建筑，然后在消防队到达时设伏袭击。

2008年9月中旬，荷兰豪达市的公交服务宣布不会通过奥斯特韦克社区。这一决定是在收到公交车司机的投诉后做出的，他们报告说，在穿越该社区的线路上，他们被摩洛哥青年吐口水、威胁和抢劫，这些青年还有系统地踢公交车。

此前已经披露，在荷兰的摩洛哥人、土耳其人和其他穆斯林从他们的祖国接受指导。土耳其裔人收到了来自土耳其的关于如何在荷兰选举中投票的指示。

同样在2008年9月中旬，政治家赫尔特·维尔德斯在议会的一次演讲中说，摩洛哥人正在殖民荷兰。据维尔德斯说，摩洛哥人去那里不是为了融合，而是"为了征服荷兰人"并统治他们。"我们把国家输给了摩洛哥渣滓，他们一生都在虐待、吐口水和猥亵无辜的人，"维尔德斯说。"他们愉快地接受我们的救济金、住房和医生，但不接受我们的规则和价值观，"他说。据他说，存在"两个国家"。内阁的国家是"气候歇斯底里和不可控的伊斯兰化"的国家。另一个国家，"我的国家，"赫尔特·维尔德斯说，"是那些必须买单、被伊斯兰街头恐怖分子抢劫和威胁的人民的国家。"

演讲开始七分钟后，电视广播出现了"技术错误"，这个错误神秘地在维尔德斯完成演讲的那一刻得到解决。国家电视台在支持多元文化观点和认为维尔德斯是邪恶的方面是出了名地有偏见。

正如我在我的在线文章"民主与媒体偏见"中所写的，在民主社会中，新闻界，第四等级，本应确保政府妥善履行职责，并提出公众关心的问题。实际上，我们现在似乎面临的情况是，政治精英与媒体合作，确保某些话题得到不充分或不平衡的关注，而其他话题则被完全排除在议程之外。他们共同构成了一个新的政治阶级。

在特立独行的政治家皮姆·福图因崛起之前，荷兰政局在很大程度上是一个封闭的俱乐部，其成员无论党派归属，在最广泛的意义上共享相似的观点。大多数记者属于同一个俱乐部。如果大多数民众在敏感问题上不完全同意这种精英——而其中最敏感的问题是穆斯林移民——这几乎无关紧要。由于所有掌权者基本一致，人民的意志可以被安全地忽视。记者和竞争对手政治家们——注意他们是如何步调一致地行事的——将福图因抹黑为危险的"右翼极端分子"。这间接导致了他被一名左翼活动分子谋杀，该活动分子声称他是代表穆斯林杀他的，因为他对少数群体是"危险的"。

皮姆·福图因被政治、文化和媒体精英间接谋杀，而电影制片人特奥·范霍赫被穆斯林谋杀。议员阿娅安·希尔西·阿里已被赶出国家。批评伊斯兰教的议员赫尔特·维尔德斯还在那里，但他受到与福图因类似的关于他是种族主义者的抹黑，每天收到来自穆斯林的威胁，以及来自建制派对减少对穆斯林移民批评的不那么微妙的暗示。2001年至2008年之间的荷兰精神似乎已经被打破，事情正在慢慢恢复正常。扩大的政治精英再次牢牢控制着公共辩论，令人难堪的农民起义已被成功镇压。

也许荷兰拯救自己的机会随着福图因而死去了。我希望不是。我一直热爱荷兰，如果一个曾如此伟大的国家——投入了如此多的时间和精力来将海水挡在外面——被伊斯兰教法野蛮的浪潮所摧毁，那将是非常可悲的。

2008年5月，以笔名格雷戈里乌斯·内克斯霍特写作的漫画家成为现代西方世界有史以来第一位被逮捕的漫画家。他在阿姆斯特丹的家中被捕并被拘留审讯，涉嫌"发表歧视穆斯林和深色皮肤人群的漫画"。与此同时，阿姆斯特丹市开发了教学材料，警告儿童不要相信批评伊斯兰教的政治家赫尔特·维尔德斯的政治。维尔德斯议员称这场运动"令人作呕"。

维尔德斯2008年3月的电影《Fitna》引起了全球穆斯林强烈反应，以及西方世界迪米绥靖主义者的谴责。虽然这部短片除了引用古兰经和穆斯林领袖的声明外没有做任何其他事情，联合国秘书长潘基文却谴责它是冒犯性的反伊斯兰。潘基文在一份声明中说："没有任何理由为仇恨言论或煽动暴力辩护。这里的问题不是表达自由的权利。"联合国人权事务高级专员路易丝·阿尔布尔加入谴责《Fitna》的基调和内容，并指出荷兰和其他国家应禁止任何构成煽动歧视或敌意的种族或宗教仇恨的鼓吹。用简单的话说，他们应该禁止批评伊斯兰教。

已经有例子表明，小型荷兰网站因承载读者对伊斯兰教和穆斯林的批评性评论而被起诉，从而为在互联网上压制言论自由树立了法律先例。尽管在伊斯兰在线论坛上经常发布更具攻击性的材料，但从未受到任何惩罚。类似的发展正在其他欧洲国家发生。这不仅得到国家当局的鼓励，也得到欧盟官员的鼓励，他们表达了对博客和类似网站进行更多"监管"的愿望，因为这些网站往往比主流媒体更批评多元文化主义、大规模移民和总体欧盟政策。

这意味着荷兰当局正在屈服于伊斯兰国家的要求，并跪倒在他们的新穆斯林主人脚下。毫无疑问，存在对警察和其他人施加的相当大的政治压力，要求执行多元文化言论守则并压制本土异议者。既然我们清楚地看到穆斯林可以发布关于本土人的负面评论，但本土人不被允许发布关于伊斯兰文化的负面评论，这意味着本土人在自己的国家事实上是二等公民。这恰好是他们根据伊斯兰教法应有的地位，这意味着当局现在是伊斯兰法律执行者。

一些观察者说，西欧的政治精英"无力"阻止街头暴力。但他们在压制对持续大规模移民的批评方面却咄咄逼人，这表明与其说他们是无力的，不如说他们是对本土人积极敌对的。

在比利时布鲁塞尔，穆斯林移民帮派每天都在骚扰本土人。我们最近有几个案例，本土女孩在欧盟首都中心地带被移民轮奸，然而当本土人想要在2007年9月11日抗议自己大陆的伊斯兰化时，示威被布鲁塞尔社会主义市长禁止了，他的执政党被穆斯林大量渗透。那些试图继续和平抗议的人被警察逮捕。

数十名男孩和女孩在乌得勒支奥弗费赫特区遭到一群年长男孩的系统性虐待、恐吓和猥亵。8岁、9岁和10岁的儿童被拖进灌木丛中，被迫对年长几岁的男孩进行性行为。涉及的大多数儿童是摩洛哥裔背景。市议会成员说这些问题没有得到解决。"这已经持续近一年了。人们只是谈啊谈啊谈。什么都没做，"她说。

2007年3月，乌得勒支市本土荷兰居民骚乱，抗议穆斯林青年的骚扰和政府不作为以阻止这种情况。当局立即通过封锁该地区并安装监控摄像头来控制荷兰非穆斯林来镇压骚乱，但他们几乎没有采取任何行动来解决移民帮派暴力的根本问题。这种情况远非特例。

这类事件表明，西欧各地的当局现在致力于实施持续的大规模移民和多元文化主义，而不管本土人怎么想。如果他们反对，他们就会被压制。荷兰人以非常大的优势对将正式拆除他们国家的拟议欧盟宪法投了"反对票"，爱尔兰和法国选民也是如此，但他们完全被忽视了。与此同时，欧盟精英顺从地回应伊斯兰国家的呼吁，要求禁止针对伊斯兰教的"刻板印象和偏见"。欧洲政治精英执行我们敌人的议程，忽视自己人民的利益。因此他们是合作者和叛徒，应当被相应地对待。

有趣的问题是：精英们仅仅是绥靖伊斯兰教——他们肯定在这样做——还是他们正在为自己的目的积极推广和利用它？正如伊朗前穆斯林阿里·希纳在他优秀的著作《理解穆罕默德》中所说："伊斯兰教是统治的工具。在穆罕默德之后，其他人为完全相同的目的利用他的邪教。"也许这在欧盟也在发生？

阿里·希纳拒绝多元文化的概念，即所有文化（欧洲文化除外，它是邪恶的，应该被根除）都值得保留："如果任何文化需要被保护，那就是西方的希腊-基督教文化。正是这种文化面临灭绝。正是仅对这种文化，我们才拥有启蒙运动、文艺复兴和民主。这些是我们现代世界的基石。不保护这种文化将是一个可怕的错误。如果我们什么都不做，我们面临的未来将是民主和宽容消退，伊斯兰教更原始的野蛮本能将征服人类。所有文化并非生而平等……伊斯兰教不是一种文化。它是文化的对立面。它是野蛮、暴行和不文明。伊斯兰文明是一个矛盾的修辞，而伊斯兰恐怖主义则是冗余同义反复。我们欠西方文化我们的自由和现代文明。现在正是这种文化受到攻击并需要保护。"

正如我在我的文章"福利国家已死，福利国家万岁"中所说的，美国人说"我们信靠上帝"。福利国家在某种深层的潜意识层面上，是否成为了上帝的替代品？一个无处不在的国家取代无处不在的上帝？欧洲人在奥斯维辛和一战战壕中丧失了对上帝的信仰。我们不再信靠上帝，所以我们把信任交给国家，在这个有着如此动荡历史的大陆上创造一个小小的安全绿洲。

口号是"从摇篮到坟墓的安全"。但现在福利国家在西欧大部分地区显然无法提供经济或身体安全，至少对本土人来说是这样。它为穆斯林移民支付更多摇篮，却将一些本土人过早地引向坟墓。税收是一种伪装的吉兹亚——顺从的非穆斯林在伊斯兰统治下缴纳的人头税，从我们这里拿走，交给那些殖民我们土地的人。

福利国家滋生被动性和对国家议程的服从，因为有太多人依赖它谋生。对统治者来说，这可能相当有用。福利国家的公开目的是减轻贫困，但我们应该记住，一个处理生活方方面面的强大国家官僚机构，将大量权力留给官僚机构的顶层人物，统治着那些被数十年的国家灌输和对其私人生活的干涉所安抚和阉割的人。我怀疑欧洲人容忍强大的欧盟官僚机构管理欧洲大部分事务的原因之一是，我们已经在国家层面上习惯了这一点。

我对西方人的总体建议是立即武装自己，首先是用对敌人的了解和对自身文化和遗产的自豪来武装精神，但也要用枪支和使用它们的技能进行身体武装。弗里德里希·尼采在19世纪宣称"上帝已死"。在21世纪初，公平地说"国家已死"，这是我们在另一个上帝死后信任的替代上帝。

每一天我们都有更多证据表明当局完全无法保护公民的任何表面安全和自由。国家似乎仍然能做的唯一事情是向我们的孩子灌输对自己文明的仇恨，并拿走我们的钱以便给那些殖民我们国家、在言语上和身体上虐待我们孩子的人。

我仍然押注英国，或者可能是丹麦，成为第一个因穆斯林移民而面临内战的西方国家，但荷兰也是一个潜在的候选。我只是想知道荷兰人是否在心理上已经是一个被打败的民族。他们的政治精英已经选择了正式投降，将强制执行伊斯兰教法，并将所有不同意这一点的人作为"极端分子"加以禁止。本土荷兰人要么必须反击，要么离开自己的国家，远距离地看着它死亡，正如相当数量的荷兰人已经做的那样。

现在西欧发生的情况是契约不再被遵守的教科书案例。本土人为不再保护其边界的民族国家缴纳极高税率，这些国家既不愿意也无法维护最基本程度的法律和秩序。无论如何，法律不再以我们的利益为念而通过，而是由专门反对我们利益的多诚的多元文化主义者和全球主义者通过的。欧盟就是暴政的完美定义。

正如约翰·洛克在《政府论》（下篇）中所说：人们进入社会的原因是保护他们的财产，他们选择并授权立法机构的目的，是有法律可以制定，规则可以……
