# 挪威宗教裁判所——午夜太阳之地的日落
# The Norwegian Inquisition - Sunset in the Land of the Midnight Sun
Orginal: 611-613

作者：Fjordman

挪威议会（Stortinget）在2005年4月通过了一项新的《反歧视法》。该法以相当明确的措辞[1]规定，在涉嫌因宗教或族裔而直接或间接歧视的案件中，挪威人在被证明无罪之前是有罪的。对我来说，他们竟然被允许通过这样的立法是令人惊讶的。在所有国际法中，一个人在被证明有罪之前应该无辜，难道这不是基本的部分吗？我们的政治家们难道不是在剥夺挪威人最基本的人权吗？然而，我听说有说法称这样做在技术上是合法的。该法案于4月通过，得到了议会中所有政党的批准，即超过80%的议员，唯一的例外是右翼进步党。进步党移民事务发言人佩尔·桑德伯格深感失望[2]，并担心新立法的后果。"这项法律将危及普通守法挪威公民的权利。举证责任倒置原则意味着挪威人在歧视方面是有罪的，除非他们能证明自己无辜。这将导致许多无辜者被定罪。举证责任倒置还与支付赔偿的责任结合在一起，这意味着无辜者有可能为他们没有做过的事情支付巨额赔偿。"

鉴于仅仅几个月前的一项移民调查[3]表明，绝大多数人不认为他们在挪威受到歧视，这项法案为何需要就不得而知了。种族主义似乎不如先前认为的那样普遍。我们究竟为什么首先要对他们完全失败的伊斯兰文化表现出这种荒谬的敬畏和尊重？为什么来自这个星球上最发达国家的人必须向来自最落后国家的人卑躬屈膝？如果他们的伊斯兰社会如此伟大，为什么来自这些国家的人会烧掉他们的护照，用酸处理手指以去除指纹，所有这些都是为了进入我们的国家？

"反种族主义"组织在新法律中被赋予了重要角色。有一个新的、国家资助的平等监察员，将负责执行它，并强制所有拒绝遵守的雇主服从。换句话说，一个多元文化宗教裁判所。内阁部长埃尔娜·索尔贝格，她之前曾呼吁在挪威建立沙里亚委员会[4]，提出了这项新法案。它将涵盖从工作场所到住房市场的一切。在最近的一个案例中，一家当地家具店不允许一名女员工佩戴头巾，认为这违反了商店的着装规定。索尔贝格的提案将加强法律[5]，并要求被指控者提供自己无罪的证据。索尔贝格认为，现有法律已经规定雇主不得阻止女性佩戴头巾（如果她们的宗教要求的话）是非法的。她提出的法律"将使这一点更加明确"。这项法律可能为特别是穆斯林移民的各种不合理要求打开闸门，他们将获得由挪威议会提供的向雇主敲诈的许可。例如，他们现在可能声称，如果没有提供专门的祈祷室，那就是"歧视"。已经有一些穆斯林出租车司机要求奥斯陆机场提供单独的祈祷室[6]，在那里他们可以在工作时间祈祷，但得到了否定的回答。索马里出租车协会领导人阿里·哈桑认为这是歧视性的和不可接受的，并计划就此提起诉讼："我们认为我们有权在工作时间祈祷。我们要求得到一个可以祈祷的房间，而不会失去在出租车排队中的位置。"与此同时，携带导盲犬的盲人[7]发现，在穆斯林占出租车司机很高比例的奥斯陆地区，打车越来越困难。

令人沮丧的是，挪威当局强制所有非穆斯林在其工作场所接受伊斯兰面纱（希贾布）。许多非穆斯林觉得希贾布具有冒犯性，甚至一些穆斯林也是如此。面纱不是"一块布"。它是在适当的、顺从的穆斯林妇女与妓女——不值得尊重、自找强奸的非伊斯兰妇女——之间的一条分界线。面纱更应被视为极权主义运动的制服，以及攻击运动以外者的信号。丹麦哥本哈根的一位伊斯兰穆夫提在公开宣称拒绝戴头巾的妇女是"自找强奸"[8]后引发了政治抗议。显然，他不是欧洲唯一这样想的穆斯林。挪威报纸《每日新闻》在2001年报道，奥斯陆65%的强奸案[9]是由"非西方"移民实施的——在挪威，这一类别主要由穆斯林组成。文章引用了奥斯陆大学社会人类学教授乌尼·维坎的话，她说"挪威妇女必须为这些强奸承担自己的责任[10]"，因为穆斯林男性觉得她们的穿着方式具有挑衅性。这位教授的结论不是生活在西方的穆斯林男性需要适应西方规范，而是恰恰相反："挪威妇女必须认识到我们生活在一个多元文化社会中，并适应它。"2005年1月，挪威媒体报道[11]，2004年首都奥斯陆记录了有史以来最高的强奸指控数量。奇怪的是，现在没有提到移民如何在强奸案件中严重比例过高。为什么不呢？除非2001年至2004年间强奸挪威妇女的移民数量突然出现无法解释的下降（这不太可能），否则2005年的统计数据至少应该和以前一样惊人。如果它们没有被披露，那只能意味着"某人"不喜欢2001年呈现的数字[12]，并决定将其掩埋。那个"某人"一定是在警察等级制度中地位很高的人，甚至可能在政府中。同样的事情在瑞典[13]发生。所以最终，年轻斯堪的纳维亚妇女的安全被牺牲了，以维持多元文化社会的光鲜形象。

像毛拉·克雷卡尔[14]这样的激进伊斯兰主义者确实居住在挪威。此外，挪威警方已经向进步党领袖卡尔·I·哈根发放了移动安全警报[15]。他们担心他是因为去年夏天发表的一些反伊斯兰言论而成为恐怖分子的目标。哈根批评了伊斯兰教，看不出与基督教中的道德和正义概念有任何相似之处。哈根还说，如果以色列在中东失败[16]，欧洲接下来将屈服于伊斯兰，如果伊斯兰原教旨主义者得逞的话。基督徒应该支持以色列并反对伊斯兰对欧洲的侵入[17]。以前所未有的一步，一群穆斯林驻挪威大使[18]在致《晚邮报》的一封信中抨击了卡尔·I·哈根，声称他冒犯了全世界13亿穆斯林。特别是巴基斯坦大使，之前就以不可接受的方式干涉挪威内部事务，试图指导和恐吓一位巴基斯坦裔的挪威政治家，该政治家敢于表达她支持在挪威学校禁止伊斯兰面纱的观点。[19]

诚实思考网站（HonestThinking.org）[20]编辑奥勒·约根·安芬森博士认为，挪威当局在移民数量预测上做了手脚。根据他的数字，如果当前趋势继续下去，挪威族裔将在2050年前在自己国家成为少数族裔。挪威穆斯林的数字在15年内翻了四倍[21]，这意味着年增长率超过9%。一位挪威研究人员警告说，族裔帮派[22]可能给挪威带来伴随移民潮进入美国的移民相关有组织犯罪。英格-莉丝·利恩博士的结论是，族裔群体本身对此感到担忧。奥斯陆曾经是一个安全的斯堪的纳维亚城市，今天看起来更像朱利安尼之前的纽约市。土生土长的挪威人正在大批安静地搬出[23]奥斯陆内城的移民贫民窟。与移民的紧张关系甚至已经蔓延到较小的城镇[24]。这一趋势与邻国瑞典的情况相同，在瑞典，几个城市现在处于崩溃边缘[25]。挪威已经经历了似乎是一次未遂的伊斯兰恐怖袭击[26]。与此同时，有新的立法被提出，反对针对特定群体做出的"歧视性"言论[27]，甚至在互联网论坛和网站上。这意味着挪威人不能过多谈论穆斯林移民摧毁国家的事情，因为那是种族主义，因此是非法的。在许多国家越来越认为多元文化意识形态是一个错误的时候，这里的当局却通过将其定为官方国家意识形态并禁止一切反对来回应。严重左倾的挪威媒体对此出奇地安静，不对这种侵犯言论自由的行为大惊小怪。

斯堪的纳维亚是一个失去的乌托邦。以前安静的斯堪的纳维亚国家现在遭受伊斯兰恐怖威胁和对批评伊斯兰的人的死亡威胁[28]。挪威今年庆祝独立100周年[29]。从这项新的反歧视法和失控的穆斯林移民来看，也许这个周年纪念应该被称为"从独立到殖民化"。在他们的妇女因移民帮派而不再在街头安全的同时，当局却通过使挪威人事实上成为自己国家中的二等公民来回应。他们利用自己的人民作为他们在联合国官僚体系中个人职业生涯的垫脚石。自大、虚伪的斯堪的纳维亚小丑，得意洋洋地讲授如何创造完美社会，而他们自己的公民却发现在自己的大城市生活越来越困难。

致美国的读者们：厌倦了听到欧洲人批评你和你的政府是多么"法西斯"吗？那么，刚刚通过一项法律——得到超过80%立法者支持——规定自己公民在被证明无罪之前有罪的国家，可不是美国，而是挪威。想报复一些自鸣得意的反美欧洲人吗？这是你的机会。如果你想就此事写些什么，请在我的Fjordman博客上留言。我将尽可能帮助翻译。不幸的是，材料不乏其数。
