# 唯一的英格兰人慈善组织被标为种族主义，而215个索马里组织受到欢迎
The only English charity organisation is branded as racist while 215 Somali organizations are welcomed
唯一的英格兰人慈善组织被标为种族主义，而215个索马里组织受到欢迎

多元文化主义的表现形式导致群体导向利益组织的代表程度极其不成比例。唯一一个为英格兰人利益工作的慈善组织被嘲笑和迫害，被贴上了种族主义组织的标签，而数千个其他组织（215个为索马里人利益工作，92个为巴基斯坦人，甚至144个为黑人利益）从未因被称为政治正确/文化马克思主义/多元文化主义的教条而受到批评。这种趋势进一步强调并证明了多元文化主义是一种反欧洲的仇恨意识形态，旨在摧毁欧洲的传统、文化和认同。

同样的趋势在其他西欧国家也普遍存在。

居住在英格兰和威尔士的芬兰人有11个专门为芬兰人利益和福祉工作的慈善组织。中国人有164个这样的慈善组织，索马里人有惊人的215个，此外还有"约1200个"犹太慈善组织，但在表格底部的是英格兰人，他们（目前）构成了英格兰和威尔士人口的绝大多数，只有1个（就一个！）称为Steadfast Trust的慈善组织，在一个日益碎片化和分裂的社会中为英格兰人及其利益工作。

10月8日，Steadfast Trust的一名支持者向慈善委员会提出了一项信息自由请求，询问有多少慈善组织像Steadfast Trust一样，将其受益人分类代码之一指定为"为特定种族或族裔出身的人"。

由于一个幸运的巧合，委员会已经在进行这一领域的研究，并提供了一份中期结果表格。

慈善委员会约有159,600个注册慈善组织，但应该注意，由于豁免慈善组织的注册和非活跃慈善组织的移除，这一数字相当不稳定。

大约15,500个慈善组织勾选了"为特定种族或族裔出身的人"的代码，但许多组织被排除在研究之外，因为它们仅在海外运营。约有6,860个被确定的慈善组织在英格兰和威尔士的国家范围内运营。

*EHRC已被告知这一类别术语（即"黑人"），因为担心它可能违反1976年《种族关系法》，该法据信禁止使用肤色作为区分受益人的手段。

慈善组织的目的是为其目标群体的利益和福祉而工作。正如完整表格所示，英格兰人甚至没有被列出（意味着在整个英格兰和威尔士有不到10个），这表明他们在特定基础上未得到良好的族裔代表。这既令人惊讶又令人遗憾，因为它意味着最大的族裔群体被剥夺了权利并可能被忽视。

希望Steadfast Trust（作为唯一已知的为英格兰人民服务的慈善组织）能够以微小的方式解决这种不平衡。Steadfast Trust受托人林恩·乔利说：

"可悲的是，极少数人对我们的慈善组织可能具有歧视性提出了担忧。当然，事实并非如此；事实上，歧视发生在英格兰多数族裔群体不被允许拥有为自己人民服务的慈善组织时，导致我们被严重低度代表。

我们相信，我们通过教育和活动提供重要的文化认同感的工作将对整个社会产生积极影响，我们欢迎所有给予的支持。"
