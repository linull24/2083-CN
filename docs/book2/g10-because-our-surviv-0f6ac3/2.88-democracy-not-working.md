# 2.88 民主失灵

2.88 Democracy not working

作者：Fjordman

目前，在我看来，民主制度在任何西方国家都没有正常运作。它在西欧基本上已经死亡，那里大部分真正的权力无论如何都已转移到欧盟的非选举机构手中。几乎所有西方国家都失去了对其边界的控制。这不是一个可持续的局面。你可以将你的政治制度称为民主、独裁、共和、君主制或任何你想要的，但一个不控制自己领土的国家终将死亡。这是不可避免的。

运输的全球化对我们国家施加了严重压力，这种压力在几十年前是难以想象的，这使情况变得更糟。当公元一世纪末第一部基督教福音书被写下来时，罗马帝国的人口约为6000万。这相当于21世纪初的全球年人口增长。换句话说：全球人口每年增长一个罗马帝国。我们的制度不是为这样的数字而设计的。它需要根本性的变革，否则很快就会崩溃成内战或独裁，或两者兼有。我们还面临这样一种情况，一些左翼政党特别蓄意地引进穆斯林和其他人，因为他们绝大多数投票给左翼政党。一个引进敌人会获利的政治制度显然是不可持续的。

当我批评民主时，这不应被理解为我信奉精英统治。我之所以批评它，是因为它显然不能自动确保言论自由和生命及财产的安全，而这才是真正自由的标志。另一个问题是，它并不总是长期决策的最佳制度，因为人们倾向于短期利益。然而，我仍然认为，应该有强大的真正公共影响力元素，以限制绝对统治者和权力滥用的潜力。在欧盟方面，我们已经明显过分偏向于后者，统治精英们巧妙地消除了对其权力的任何制约。

民主制度有重大缺陷，但只要存在demos（人民）的意识，即一个拥有共享身份和共同利益的人民群体，它在某种程度上还是有效的。我们现在目睹的正是这个demos的逐步解体，从顶层开始。强大的团体往往与其他国家的精英有更多共同点，而不是与自己国家的普通公民有共同点。如果你不再相信你的国家是一个具有特定文化的真实实体，它就仅仅成为获取权力的工具，你全球职业生涯的垫脚石。如果没有前政治忠诚、情感纽带甚至支持民族国家的务实利益，民主制度就成为向国内外的朋友分发好处、在执政期间剥削选民并希望在此过程中确保有利可图的国际职业生涯的工具。你将没有多少道德抑制来引进外国选民以维持权力，或者因为给你财政支持的商业伙伴希望这样做。这个过程与技术全球化有关，但在自我憎恨的西方，它比其他任何文明都走得更远。

因此，仍然认同自己民族国家的普通公民们不断选出背叛他们信任的人。由于精英们几乎不认同他们本应服务的国家，给他们更多权力只会使情况变得更糟，正如欧洲已经发生的情况那样。腐败和无能的个人永远存在。如果你偶尔遇到一个腐败的领导人，你面对的是一个有缺陷的个人。如果你一次又一次地不断遇到腐败的领导人，你面对的是一个有缺陷的制度。我们的政治制度现在有深层次的缺陷。问题是我看不到简单修复它的方法。

最重要的一点是认识到民主是一种工具，我们用来达到目的的手段。现在太多人将其与目的本身混淆了。"民主"已经开始意味着某种好的东西，每个人都想要的东西，有点像性或巧克力。但没有理性理由认为普选民主一律是好的，可以在所有情况下同样成功地应用，这是美国人在伊拉克犯下的巨大错误。

任何政治制度首先必须确保它所服务的共同体的生存，持续的物理存在。然后才是以最佳方式确保这个共同体的繁荣和自由。然而，当我审视今天西方国家的状况时，我看不到民主总是确保我们的自由或繁荣，而且在很多情况下，它运作得如此糟糕，以至于威胁到我们的生存本身。也许为了确保我们的持续存在，我们需要用工具箱中的其他工具来补充民主。
