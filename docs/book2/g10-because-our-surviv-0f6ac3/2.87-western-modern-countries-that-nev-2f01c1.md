# 2.87 "西方"现代国家中从未采纳多元文化主义的——日本、韩国和台湾
# 2.87 "Western", modern countries that never adopted multiculturalism - Japan, South Korea and Taiwan
Orginal: 693-694

日本和韩国二战后四十年来在没有移民的情况下实现的经济增长激增一直让移民和多元文化主义热衷者感到尴尬。1990年，当时的指定热衷经济学家朱利安·西蒙不得不承认："我不知道日本是怎么做到的。但我们可能不得不承认有些国家在其特征上是独特的。"
二战后西欧与这些国家保持了长期而密切的友谊。他们没有发起媒体运动将这些国家描绘成纳粹，没有在经济上抵制它们，也没有威胁如果它们不开始实施多元文化主义就入侵它们。因此，这与西欧嘲笑、骚扰和迫害自己国家内任何支持单一文化主义的个人和团体形成了鲜明的矛盾。
日本与多元文化主义
日本社会以其同质性意识形态，反对了被西欧和美国所采纳的多元文化主义。
根据联合国难民署的数据，日本在1999年只接受了16名难民，而美国接收了85010人用于重新安置。比日本小的新西兰在1999年接受了1140名难民。从1981年日本批准《联合国难民地位公约》到2002年，只有305人被日本认定为难民。

韩国与多元文化主义
韩国是世界上最族裔同质化的国家之一。如同日本和台湾，韩国拒绝多元文化主义的教义。大多数韩国人倾向于将国籍或公民身份等同于一个共享相同"血统"和历史的单一、同质的族裔群体的成员身份。共同的语言和文化也被视为韩国身份的重要元素。
多种族或多族裔国家的理念，如加拿大或美国，总体上遭到反对，许多韩国人觉得这种想法很奇怪甚至矛盾。

中国、非洲、伊斯兰世界与多元文化主义
欧洲的文化马克思主义者/多元文化主义者经常直接或间接地声称，所有反对多元文化主义的人都是种族主义者、法西斯主义者和纳粹分子。因此，按照他们的逻辑，中国、非洲和伊斯兰世界都是纳粹政权，因为它们从未实施多元文化主义。截至目前，只有欧洲国家（包括美国、加拿大、澳大利亚）和少数印度教和佛教国家是实际的多元文化主义政权（印度、泰国、菲律宾和其他几个）。那么为什么存在双重标准？为什么他们只使用法西斯/纳粹抨击修辞来瘫痪欧洲国家内及针对爱国团体和个人的所有抵抗企图？这个答案已在本简编中详细讨论。这种现象被称为自我憎恨情结，也被称为"欧洲内疚"，与马克思主义者要摧毁欧洲人、基督徒、异性恋男性的目标结合在一起。许多马克思主义者的一个主要信念是，一切欧洲的东西都是世界上的癌症，必须被根除。根据他们的逻辑：欧洲人是世界上所有邪恶的根源，所以他们必须根除所有欧洲人（解构欧洲多数群体），以便能够创建永恒和平与和谐的马克思主义乌托邦。
非洲、中国和伊斯兰世界
中国是一个单一文化的国家，就像非洲国家和伊斯兰世界的大多数国家一样。任何在非洲旅行和生活过的人都知道，所有非洲国家，可能除了南非之外，都是单一文化国家，他们对所有欧洲人都非常种族主义。从我踏上肯尼亚、科特迪瓦和利比里亚土地的第一秒起，我就亲身经历了压倒性的歧视。你真的不需要去非洲国家旅行就能理解这一点，因为这是一个广为人知的事实。与任何去过非洲的欧洲访客交谈，问他是否愿意带着家人在那里定居，答案将是"不"。这主要是由于歧视。对中国和伊斯兰世界也可以说同样的话。尽管伊斯兰或穆斯林国家中有相当数量的公民对欧洲人直接怀有敌意，但中国人通常只是想要你的钱。在一个存在持续且压倒性的被"忽悠"危险的国家度过余生是不可忍受的。我的继母曾是挪威驻上海副领事，我认识几位在那里工作的欧洲人。他们愿意为了公司任期的期限"应付"，因为薪水好，但他们永远不会带着家人永久定居在那里。我记得几年前我在上海的时候。在我三周的逗留期间，有超过10个人请求和我合影。显然，北欧人的外貌是相对罕见的景象，尤其是对在假期期间访问大城市的农村中国人来说。虽然这可能是一种迷人甚至是令人愉悦的经历，但有一群人整天不断盯着你是心理上的负担。一个例子是当我和我的朋友Xun去麦当劳时。顺便说一句，他是挪威华裔，曾是我在奥斯陆积极进行日内交易时的搭档。他教会了我与股票分析相关的高级中国K线图/MACD策略。总之，在麦当劳如同在任何地方，当有50个中国人不断盯着你时，你不太可能享受你的餐点。在上海逗留期间，有5次有人试图忽悠我（他们针对欧洲人），其中一次不幸成功了（一道简单的1欧元菜要价30欧元）。
至于国家层面的中国，他们积极对西藏和穆斯林维吾尔人居住的地区使用人口统计战。中国是一个单一文化国家，就像非洲国家和伊斯兰世界的大多数国家一样。
