# 2.24 无知与多元文化主义必须被摧毁
# 2.24 Ignorance and Multiculturalism must be destroyed
Orginal: 400-402

作者：D.L. Adams

有两个关键的障碍阻止西方有效参与或根本参与这场与伊斯兰的生死冲突。无知和多元文化主义（被原始合法化的道德相对主义）必须被解构和去合法化。

对政治伊斯兰的目的和教义的无知是一种道德和伦理的失败。对存亡之敌的无知是可耻和不可辩护的。多元文化主义必须被揭露为其实际所是：一种灾难性的无知伪哲学。多元文化主义建立于无知和愚蠢的乌托邦反智主义之上，它阻止我们回应政治伊斯兰，因为在我们道德破产的多元文化世界中这样做被视为不宽容。保卫你的国家免于文化和文字上的毁灭不是"不宽容"，而是最高的爱国主义和理性。

多元文化主义的基本双重概念是"宽容"和"包容性"。这些非常"美好"的概念建立在所有文化都具同等价值、文化之间的差异应当受到赞扬和"接受"而非在谴责适当时予以谴责的观念之上。一种文化可能是腐败甚至邪恶的观念在多元文化主义者破碎的反知识世界中被当作"偏执"予以摒弃。多元文化主义是伊斯兰在其中生长的污秽花园。

知道政治伊斯兰教义并有胆量公开讨论它的人被多元文化主义者和无知者指控为偏执。西拉、古兰经和圣训中的政治伊斯兰教义本质上是不能容忍和排他的，但这并没有阻止多元文化主义者。在他们的世界中存在着一种未明言的优越感；他们认为自己是优越的，因为他们是如此包容、接纳和"美好"。邪恶势力将这种来自西方文化的欢迎性的包容视为傻瓜的行为。这些愚蠢的行为对政治伊斯兰具有巨大价值，因为在错误优越感中，多元文化主义者对伊斯兰来说是完美的"顺民"（dhimmis），是一种完全旨在摧毁包容性和宽容的极权意理观念的完美仆人。多元文化主义者是文化毁灭和文字毁灭力量——即伊斯兰——的反知识的有用白痴。

美国长期以来有一个关于移民和同化非常流行的概念，即如果我们向新来者展示我们是多么美好、多么欢迎、多么善良，他们都会变成美国人。这个模式对欧洲人、亚洲人、南美人和来自世界各地的移民已经运作了好几代人。伊斯兰不适应这个模式。

该模式失败了，因为伊斯兰不被同化。伊斯兰不被同化，因为伊斯兰是关于顺服的。"伊斯兰"一词翻译为"顺服"。这个所谓"和平宗教"的信徒对同化不感兴趣，他们只感兴趣于将整个世界置于安拉的"宗教"之下。对于这种以仇恨和征服为核心的、可悲而残忍的政治意识形态的信徒来说，只允许一种宗教、一种政治制度、一种文化——即安拉和穆罕默德的。所有其他宗教、文化和政治制度都应被消灭。

因为多元文化主义者对他们所捍卫的文化是有意无知的——对他们来说所有文化（无论其多么卑劣）都只因为它们是"不同的"而值得捍卫——伊斯兰看到西方文化无法抵御它们。

事实上，多元文化主义是伊斯兰渗透我们制度和政治体系的工具。伊斯兰与所有非信徒处有永不完结的战争状态，这被称为"圣战"或"为安拉的事业而战"。圣战对伊斯兰如此根本，以至于在古兰经文本中被提及超过150次。因为多元文化主义者不需要对任何文化有任何了解（毕竟所有文化都是"平等的"，因此任何单一文化的知识几乎没有必要性、价值或意义），他们不明白自己正受到攻击。

多元文化主义和无知，两者携手同行，是西方文化的最大弱点。

同化模式失败了。它失败是因为复兴的伊斯兰及其圣战对同化毫无兴趣。多元文化主义者不知道这一点，当他们被告知真相时，他们顽固地拒绝相信。接受关于伊斯兰的真相就是接受多元文化主义基本概念的失败。

我们西方必须在做出移民决定时评判文化和这些文化的信徒。我们必须接受某些文化是不愉快的、且其信仰体系与我们的信仰体系相反的直白真相。人人都爱自由的观念是一种破碎和混乱的概念。它是一个破碎的空壳，因为在伊斯兰中，唯一被认为有价值的自由是完全顺服安拉和穆罕默德（以及伊斯兰教义）。伊斯兰在完全顺服伊斯兰教义中看到自由。

当我们为打开大门时，我们向拒绝我们宽容和包容性概念的力量开放自己和国家。这就是多元文化主义的荒谬和恐怖——它是一种自我毁灭的失败哲学。

我们必须学习伊斯兰教义。对该教义的了解类似于在体育比赛中拥有对方队伍的"战术手册"。但对我们来说这不是运动，这关乎我们作为非伊斯兰国家和文化的生存本身。我们必须学习伊斯兰教义，以消除我们对伊斯兰的无知。没有对伊斯兰的了解，我们就无法反对它；没有了解，有些人甚至没有意识到他们正在对我们发动战争，因此我们也正在与他们处于战争状态。

通过知识消除无知，如果我们想要作为热爱自由、宽容和"美好"的文化而生存下去，这是至关重要的。学习伊斯兰教义并教育他人这是一种爱国义务。

我们必须摧毁多元文化主义；解构它，去合法化它，并承认它是其实际所是：一种乌托邦式的自我毁灭幻想。所有文化并不等同。有些文化比另一些更好，有些是我们的敌人，有些是我们的朋友。这是合理、理性的思考。多元文化主义的基本概念是所有文化都有同等价值和价值。这是包容性、宽容和"美好"走向了极端。这是一种反智主义和非理性的思考，将导致我们文明的彻底丧失。多元文化主义必须被摧毁，并被扫进它应属的荒谬的垃圾箱。

未能摧毁多元文化主义，未能去合法化它及其信徒作为其实际所是的智识骗子和傻瓜，这是一种国家失败，其后果不仅仅是令人不快。多元文化主义是伊斯兰借此进入我们国家以消灭宿主文化的工具。

伊斯兰教义在这些观点上是明确的，所谓"和平宗教"的信徒也是如此。圣战是所有穆斯林的责任。这是安拉和穆罕默德的命令。进行圣战的穆斯林是"好"穆斯林，因为他们遵循安拉和穆罕默德的命令；不进行圣战的则是"极端分子"，因为他们拒绝伊斯兰教义的命令。拒绝圣战的人是圣战者的完美饲料，因为他们不是"好"穆斯林。这一概念必须被清楚理解。我们关于善/恶、对/错的观念在伊斯兰世界观中不适用。

多元文化主义必须被摧毁。多元文化主义者必须因其实际所是的自我毁灭的傻瓜而被嘲笑和批评。他们的哲学被我们的敌人利用，以在我们的制度和文化中获得入口和权力。多元文化主义者在他们的乌托邦无知中是敌人的工具——他们的哲学必须被拒绝、否定和解构，以便理性思考能够再次成为整个西方的主导思维模式。做得不够就是失败。

当我们对伊斯兰一无所知时，我们失败。当我们说所有文化都是等同的、我们无法再识别善恶、对错时，我们失败。对伊斯兰的无知是不可接受的；这是一种道德和伦理的失败。

多元文化主义是伊斯兰的工具；它是一种虚假"美好"的灾难性意识形态，被用来扼杀批判性思维和公开辩论。多元文化主义是一个完全的失败，因为它被我们的敌人用来摧毁我们。它是一个为"美好"的人提供的失败的虚假智识概念，以便他们逃避对自己、自己的文化和国家的责任。多元文化主义必须被摧毁。
