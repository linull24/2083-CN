# 意识形态之旅——从被灌输的多元文化主义狂热分子到保守革命者
# The ideological journey -- from indoctrinated multiculturalist zealot to Conservative Revolutionary
Orginal: 761-761

- 军事/警察/安保教育

我们的意识形态之旅始于我们在世界上的最初体验，这些体验有助于塑造我们对现实、正义和自由的感知。

当我三年前开始编写这本纂要时，我已经决定只涵盖与伊斯兰化和大规模穆斯林移民有关的问题，因为我害怕被贴上种族主义者的标签。我一直对被贴上种族主义者标签的前景感到恐惧，以至于我对自己施加了重大限制，不仅是在言语上，而且在我社会形象的所有方面。我知道这是大多数欧洲人的情况。我会说我允许自己被这种恐惧麻痹了。我倾向于不提起二战、种族的相关性，或者根本不提到种族这个词。

不幸的是，经过多年的研究和学习，我发现一切都是有联系的。我们目前的处境是二战和冷战的直接结果，是所谓的法兰克福学派和文化马克思主义/多元文化主义及政治正确性兴起的直接结果。我们不能忽视这些教义的相关性，如果我们要进行有意义的讨论，我们被迫反思这些问题。我们被迫提起所有的禁忌，重新评估所有当前"公认的真理"。

如果我12年前遇到我自己，我可能会认为我是一个极端的偏执狂，相信阴谋论：

"你说我们的学校机构在洗脑我们，我们的媒体在系统性地对我们撒谎？哈哈，你偏执了！清醒一点吧。"

所有爱国者有时都感觉像《黑客帝国》里那个想回到他的舱里、假装从未知道真相的人。那样会更好，不是吗？至少在一段时间内，但话说回来，我的孩子会因为我的懦弱而在几十年后憎恨我，因为他们将因为我的懦弱而被迫受苦。

事实是，我们的许多政治和文化精英，包括政治家、非政府组织领导人、大学教授/讲师、作家、记者和编辑——构成所谓A类和B类叛徒的大多数的个人——完全知道他们在做什么。他们知道他们正在为间接的文化和人口种族灭绝过程做出贡献，他们需要为他们的行为承担责任。真相需要大白于天下。

我们正处于一场非常血腥的文化战争的开端，一场民族主义与国际主义之间的战争，我们打算赢得它。我们不会允许欧苏联精英解构一切欧洲的东西，也不会允许他们将欧洲人民卖给穆斯林奴隶制。伊斯兰教将被第三次赶出欧洲，与乌玛勾结的马克思主义者、人道主义者和全球主义叛徒将因他们的叛国罪被追究责任。我们，文化保守派，在我们夺取我们自己国家的政治和军事权力之前，将无法决定我们自己的未来。实现这一目标的唯一途径是承担个人责任，并通过传播真相来帮助创造更多的抵抗战士。以有效的方式传播真相将加速主流欧洲人的意识形态之旅（意识形态成熟过程/意识形态信心水平）。人民必须学会克服他们的恐惧，为他们的信仰挺身而出。我们的生存取决于此。
