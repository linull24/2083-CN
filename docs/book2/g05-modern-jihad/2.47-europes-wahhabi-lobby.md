# 2.47 Europe's Wahhabi Lobby
Orginal: 500-501

欧洲的瓦哈比游说集团
极端分子聚在一起担忧不宽容问题。
斯蒂芬·施瓦茨——2005年10月6日，华沙
我早该知道事情会不对劲的。九月底，欧洲安全与合作组织（OSCE）——一个由55个国家组成的国际机构，其中包括像邻国白俄罗斯这样的独裁国家——在华沙这座美丽而充满灵性的城市召集了一个为期一天的圆桌会议。会议主题是"针对穆斯林的不宽容与歧视"。除了欧安组织外交官、工作人员以及两名美国国际宗教自由委员会的代表外，参会者包括约25名穆斯林非政府组织代表以及欧洲和北美的人权监督者。

我早该知道有些不对劲，因为自从在20世纪90年代末前往战后波斯尼亚-黑塞哥维那和科索沃以来，我目睹了欧安组织的许多恶作剧。别忘了，欧安组织就是那个有胆量提议"观察"最近一次美国总统选举以寻找所谓违规行为的国际组织。但它在巴尔干地区的记录尤其糟糕，正如《每周标准》所指出的那样。

坦率地说，欧安组织就是政治正确的人格化体现。其打击不宽容和歧视的议程涵盖了从妓女到校园欺凌受害者在内的所有人。但显而易见的是，伊斯兰教在欧洲的处境——最近在几个国家引发了流血事件——被欧安组织官僚们视为一个棘手的挑战。这些做好事的人似乎别无选择，只能将关于穆斯林的圆桌会议安排在华沙为期一周的"人类维度"大会常规议程之外，并将穆斯林会议安排在一家酒店的地下室举行。

据可靠消息来源报道，欧安组织在华沙举行的伊斯兰教会议，是对今年早些时候在西班牙科尔多瓦举行的反犹主义会议的一种交换。很快就清楚地表明，这次活动不过是为来自英国和丹麦等国的咆哮者和怪人提供的一个平台，他们来到这里是来捍卫激进伊斯兰教的。事实证明，那边的伊斯兰极端主义支持者比他们的美国同行更加咄咄逼人、公然挑衅和对抗性强。

因此，来自英国的一位宗教官员——华丽地（而且似乎是虚假地）冠名为"穆斯林宗教与种族和谐委员会"的伊玛目阿卜杜勒贾利勒·萨吉德博士——用大声谴责托尼·布莱尔的方式占据了上午讨论的大部分时间，指责他在"7/7"事件后对民权进行了所谓的攻击。在此之前，当被问及他遵循哪个伊斯兰法学派（madhhab）时，这位宗教领袖说："我枪毙所有法学派。"

伊玛目萨吉德向听众讲述了他与布莱尔对峙的许多经历，坚称对英国首相说伊斯兰教和恐怖主义之间完全没有联系。他还对关塔那摩的拘禁进行了长篇抨击。在参加会议的许多穆斯林看来，伦敦爆炸案和之前发生的袭击，以及激发这些袭击的激进意识形态，似乎都无关紧要；唯一重要的事情是反击英国、美国和其他欧洲当局的法律回应。

"打击极端主义"这句话被列入了会议议程，但直到最后才被提及。土耳其外交官奥穆尔·奥尔洪将声音压到近乎耳语的程度，在结束审议时确认，极端主义的问题最终必须得到处理，"因为这就是让我们所有人来到这里的原因。"但是听许多其他与会者的发言，你可能会以为欧洲非穆斯林对穆斯林的恐惧纯粹是一种毫无道理的偏见表现，或者如加拿大穆斯林妇女委员会的努兹哈特·贾弗里所说，是"美国外交政策决定"的产物。

当我向她指出，沙特资助的瓦哈比恐怖分子袭击了土耳其（一个反对美国伊拉克政策的国家），以及摩洛哥和印度尼西亚（这些国家与华盛顿的政策毫无关系）时，贾弗里女士仅仅承认确实存在其他"根源性原因"；但她对这些原因未作说明。

其他人则更不加掩饰。斯堪的纳维亚国家似乎特别没有能力将穆斯林极端分子排除在其领土之外。巴希·库赖希——一个否认自己信教、宣称自己不是实践穆斯林的男子——似乎将捍卫激进伊斯兰教作为一种职业选择，并自称是"丹麦少数民族组织联合会"的官员。尽管他承认自己无宗教信仰且与伊斯兰教疏远，但库赖希毫不犹豫地将自己呈现为这方面的专家。

库赖希竭尽全力占据会议进程的主导地位。当伊玛目萨吉德声称伊斯兰教与恐怖之间没有任何联系时，库赖希要求全球媒体甚至不得暗示这样的事情。他堆放在边桌上的印刷传单，其语气如同幻觉一般。在他看来，"美国遭受攻击"——美国有线电视新闻网在2001年9月11日后的标题——是冒犯性的偏见。此外，库赖希的传单坚称，"世贸中心和五角大楼的袭击没有证据，没有人承担责任，也没有任何一个特定的国家或团体被明确指认"负有责任。只不过是对伊斯兰教的"指指点点"而已。

库赖希还重复了已故的裘德·万尼斯基对理查德·珀尔的攻击，称其为"不加批判和民族主义式的新闻报道以及故意使用反伊斯兰术语作为宣传工具"的邪恶操控者。库赖希复制了基地组织及其支持者使用的陈词滥调："十字军东征又回来了"，伊拉克的萨达姆不过是一个"小独裁者"。库赖希的小册子甚至声称，"原教旨主义者"、"贫民窟"和"种族帮派"是仇恨用语，不应在任何媒体中使用。

其余的闲聊不那么激烈，但同样荒谬。加拿大穆斯林抱怨美国爱国者法案对他们国家的影响。下午渐进，像"所谓的恐怖分子"这样的措辞越来越频繁地被听到。英国人穆罕默德·阿齐兹来自"Faithwise"组织，宣称他的社区成员"首先对上帝负责……然后对乌玛（全球伊斯兰共同体）负责"，最后才对他们所居住的国家负责。

所有这些都发生在伦敦恐怖事件发生约三个月之后。会议结束时除了同意召开更多会议之外，没有达成任何其他成果。看起来，欧安组织和欧洲大部分地区一样，除了关于歧视的虔诚说教之外，对激进伊斯兰的挑战几乎拿不出什么答案。

斯蒂芬·施瓦茨是《每周标准》的经常撰稿人。
