# 全球有多少穆斯林支持激进伊斯兰或吉哈迪萨拉菲主义？
2.46 How many Muslims worldwide support militant Islam or Jihadi Salafism?
真相却是各种版本的萨拉菲主义（吉哈迪萨拉菲主义是最极端的）在每一个穆斯林国家都是一个重要因素。
瓦哈比主义是沙特阿拉伯的主导伊斯兰教形式，在科威特和阿联酋也很流行。瓦哈比主义也在索马里、阿尔及利亚和毛里塔尼亚的部分地区存在。
严格的萨拉菲主义版本（圣训派）存在于几个中东国家和南亚，特别是巴基斯坦和印度。
其他形式的萨拉菲主义在巴勒斯坦、叙利亚和约旦相当主导。它也存在于大多数其他穆斯林占多数的国家，包括土耳其、波斯尼亚和科索沃。

全球有多少穆斯林支持激进伊斯兰或吉哈迪萨拉菲主义？
这实际上是一个非常复杂的问题，涉及多个方面。首先，你必须考虑到积极支持、同情和共情这一事业之间存在很大的差别。你还必须意识到武装吉哈德（武装斗争）总是有特定的目标。达吉斯坦、车臣、泰国南部、菲律宾南部、叙利亚（巴勒斯坦）等地的吉哈德分子都想建立伊斯兰国家并实施伊斯兰教法。为了达到这一目标，他们必须发动战争并击败异教徒（非穆斯林）。然而，基地组织和西方的吉哈德分子有其他目标。他们承认伊斯兰教在欧洲仍然非常弱小，所以目前在欧洲首都街头进行武装吉哈德是不明智的。伊斯兰教和穆斯林的数量还需要再增长几代人。因此，基地组织试图通过在我们心中制造恐惧来压制西方媒体和西方政治家，以迫使他们向激进伊斯兰投降。在这个背景下，所谓温和穆斯林扮演着"好警察"的角色。他们会把贫困、现代殖民主义（以色列、伊拉克和阿富汗）、歧视、污名化和其他根源性原因作为吉哈迪萨拉菲主义兴起的原因。他们会要求欧洲政治家和媒体做出让步以"防止"激进伊斯兰。

根据丹尼尔·派普斯[11]的说法，全球大约10-15%的穆斯林积极支持激进伊斯兰主义者（通常是向哈马斯、真主党、伊斯兰圣战组织、基地组织或其他恐怖组织提供经济支持）。由于全球有超过12亿穆斯林，这总计为1.2亿至1.8亿人。这些穆斯林遍布世界各地，他们要么本身就是激进分子，要么是激进伊斯兰主义者的直接支持者。
被动支持的比例则高得多。一些民调显示，大约60-65%的穆斯林支持"伊斯兰吉哈德"和类似的中东恐怖组织。36%的巴基斯坦人认为巴基斯坦军队不应该在巴基斯坦追击基地组织[12]。
对吉哈迪萨拉菲主义的同情或直接支持绝不限于萨拉菲派穆斯林（瓦哈比派或其他萨拉菲派群体）。来自伊斯兰教所有学派的穆斯林都同情并支持激进的吉哈德分子。

几乎所有巴基斯坦逊尼派穆斯林都属于哈乃斐伊斯兰法学派。哈乃斐学派是四大法学派中最古老、最自由和宽容的学派。它也是伊斯兰教中最大的运动，全球约有30%的追随者。那么这是否意味着哈乃斐穆斯林是和平、温和的穆斯林呢？不幸的是，情况比这复杂得多。所有法学学派都相当不宽容（罕百里派是最不宽容的）。举例来说：所有伊斯兰法学派都宣扬对同性恋者和叛教者（脱离伊斯兰教者）处以死刑，并誓言将伊斯兰教法在社会中实施的最终目标。

哈乃斐派是巴基斯坦和欧洲的主导伊斯兰教形式，这似乎表明他们比其他穆斯林更温和宽容，然而，来自巴基斯坦的穆斯林以其保守和不宽容而臭名昭著。历史上记录有数十起种族灭绝和数百起大规模屠杀（见大屠杀），这些都是由哈乃斐穆斯林犯下的。历史上哈乃斐穆斯林实施吉哈德杀戮的事例数以千计，而且这一趋势甚至至今仍在持续。它是最大的一派，全球约有30%的穆斯林追随它。哈乃斐学派在土耳其、埃及北部、黎凡特地区以及巴尔干半岛、中亚和南亚、中国、俄罗斯和乌克兰的穆斯林社区中占主导地位。换句话说，对超过150万基督教亚美尼亚人的种族灭绝、希腊人和亚述人的种族灭绝也是所谓温和的"哈乃斐"伊斯兰教的杰作。
