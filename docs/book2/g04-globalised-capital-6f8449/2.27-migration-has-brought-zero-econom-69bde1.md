# 2.27 移民带来的经济利益为"零"
2.27 Migration has brought 'zero' economic benefit
作者：菲利普·约翰斯顿和罗伯特·温奈特

十年创纪录的英国移民对国家几乎没有产生任何经济利益，一项议会调查发现。

一个上议院委员会预计将于下周二报告，将质疑政府声称的外国工人每年为国家财富增加60亿英镑的说法。

预计会说这必须与人口增长及其对卫生和教育等地方服务的使用相平衡，导致人均利益微乎其微。

"我们的总体结论是，净移民对常住居民的经济利益很小，长期来看接近于零，"报告将这样说。

上议院经济委员会的调查结果有可能推翻部长们用来证明该国历史上最高移民水平合理性的关键论点。

该委员会的调查包括两位前财政大臣和几位前内阁部长，是首次试图平衡大规模移民的成本和利益。

人口每年增长超过19万，主要是由于移民。

外国工人现在占劳动力的12.5%，而十年前是7.4%。批评者说工党失去了对边境的控制，发放了太多工作许可，不应该向东欧开放劳动力市场。

然而，部长们说没有大规模移民，经济增长将会更慢。

一个为委员会制作的白厅文件说，过去五年平均产出增长率为2.7%，移民贡献了估计其中的15%到20%。政府表示这表明外国工人的贡献为60亿英镑——或每天70万英镑。

然而，委员会的最终报告预计会说，政府本应关注移民对人均GDP的影响，而不是整个经济。

牛津大学人口学教授戴维·科尔曼在他向委员会提供的证据中说，政府排除了来自犯罪、安全、种族关系进程、健康"旅游"和如结核病等输入性疾病的成本。

苏塞克斯大学移民研究中心的客座教授理查德·皮尔森说："虽然移民显然帮助缓解了通常长期存在的技能短缺，但他们也填补了许多低技能工作，通常工资非常低。"

"这些移民很可能会取代低技能的本土工人，并减少雇主招募和培训他们的激励。"
