2.2 欧拉比亚密码——2008年更新
作者：Fjordman

自2006年以来发生的情况是，欧洲领导人越来越公开地谈论扩大欧盟以包括阿拉伯世界的想法，尽管他们当然不会将此描述为将欧洲大陆投降给伊斯兰教。这在以前并非完全保密——2002年，时任比利时外交部长、现任欧洲委员会委员的路易·米歇尔告诉比利时议会[2]，欧盟最终将涵盖北非和中东以及欧洲——但为什么现在要公开此事？我的理论是，欧盟领导人认为他们的人民已经被打败且无关紧要。在里斯本条约/欧盟宪法[3]实施之后，人民不再有发言权，可以被安全地忽视。他们多年来一直蔑视我们，不再费心隐藏这一点。我们是羊，不构成威胁，而他们必须继续安抚穆斯林。

组建"地中海联盟[4]"或"地中海联合体"的公开计划——将包括所有欧盟成员国、毛里塔尼亚、摩洛哥、阿尔及利亚、突尼斯、利比亚、埃及、约旦、巴勒斯坦权力机构、黎巴嫩、叙利亚和土耳其——于2008年中启动，阿拉伯人对这样的联盟可能使它们与以色列关系正常化表示担忧[5]。该计划伴随建立一个"南北共同主席制"和一个常设秘书处，以及为该地区定义"优先项目短名单"。欧洲委员会提议在欧盟和一个地中海（解读为：穆斯林）国家之间建立一个共同主席制，通过共识选择，任期两年。布鲁塞尔正在绘制将被称为"巴塞罗那进程——地中海联合体"的机构轮廓。注意他们如何明确将其与巴特·耶奥尔描述的欧拉比亚巴塞罗那进程联系在一起。尽管如此，即使在地中海联盟启动后，我仍听到声称任何关于欧拉比亚的谈论都是"危险的伊斯兰恐惧症阴谋论"。

在一封出现在受人尊敬的意大利报纸《晚邮报》上的信[6]中，前意大利总统弗朗西斯科·科西加在2008年透露，意大利政府在20世纪70年代同意允许阿拉伯恐怖组织在该国自由活动，以换取免遭攻击。已故总理阿尔多·莫罗的政府达成了一项"意大利国家与巴勒斯坦抵抗组织（包括恐怖组织）之间的秘密非交战协议"。据这位前总统称，是莫罗本人设计了与外国阿拉伯恐怖分子的协议条款。"协议的条款是，巴勒斯坦组织甚至可以在该国维持武装行动基地，他们有出入自由，不受正常的警察管控，因为他们由特工部门'处理'。"作为内政部长，科西加说他了解到在意大利的巴解组织成员作为阿拉伯联盟代表享有外交豁免权。"巴勒斯坦组织甚至可以在该国维持武装行动基地。"

这是欧拉比亚的正式诞生，当时西欧各国政府在阿拉伯恐怖分子和产油国的压力下屈服，放弃了传统的亲以色列立场，逐步与阿拉伯-伊斯兰世界结盟。绝对没有理由认为[7]意大利人是唯一做出此类"交易"的国家。除了文化和政治合作之外，欧洲政府还同意向阿拉伯人——特别是巴勒斯坦人——支付大笔"保护费"以减少恐怖主义威胁。这只能被视为吉兹亚，这种做法后来蔓延到整个欧盟，欧盟每年向巴勒斯坦人支付数千万甚至数亿欧元。

MEDA项目[8]是实施欧地中海伙伴关系的主要财政工具，在2000-2006年期间花费了53.5亿欧元用于各种项目，根据欧盟官方网站。在1995-1999年期间，分配给MEDA的资源中约86%流向阿尔及利亚、埃及、约旦、黎巴嫩、摩洛哥、叙利亚、突尼斯、土耳其和巴勒斯坦权力机构。

从2007年起，MEDA被欧洲邻里和伙伴关系工具[9]取代，该工具在2007年至2013年期间预计将花费110亿欧元，用于：除其他事项外，促进欧洲和阿拉伯国家在能源和交通领域的合作；在高等教育以及教师、研究人员和学生流动方面；通过人民对人民的接触进行多元文化对话，包括与居住在欧盟国家的移民社区的联系，以及公民社会之间的合作、文化机构和青年人交流。欧洲委员会——拥有广泛立法权力的欧盟强大政府——应协调与非政府组织、大学、教会、宗教协会和媒体在此项目相关事务上的合作；所有这些都根据互联网上可获取的文件，但由于主流媒体很少提及，公众几乎不知道。

我收到了对我原始欧拉比亚密码的一些批评性评论，其中一项声称文中提到的2006年阿尔及尔宣言未由任何官方欧盟机构签署。然而，组织此次会议的安娜·林德欧地中海文化对话基金会，作为欧盟对外关系计划和欧地中海伙伴关系的一部分，在欧盟官方网站上有链接。参与其中一些组织在技术上可能是独立组织，但毫无疑问，欧洲委员会是这些网络背后的推动力量。此外，如果你仔细阅读这些文件，你会发现它们提到欧洲和阿拉伯世界之间"教育体系的协调统一"。这需要最高层面的当局参与。

2008年9月，丹麦一些媒体中的一份简短声明[10]（我在大多数欧洲国家的主流媒体中看到很少提及这一深远建议）说，居住在欧盟的穆斯林将来将能够根据伊斯兰教法离婚。这是委员会的观点，委员会建议夫妻如果能选择他们离婚时将遵循哪国的法律——只要他们与该国存在某种联系。丹麦人民党发言人莫滕·梅塞施密特对该建议极为担忧："这是一个完全脑残的想法，委员会将在欧盟使用不人道的伊斯兰教法的可能性，"他说。

人们不理解的是，这是欧盟领导人早已同意的长期交易的一部分。几乎所有西欧领导人都已经投降了。不再存在伊斯兰教法是否会被正式接受为法律的问题（就像对居住在英国穆斯林那样），只是如何实施的问题。

与此同时，拟议的欧洲逮捕令[11]列出了一些罪行，包括恐怖主义、武装抢劫、强奸以及种族主义和仇外心理，这些在整个欧盟都是应受惩罚的。欧洲逮捕令要求，任何被某一成员国根据所列罪行类别（这几乎可以涵盖任何东西）指控的人，可能在发出逮捕令的国家的当局在没有另一个成员国任何干涉的情况下被逮捕。被告必须在十天内被移交到发出逮捕令的国家接受审判，执行国的司法或其他干涉均不被允许。

根据欧盟文件，种族主义包括"伊斯兰恐惧症[12]"，这意味着"伊斯兰恐惧症"可能很快在整片欧洲大陆被视为与强奸和武装抢劫一样严重的罪行。与此同时，欧盟领导人正忙于将欧盟扩大到北非和中东，从而向欧洲涌入数千万额外的穆斯林。在不久的将来，我们可以想象这样一种情况：当局可以在比如说丹麦或意大利逮捕一个人，这个人发表了一幅可能被认为冒犯伊斯兰教的漫画。然后他将会被移交给阿尔及利亚、埃及或约旦的当局。

请记住，根据伊斯兰教法，亵渎伊斯兰教将被处以死刑。欧洲的多元文化主义即将达到其公开极权主义的阶段。那些认为这是一个笑话的人可以看看荷兰漫画家格雷戈里乌斯·内克斯霍特[13]，他于2008年因"侮辱"穆斯林的漫画而被捕。一些公开可获取（但公众几乎不知道，因为主流媒体从不提及）的文件声明，欧盟应在未来十年内与阿拉伯"伙伴国家""协调统一"教育和法律体系。就在我们说话的同时，这正在我们背后被谈判。

欧洲委员会主席若泽·曼努埃尔·巴罗佐此前对荷兰就里斯本条约/欧盟宪法举行第二次公投的前景表示不安[14]。"公投使批准欧洲条约的过程变得更加复杂且更不可预测，"他说，要求"每个成员国"考虑公投时要"三思"。巴罗佐先生在担任葡萄牙总理的先前职务时，于2004年支持在本国就欧盟宪法举行公投——但自那以后他的想法已经改变。"作为总理，我曾支持公投，但它确实使我们27个成员国的生活更加困难。如果就建立欧洲共同体或引入欧元举行过公投，你认为这些会通过吗？"

正如记者尼克·法奇在2008年10月的英国报纸《每日快报》[15]中所说：
"超过五千万非洲工人将被邀请到欧洲，这是《每日快报》今天可以揭示的一项深远的秘密移民协议。本周在马里开设的一个有争议的纳税人资助的'就业中心'，只是推动'非洲和欧盟人员自由流动'的第一步。布鲁塞尔经济学家声称，英国和其他欧盟国家到2050年将'需要'5600万移民工人来弥补由于欧洲各地出生率下降和死亡率上升导致的'人口衰退'。由欧盟统计局编写的报告警告说，如果欧洲希望有希望满足其日益增长的老年人口对养老金和医疗保健的需求，可能在两年内需要大量移民来弥补缺口。它说：……'拥有足够的工作年龄人口对经济和税收收入至关重要。'这份由法国欧洲议会议员弗朗索瓦丝·卡斯泰编写的报告呼吁给予移民合法权利和获得社会福利如福利金的机会。卡斯泰女士说：'成员国必须冷静地对待移民问题。说"是的，我们需要移民"……并不是一个新进展，我们必须接受它。'"

让我们总结一下迄今为止的发现：欧盟已接受联盟应扩大到包括穆斯林中东和北非。欧盟已接受未来几年应允许来自北非以穆斯林为主的国家的数千万移民在欧洲定居。这被认为"对经济有利"。它计划为它邀请到欧洲定居的数百万穆斯林实施伊斯兰教法。它通过了更严格的反种族主义法律，同时明确表示"伊斯兰恐惧症"构成一种种族主义，并与伊斯兰国家合作重写学校教科书，为欧洲儿童提供伊斯兰教的"正面"形象。最后，欧盟正在制定一项逮捕令，规定那些被指控犯有严重罪行（例如种族主义）的人可以在没有他们所居住的民族国家不当干涉的情况下被逮捕。实质上，欧盟正在将一个整个大陆正式投降给伊斯兰教，同时摧毁既定的民族文化，并准备骚扰那些不同意这一政策的人。这构成西方历史上——也许是人类历史上——最大的有组织背叛，却被誉为"宽容"的胜利。

若泽·曼努埃尔·巴罗佐，这位对五亿人口不负责任的政府领导人，曾声明[16]欧盟是一个帝国。也许我们认为将欧盟视为帝国是荒谬的，但他的声明表明，一些处于领导职位的人确实这样想。记住历史上所有帝国都是通过战争创建的是有意义的。如果欧盟是一个帝国，这意味着正在对某人发动一场战争。确实如此：一场通过大规模移民对本土欧洲人发动的文化和人口战争。虽然帝国通常是通过对其他人发动战争而创建的，但欧盟是历史上第一个由领导人允许其他人对其自己人民发动战争而创建的帝国。

欧盟——或者如果你想的话，欧拉比亚帝国——是精英为了拆除他们本该服务的国家而进行的赤裸裸的权力夺取。他们不再满足于作为小国人民的仆人，而是渴望成为一个不负责任的精英的成员，随心所欲地统治一个庞大的帝国。这就是为什么即使人们在自己的城市中被炸死、强奸、抢劫和谋杀，他们仍然继续推动大规模移民，仿佛什么都没有发生。他们不在乎。他们是战斗中的将军。普通公民只是炮灰，是在征服他们光荣的新帝国中将被牺牲的棋子。大规模移民被用来同时粉碎所有民族国家，使本土人没有真正的替代避难所，也没有国家能够互相援助对抗推进的伊斯兰化。

最后，我们应该记住一件事：这一切都始于20世纪70年代对阿拉法特等阿拉伯恃强凌弱者的绥靖，他们使用石油或恐怖主义或两者兼有作为武器。欧洲人应该努力摆脱背叛文化，但我们也需要摆脱首先将我们带入这个乱局的绥靖文化。不给巴勒斯坦人一分钱；一分钱也不给。如果他们需要钱，可以向沙特人要。不再对伊斯兰恐怖分子进行绥靖或交易。正是"对话"——欧阿对话——制造了这些问题。不再"对话"。对付恃强凌弱者的唯一方法是通过揍他的鼻子让他退缩。这是对付圣战分子的唯一恰当方式。
